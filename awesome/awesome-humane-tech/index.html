    Humane Technology




    About this list¶

    科技和社交媒体对我们的社会产生了重大影响. 虽然许多创新技术发明正在改善我们的生活，但人们越来越意识到这些趋势带来的负面影响，例如社交媒体成瘾，心理健康问题和社会侵蚀. 讲话 these problems一个由特里斯坦哈里斯领导的深受关注的前技术内幕人士组成的世界级团队创造了 The Center of Humane Technology . 我们的愿景是：

    > 无处不在的人道技术，刺激人类蓬勃发展，人类蓬勃发展.

    此列表侧重于（主要是开源）项目，这些项目与人道科技主题直接相关. 请阅读 guidelines for contributing 在发送PR之前.

    重要：下面列出的资源是本清单的贡献者真诚收集的，希望它们对您有用. 但按照 License 我们不能保证所引用软件的质量或适用性. 所以在安装之前要正确地告知自己，如果遇到不合适或错误的事情，请在我们的文件中提出问题 Issue Tracker 所以我们可以采取适当的行动. 谢谢！

    Community¶

    我们邀请您参加我们的会议 Humane Tech Community Forum 阅读更多关于这些有趣的人文科技科目. 如果您想参与我们的讨论，帮助提供真正的解决方案，那么我们很乐意欢迎您成为我们的新成员之一. 我们正在建造“人文科技金字塔”，您可以与我们共同建设.

    Be a Builder Help Improve Wellbeing, Freedom and Society! 并成为真正的_Hmane人技术活动家！_

    • Stretchly - 一个跨平台的应用程序，提醒您在使用计算机时休息一下.

    Mindfulness¶

    • Mindfulness at the Computer - 帮助您在使用计算机时保持注意力.

    • Chomper - 用于Linux的Internet阻止程序，带有allowLists / blockLists和计时器，可帮助您提高工作效率.
    • Jobs Done - 一个由Deep Work启发的结束工作日的仪式应用程序.
    • Break Timer - Chrome / Firefox扩展程序，用于定期休息，防止RSI，眼睛疲劳和疲劳.
    • MindBell - Android的正念铃和冥想计时器.

    Awareness¶

    • Hack the Media - 媒体素养指南，强调我们的媒体资源如何能够给我们一个有缺陷的世界观.
    • Hack an Engineer - 软件工程师的媒体素养指南.
    • Facebook Data Analyzer - 分析您自己数据的Facebook副本.
    • HTTP Leaks - 所有可能的方式，网站可以泄漏HTTP请求.
    • Who Tracks Me - 为在线跟踪带来透明度 - 由Cliqz和Ghostery建立.
    • Awful AI - 一个精选的列表，用于跟踪当前人工智能的可怕用法 - 希望提高认识.
    • Firefox Lightbeam - Addon可实时显示第一方和第三方网站之间的Web请求.
    • OpenWPM - 普林斯顿网络透明度和问责制项目，一个隐私测量框架.
    • Email Privacy Tester - 根据许多已知的跟踪技术检查您的电子邮件提供商和客户端保
    • Browser Autofill Phishing - 演示对用户隐藏的自动填充表单字段的网络钓鱼.

    • Who Targets Me? - A browser extension that tracks which entities are targeting which demographics with adverts.
