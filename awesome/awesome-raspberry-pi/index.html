    Raspberry Pi

    Awesome Raspberry Pi¶

    > Raspberry Pi是Raspberry Pi基金会在英国开发的一系列信用卡大小的单板计算机，用于促进学校和发展中国家的基础计算机科学教学. 官方链接： Raspberry Pi Homepage

    此列表是符合的列表，包括工具，项目，图像和资源 Awesome Manifesto

    Models¶

    （消费者）Raspberry Pi模型的完整集合包括：

    • Raspberry Pi 1，A型和B型（2012年，已停产）
    • Raspberry Pi 1, Model B+ （2014年7月）
    • Raspberry Pi 1, Model A+ （2014年11月）
    • Raspberry Pi 2, Model B （2015年2月）
    • Raspberry Pi Zero （2015年11月）
    • Raspberry Pi 3, Model B （2016年3月）
    • Raspberry Pi Zero W （2017年2月）
    • Raspberry Pi 3, Model B+ （2018年3月）
    • Raspberry Pi 3, Model A+ （2018年11月）

    OS Images¶

    • Raspbian - 官方支持的Raspberry Pi OS，基于Debian，可作为精简版.
    • NOOBS - New Out Of the Box软件，适合初学者的简易操作系统安装程序.
    • Ubuntu MATE - 基于MATE桌面的Raspberry Pi的Ubuntu发行版.
    • Snappy Ubuntu Core - 物联网的官方（最小）Ubuntu分发.
    • Windows 10 IoT Core - 物联网的Windows 10发行版.
    • OSMC - 基于Kodi的开源媒体中心.
    • PiNet - 管理Raspberry Pi教室的系统.
    • Risc OS - 非Linux操作系统源自开发ARM微处理器的组.
    • RuneAudio - 免费和开源的操作系统，可将嵌入式硬件转换为Hi-Fi音乐播放器.
    • OpenELEC - 开放嵌入式Linux娱乐中心，一个非常轻量级的Kodi构建.
    • HypriotOS - 基于Debian的最小操作系统，针对运行Docker进行了优化.
    • Kali Linux - 针对ARM设备的渗透测试和道德黑客Linux发行版.
    • RetroPie - Raspberry Pi上的复古游戏.
    • Alpine Linux - 基于musl libc和busybox的面向安全的轻量级Linux发行版.
    • Arch Linux ARM - 轻量且灵活的Linux发行版，试图保持简单.
    • Volumio - 无头音响发烧友音乐播放器，旨在以最高保真度播放音乐.
    • Recalbox - 在Raspberry Pi上拖放轻量级复古游戏和媒体中心.
    • Lakka - 完全基于RetroArch构建的Raspberry Pi上的复古游戏.
    • RasPlex - Plex client for the Raspberry Pi.
    • chilipie-kiosk - 直接引导至全屏Chrome的图像，非常适合仪表板和构建显示器.
    • DietPi - 最小图像设计适合2GB SD卡，具有大量可配置的设置和脚本.
    • CentOS - Raspberry Pi上的CentOS.
    • Minibian - 最小的Raspbian（比Jessie Lite轻）.
    • BerryBoot - Bootloader /通用OS安装程序，支持VNC和HDMI-CEC.
    • PirateBox - 匿名离线移动文件共享和通信系统.
    • OctoPi - 3D打印机的分发.
    • Kano OS - 用于探索，创建和播放的开源操作系统 - 免费提供Raspberry Pi和新的Pi 3.
    • balenaOS - 开源操作系统，用于在嵌入式设备上运行Docker容器，这些设备专为提高可靠性而在生产中得到验证.
    • Fedora - 为Pi构建的Linux Fedora发行版.
    • motionEyeOS - 将单板计算机变成视频监控系统的Linux发行版.
    • NextCloudPi - 基于Raspbian的Nextcloud准备好的图像. 功能Nextcloud 11在Raspbian 8上运行，启用了PHP 7和HTTP2的Apache服务器.
    • PiDeck - 小型DVS系统，允许您使用时间码乙烯基控制数字音乐文件.
    • OpenWRT - OpenWrt被描述为用于网络管理的嵌入式设备的Linux发行版.
    • FreeBSD - FreeBSD是一种先进的计算机操作系统，用于为现代服务器，台式机和嵌入式平台提供动力.
    • NetBSD - NetBSD是一个免费，快速，安全且高度可移植的类Unix开源操作系统.
    • Hass.io - 家庭自动化操作系统/嵌入式设备应用程序，也可单独使用.
    • Android Things - 为各种消费，零售和工业应用构建连接设备.
    • Gladys Project - Gladys，你的家庭助理.
    • OpenMediaVault - OpenMediaVault是基于Debian Linux的下一代网络附加存储（NAS）解决方案，包含SSH，（S）FTP，SMB / CIFS，DAAP媒体服务器，RSync，BitTorrent客户端等服务.
    • EZ-WifiBroadcast - 经济实惠的无线数字高清视频传输变得简单.
    • DroneBridge - WifiBroadcast扩展，是DJI Lightbridge和其他类似系统的真正替代品.
    • TrueOS - 基于FreeBSD的瘦客户端解决方案.
    • OpenHABian - OpenHAB家庭自动化软件的预配置版本.
    • FabScanPi - FabScanPi是一款使用Raspberry Pi相机模块的开源3D激光扫描仪.
    • Rocket Show - 在舞台上播放同步的背景音轨，视频和DMX灯光序列.
    • SamplerBox - Drop'n'play采样器：在SD卡上放下.WAV样本，然后播放！
    • SARPi - 代表Raspberry Pi上的Slackware ARM.
    • Gentoo - 用于Raspberry Pi的Gentoo Stage 3 tarball.
    • MoodleBox - MoodleBox在Raspberry Pi上提供了Moodle学习管理系统.

    Tools¶

    • PiBakery - 基于块，易于使用的Raspberry Pi设置工具.
    • ApplePi Bakery - macOS应用程序，可以轻松地将图像安装/备份/恢复到SD卡上.
    • Etcher - SD卡刻录机应用程序，对最终用户来说很简单，可以为开发人员扩展，并且可以在任何平台上运行.
    • PiShrink - Bash脚本自动缩小pi图像，然后在启动时将其调整为SD卡的最大大小.
    • OpenVPN-Setup - 用于将Raspberry Pi设置为OpenVPN服务器的Shell脚本.
    • pi-gen - 用于创建raspberrypi.org Raspbian图像的工具. 这可以用于创建自己的自定义图像，并安装特定的包等.
    • Network Presence Detector - 设置Pi Zero以扫描WiFi网络上的设备，并使用它来确定谁是“家”.
    • Resin Bridge - 简单的应用程序/存储库，将Raspberry Pi的wlan0接口桥接到通过以太网（eth0）插入的设备.
    • WiFi config generator - 使用wifi设置生成wpa_supplicant.conf文件的简单工具
    • Raspbian QEMU with network - 在Raspbian上启动QEMU的简单工具，需要对系统和内核进行必要的修改. 自动网络访问. 方便在PC中创建图像.
    • RaspAP-webgui - 一个简单的响应式Web界面，用于控制Raspberry Pi上的wifi，hostapd和相关服务.
    • HealthyPi - HAT包括重要的健康监测，如心电图，呼吸，脉搏血氧仪和可选的血压和体温感应.
    • PoisonTap - 通过USB利用受锁/密码保护的计算机，删除基于WebSocket的持久后门，暴露内部路由器，并使用Raspberry Pi Zero和Node.js虹吸cookie.
    • P4wnP1 - P4wnP1是一个高度可定制的USB攻击平台，基于低成本的Raspberry Pi Zero或Raspberry Pi Zero W（HID后门所需）.
    • CusDeb - 用于创建安装了特定包的自定义映像的Web UI.
    • Alpha - 使用GDB与此系统级GDB服务器远程加载，调试和测试裸机程序.
    • Pieman - 基于Raspbian，Devuan，Ubuntu和Alpine Linux创建自定义图像的脚本.
    • iotwifi - 8MB的Wifi AP +客户端管理 Docker Container，使用REST API同时控制wifi客户端和站点模式.
    • TorTiPi - Shell脚本，用于自动完成将Raspberry Pi转换为基于tor的wifi热点的任务.
    • Processing - 在Raspberry Pi上运行的处理开发环境.
