    AutoHotkey

    Awesome AutoHotkey ¶

    精彩的精选列表 AutoHotkey 库，库分发，脚本，工具和资源. 受到另一个人的启发 awesome lists . 请阅读 CONTRIBUTING.md 在贡献之前.

    已过时或已停产，但历史上相关的项目可以找到 Historical.md

    发展状况：

    Libraries¶

    有用的AutoHotkey库列表. 库是具有一些可重用功能的代码，可以与您自己的代码结合使用以创建新功能.

    Clipboard¶

    • WinClip - 通过Deo-WinClip是一个剪贴板操作类，扩展了AutoHotkey的剪贴板功能，包括对RTF，HTML和图像的支持. 论坛帖子： link.

    Console¶

    • AHKonsole - 通过G33kdude - 基于类的AutoHotkey库，用于控制台支持. 此库使您可以创建表示要与之交互的控制台的对象，以及多个控制台缓冲区对象，以便于双缓冲. 论坛帖子： link.
    • LibCon - by joedf - AutoHotkey Library For Console Support. 此库使您可以编写控制台应用程序并与其他控制台实例交互. 基本上，这个库可以促进与编写和与控制台交互有关的任何事情. 论坛帖子： link.

    Data format¶

    • AHK_ctable - 通过hoppfrosch - 以表格格式处理字符串的库 - 论坛帖子： link.
    • AutoHotkey-JSON - 通过cocobelgica - AutoHotkey的JSON lib. 论坛帖子： link.
    • CSV - 通过trueski / kdoske - 使用CSV文件和Listview函数的库. 论坛帖子： link.
    • List manipulation functions - 由Laszlo - 函数库来操作逗号分隔的列表. 论坛帖子： link.
    • ObjCSV - 由JnLlnd - 库将CSV文件加载/保存到Objects和Listview函数. 论坛帖子： link.
    • ObjDump/ObjLoad - 通过HotKeyIt - 将对象序列化/反序列化到变量/内存.
    • SerDes - by cocobelgica - 序列化/反序列化AutoHotkey对象结构. 论坛帖子： link.
    • Table - 通过VxE - Library以表格（TSV）格式和Listview函数操作字符串. 论坛帖子： link.
    • XA - 通过trueski / hi5 - 将数组序列化/反序列化为XML. 论坛帖子： link.

    Data Structures and Algorithms¶

    • Facade - Shambles - 一套功能编程库. - 论坛帖子： link
    • HashTable - 通过Shambles - AutoHotkey的哈希表实现.
    • LibCrypt - 由不同的作者 - 加密和编码功能的集合.
    • Type_Checking - 由Shambles - AutoHotkey的类型检查 - 论坛主题： link

    Database¶

    • ahkDBA - by IsNull - 一个OOP-SQL数据库访问框架. 论坛帖子： link.
    • Class_SQLiteDB - 通过我 - AHK SQLite API包装类. 论坛帖子： link.

    Filesystem¶

    • FileGetProperties - by kon - 用于检索扩展文件属性的函数.

    Graphics¶

    • GDIp - by tic - 全功能库，有助于与微软的gdiplus.dll交互 - 论坛主题： link.
    • AHKv2-GDIP - 更新上述GDI +库，兼容AHK v1.1和AHK v2 - 论坛主题： link.
    • GDIp_ImageSearch - by tic - 使用gdiplus.dll在库中搜索图像实例的库. 有关MasterFocus的改进版本，请参阅该线程的结尾，或者查看他 GitHub repo here
    • Simple GDI class - 作者GeekDude - 一个旨在简化使用低级GDI函数的类.
    • Particle System - 通过tidbit - 一个简单的类，使用GDI +将粒子添加到GUI或屏幕上. 论坛帖子： link.

    Hotkeys¶

    • CHotkeyControl - 由evilC - 替换支持鼠标按钮等的AHK热键GuiControl（部分成熟）.

    • calc() - 数学表达式评估包含括号.

    Networking¶

    • AHKsock - TheGood - 基于函数的套接字库. 支持TCP. 论坛 link
    • Socket Class (überarbeitet) - Bentschi - 基于类的套接字库. 支持TCP和UDP.
    • Socket.ahk - GeekDude - 基于Bentschi的论坛库 - 论坛 link
    • WebSocket.ahk - 由GeekDude - 基于类的WebSocket库 - 论坛 link
    • WinSCP.ahk - 通过Lipkau - Lib允许在AHK中使用WinSCP

    System¶

    Text manipulation¶

    • String Things - 通过tidbit - 独立的字符串操作函数.
    • TF - by hi5 - 用于处理文本文件的函数，如 .txt， .ahk， .html， .css等和字符串（或变量）. 论坛帖子： link.

    有用的AutoHotkey脚本列表. 脚本是旨在用作独立程序的代码，并不打算与其他代码集成.

    Filesystem¶

    • Belvedere - 根据文件名称，扩展名，大小，年龄等设置对文件执行操作（移动，复制，删除等）的规则. 更多信息 link.

    Graphics¶

    • Fun with GDIPlus - 有趣的GDI +示例.

    GUI¶

    • Examples of Non-Standard GUIs (ActiveX, GDI, etc.) - 使用非标准方法生成漂亮用户界面的GUI示例.

    Maths¶

    • Monster - 评估字符串中的数学表达式（计算器）.
    • Unit Converter - 单位转换器，具有最常见的英语和科学单位，最常见的数量从长度到密度到导热系数. 还包括物理和数学常数部分.

    Mouse¶

    • EitherMouse - 多个鼠标，单独设置，第二个鼠标上的自动交换鼠标按钮. 论坛帖子： link.
    • MouseGestureL - 通过鼠标手势控制应用程序. 可以通过可定制的界面定义手势和动作. 英文和日文文件 - 日文主页 link
    • Radial Menu - 强大的热键，启动器，鼠标手势系统等等（肤色） - 论坛主题： link

    Typing¶

    • AutoComplete - 在您键入时建议并完成单词. 论坛帖子： link.
    • Half-QWERTY - 单手打字. 使用空格键作为修饰符，用户只需单手即可生成全尺寸键盘任一侧的字符. 更多信息通过论坛帖子： link
    • KeyPress OSD - 屏幕显示，以明显可见的文本大小显示每个按键或鼠标按钮. 论坛帖子： link
