    Awesome Elixir ¶

    精选的Elixir图书馆，资源和闪亮的东西，精选的精选列表 awesome-php.

    如果您认为应该添加一个包，请在相应的问题上添加：+1 :(：+1：）或创建一个新包.

    有 other sites with curated lists of elixir packages 你可以看看.

    Actors¶

    与演员等合作的图书馆和工具.

    • dflow - 流水线流程处理引擎.
    • exactor - 帮助更容易实现Elixir中的演员.
    • exos - 端口包装器，用于转发演员和呼叫链接端口.
    • flowex - 使用Elixir GenStage进行基于铁路流量的编程.
    • mon_handler - 监视给定GenEvent处理程序的最小GenServer.
    • pool_ring - 根据哈希环创建池.
    • poolboy - 一个hunky Erlang工人池工厂.
    • pooler - OTP流程池应用程序.
    • sbroker - 基于逗留时间的主动队列管理库.
    • workex - EVM过程中的背压和流量控制.

    Algorithms and Data structures¶

    算法和数据结构的库和实现.

    • array - Erlang数组的Elixir包装器库.
    • aruspex - Aruspex是一个可配置的约束求解器，纯粹用Elixir编写.
    • bimap - 纯Elixir实施 bidirectional maps 和多图.
    • bitmap - 纯Elixir实施 bitmaps.
    • blocking_queue - BlockingQueue是一个作为GenServer实现的简单队列. 它具有固定的最大长度，在创建时建立.
    • bloomex - 可扩展布隆过滤器的纯Elixir实现.
    • clope - Elixir的实施 CLOPE：一种快速有效的事务数据聚类算法.
    • combination - Elixir库，用于从Enumerable集合生成组合和排列.
    • count_buffer - 缓冲大量计数器并定期冲洗.
    • cuckoo - 纯粹的Elixir实现 Cuckoo Filters.
    • cuid - 用Elixir编写的针对水平缩放和顺序查找性能优化的抗冲突ids.
    • data_morph - 从数据创建Elixir结构.
    • dataframe - 包提供类似于Python的Pandas或R的data.frame（）的功能.
    • datastructures - 用于处理数据结构的协议，实现和包装器的集合.
    • def_memo - 使用genserver后备存储的elixir的memoization宏（defmemo）.
    • dlist - 在所有药剂执行.
    • eastar - 纯Elixir中的A *图形寻路.
    • ecto_materialized_path - ecto模型的树结构，层次结构和祖先.
    • ecto_state_machine - 在Elixir上实现的有限状态机模式，用于Ecto.
    • elistrix - 延迟/容错库，帮助您将应用程序与缓慢或失败的服务的不确定世界隔离开来.
    • emel - 用elixir编写的简单实用的机器学习库.
    • erlang-algorithms - 流行数据结构和算法的实现.
    • exconstructor - 用于生成结构构造函数的Elixir库，可轻松处理外部数据.
    • exfsm - 简单的elixir库来定义静态FSM.
    • exkad - 一个 kademlia 在Elixir中实施.
    • exmatrix - ExMatrix是一个用于处理矩阵的小型库，最初是为并行测试矩阵乘法而开发的.
    • ezcryptex - Cryptex顶部的薄层.
    • fnv - 纯Elixir实现Fowler-Noll-Vo散列函数.
    • fsm - 有限状态机作为功能数据结构.
    • fuse - 该应用程序为Erlang实现了所谓的断路器.
    • gen_fsm - 一个通用的有限状态机 - 围绕OTP的gen_fsm的Elixir包装器.
    • graphmath - 用于执行2D和3D数学的Elixir库.
    • hash_ring_ex - Elixir的一致哈希环实现.
    • hypex - HyperLogLog的Fast Elixir实现.
    • indifferent - 使用自定义密钥转换对Elixir地图/列表/元组进行无差别访问.
    • isaac - Isaac是ISAAC的elixir模块：快速加密随机数生成器.
    • jumper - 在Elixir中跳转一致的哈希实现（没有NIF）.
    • key2value - Erlang双向设置关联映射.
    • lfsr - Elixir实现二进制Galois线性反馈移位寄存器.
    • loom - 支持δ-CRDT的CRDT库.
    • luhn - Elixir中的Luhn算法.
    • lz4 - 用于Erlang的LZ4绑定，用于快速数据压缩.
    • machinery - 一般的结构状态机库，它与Phoenix开箱即用.
    • mason - Coerce映射到结构. 这很有用，例如，当您连接REST API并希望从响应中创建结构时.
    • matrex - 使用CBLAS进行C实现的Elixir / Erlang超快速矩阵库.
    • merkle_tree - Elixir中的Merkle哈希树实现.
    • minmaxlist - Elixir库扩展Enum.min_by / 2，Enum.max_by / 2和Enum.min_max_by / 2以返回结果列表而不是一个.
    • mmath - 用于对二进制文件中的数字“数组”执行数学运算的库.
    • monad - Haskell激发了Elixir时尚语法中的monad.
    • monadex - 使用monad升级Elixir管道.
    • murmur - 非加密哈希Murmur3的纯Elixir实现.
    • natural_sort - 字符串列表的Elixir自然排序实现.
    • navigation_tree - 带有助手的导航树表示，用于生成HTML.
    • parallel_stream - Elixir的并行流实现.
    • paratize - Elixir库提供一些方便的并行处理（执行）工具，支持配置工作人员数量和超时.
    • parex - Parallel Execute (Parex) is an Elixir module for executing multiple (slow) processes in parallel.
    • qex - 包装：queue，改进的API和Inspect，Collectable和Enumerable协议实现.
    • ratio - 添加Rational Numbers并允许它们用于常见的算术运算. 还支持Floats和Rational Numbers之间的转换.
    • red_black_tree - Elixir中的红黑树实现.
    • remodel - 用于转换地图结构的Elixir presenter包.
    • rendezvous - 在Elixir中实现Rendezvous或Highest Random Weight（HRW）哈希算法.
    • rock - ROCK的Elixir实现：分类属性的鲁棒聚类算法.
    • sfmt - 针对Erlang的面向SIMD的Fast Mersenne Twister（SFMT）.
    • simhash - 使用Siphash和N-gram实现Simhash.
    • sleeplocks - 适用于Elixir / Erlang的BEAM友好型自旋锁.
    • sorted_set - Elixir的分类集.
    • spacesaving - 使用“节省空间”算法对不同元素估计进行流计数.
    • structurez - Elixir数据结构的游乐场.
    • supermemo - Elixir的实施 Supermemo 2 algorithm.
    • tfidf - 术语频率 - 逆文档频率的Elixir实现.
    • the_fuzz - 模糊字符串匹配算法实现.
    • tinymt - Erlang的Tiny Mersenne Twister（TinyMT）.
    • trie - Erlang Trie实施.
    • witchcraft - Elixir的常见代数结构和函数.
    • zipper_tree - Variadic arity树，带有Elixir拉链.
