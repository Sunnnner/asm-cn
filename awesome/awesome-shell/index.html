    Shells¶

    选择你的基础外壳.

    • bash - GNU Project的shell（Bourne Again SHell）
    • elvish - 友好，富有表现力的shell功能，如匿名功能和数据结构
    • fish - Smart and user-friendly command line shell
    • osh - 兼容Bash，使用称为Oil的新/现代Unix shell语言
    • shenv - 简单的shell版本管理
    • xiki - 使shell控制台更加友好和强大
    • xonsh - Python-ish，BASHwards外观的shell语言和命令提示符
    • zsh - 强大的shell脚本语言

    System Utilities¶

    与操作系统相关的工具，包括系统管理，系统调试以及文件和流程管理.

    • atop - ASCII全屏性能监视器，能够报告所有进程的活动
    • catcli - 脱机数据的命令行目录工具
    • progress - 用于显示cp，rm，dd等进度的Linux工具...
    • stronghold - 从终端轻松配置MacOS安全设置.
    • glances - 浏览您的系统
    • goaccess - GoAccess是一个实时网络日志分析器和交互式查看器，可在 * nix系统中的终端中运行.
    • hblock - 基于主机文件的广告拦截器
    • histstat - netstat的历史
    • htop - 基于ncurses的交互式流程查看器，旨在成为更好的“顶级”
    • lnav - 小规模的高级日志文件查看器
    • logdissect - 用于分析日志文件和其他数据的CLI实用程序和Python API.
    • ls++ - 类固醇上的彩色ls
    • lsp - 改进的ls，文件描述采用简明语言和智能文件分组
    • mtr - 单个网络诊断工具中“traceroute”和“ping”程序的功能.
    • ncdu - NCurses磁盘使用情况
    • powertop - 电池/电源使用和设备统计监控命令行工具，带有调整选项.
    • procdog - 对服务器等长期进程的轻量级命令行控制
    • quick-secure - 快速安全和强化UNIX / Linux系统
    • rng - 将文件或标准输出的行范围复制到标准输出.
    • wifi-wand - 用于在MacOS上管理WiFi的Ruby命令行应用程序（通过gem install wifi-wand安装）
    • xiringuito - 基于SSH的“poors for VPN”

    Downloading and Serving¶

    用shell脚本编写的自托管，轻量级服务器和网络工具.

    • aria2 - aria2是一个轻量级的多协议和多源，跨平台下载实用程序，在命令行中运行. 它支持HTTP / HTTPS，FTP，BitTorrent和Metalink
    • balls - Bash on Balls
    • bashttpd - 用Bash编写的Web服务器
    • bitpocket - “DIY Dropbox”或“双向目录（r）同步并正确删除”
    • Dropbox-Uploader - Dropbox Uploader是一个Bash脚本，可用于从Dropbox上传，下载，列出或删除文件
    • httpie - HTTPie是一个命令行HTTP客户端，一个用户友好的cURL替代品
    • HTTPLab - 交互式Web服务器，允许您检查HTTP请求和伪造响应.
    • ngincat - 使用netcat的Tiny Bash HTTP服务器
    • resty - 可以在管道中使用的小命令行REST客户端
    • shell2http - 用于执行shell命令的HTTP服务器. 专为开发，原型设计或远程控制而设计
    • youtube-dl - 从YouTube.com和其他视频网站下载视频的小型命令行程序

    Multimedia and File Formats¶

    处理视频和音频文件的工具.

    • Android-Kitchen - 用于Android ROM定制的基于文本的厨房. 使用shell脚本并与Cygwin / OS X / Linux一起使用
    • Beets - 音乐库管理器和MusicBrainz标记器
    • cmus - 跨平台cli音频播放器.
    • gifgen - 简单的高品质GIF编码
    • image-scraper - 一个很酷的命令行图像抓取器，具有很多功能.
    • imgp - 快速批量图像缩放器和旋转器
    • jq - 用于json数据的sed. 您可以使用它来切片和过滤以及映射和转换结构化数据
    • korkut - 在命令行进行快速简单的图像处理.
    • mpv - 允许您在shell和GUI中播放大多数音频和视频格式（使用ASCII字符）.
    • nehm - 控制台工具，下载，设置IDv3标签并添加到您的iTunes（如果您使用它）您的SoundCloud喜欢方便的方式
    • PiCAST - PiCAST将您的35美元Raspberry Pi转变为类似设备的Chromecast
    • sejda - PDF文档的命令行操作（拆分，合并，旋转，转换为jpg，提取文本等）
    • xidel - 使用（图灵完备）XPath和XQuery过滤，映射和创建HTML / XML / JSON数据的Cli工具.
    • xmlstarlet - 用于命令行XML格式化，过滤和操作的旧但强大的工具.
    • fx - 匿名JavaScript函数的命令行JSON处理工具
