    Awesome First PR Opportunities ¶

    一系列令人敬畏的初学者友好项目.

    灵感来自 First Timers Only 博客文章和 YourFirstPR 项目.

    如果您是一名开源项目维护者，请将“first-timers-only”（或类似）标签添加到您的项目中并在此处列出，以便人们可以找到它.

    如果您希望贡献，请浏览此列表，请查看 first-timers-only labelled open issues on Github，并按照 @first_tmrs_only 在Twitter上，当创建一个新的“first-timers-only”问题时会收到通知.

    如果您不是程序员，但想贡献，请查看 Awesome for non-programmers 名单.
