    微服务

    Awesome Microservices ¶

    微服务架构相关原理和技术的精选列表.

    Platforms¶

    • Ballerina - 云原生编程语言.
    • Cocaine - 云平台，使您能够构建自己的PaaS云.
    • Deis - 公共云和私有云的开源应用程序平台.
    • Fabric8 - 基于Docker，Kubernetes和Jenkins的开源微服务平台. 使创建，编辑，部署和管理微服务变得更加容易，并且更快.
    • Hook.io - 微服务和webhook托管的开源提供商.
    • [Lightbend！[c]]（https://www.lightbend.com/） - 用于在JVM上构建可扩展的反应系统的平台.
    • Mantl - 快速部署全球分布式服务的平台.
    • Netflix OSS - Netflix开源软件生态系统.
    • OpenWhisk - Serverless, open source cloud platform that executes functions in response to events at any scale.
    • STUPS - Zalando提供的一套工具和组件，可为AWS上的多个自治团队提供方便且符合审计的PaaS.
    • [VAMP！[c]]（http://vamp.io/） - 轻松，轻松地构建，部署和管理微服务.

    Frameworks / Runtimes¶

    • Akka - 用于在JVM上构建高度并发，分布式和弹性的消息驱动应用程序的工具包和运行时.
    • Baratine - 用于构建松散耦合的POJO微服务网络的平台.
    • Erlang/OTP - 用于构建具有高可用性要求的大规模可扩展软实时系统的编程语言.
    • Finagle - 用于JVM的可扩展RPC系统，用于构建高并发服务器.
    • Flogo - Flogo是一个开源框架，用于简化构建高效和现代无服务器功能和边缘微服务.
    • GPars - JVM的并发和并行框架.
    • Grenache - 基于Bittorent-DHT的微服务框架，支持多个传输上的REQ / REP和PUB / SUB模式.
    • Ice - Comprehensive RPC framework with support for C++, C#, Java, JavaScript, Python, and more.
    • Lagom - JVM的反应性微服务.
    • Light Java - 快速，轻量级和高效的微服务框架.
    • Microserver - Java 8本机，零配置，基于标准，战斗强化库，以运行Java REST微服务.
    • Orbit - JVM语言的现代框架，可以更轻松地构建和维护分布式和可扩展的在线服务.
    • Quasar - JVM的光纤，通道和演员.
    • ScaleCube - 用于为JVM构建反应式微服务的工具包：低延迟，高吞吐量，可扩展和弹性.
    • Vert.X - 用于在JVM上构建响应式应用程序的工具包.
    • Vert.X Toolbox - 一组Vert.x组件，用于构建反应式微服务应用程序.
    • Wangle - 一个框架，以一致，模块化和可组合的方式提供一组公共客户端/服务器抽象，用于构建服务.
    • WildFly Swarm - 以足够的Java EE运行时开始的微服务，基于 Eclipse MicroProfile.

    Service Toolkits¶

    Polyglot¶

    • Apex - 用于部署和管理AWS Lambda函数的工具. 使用Lambda尚不支持的语言的填充程序，您可以使用开箱即用的Golang.
    • CoAP - 受限制的应用程序协议实现.
    • GRPC - A high performance, open source, general RPC framework that puts mobile and HTTP/2 first. Libraries in C, C++, Java, Go, Node.js, Python, Ruby, Objective-C, PHP and C#.
    • Hprose - 一个非常新的RPC库，现在支持25种以上的语言.

    C¶

    • Kore - 易于使用的Web应用程序框架，用于在C中编写可伸缩的Web API.
    • Libasyncd - 用于C的可嵌入的基于事件的异步HTTP服务器库
    • Libslack - 提供面向通用代理的编程模型，锁定策略的运行时选择，使编写守护进程变得微不足道的功能，以及简化网络服务器和客户端的实现，等等.
    • Lwan - 高性能和可扩展的Web服务器.
    • Onion - 用于创建简单HTTP服务器和Web应用程序的C库.
    • RIBS2 - 图书馆，允许建立高性能的互联网服务系统.

    C++¶

    • AnyRPC - 提供一个通用系统来处理许多不同的远程过程调用标准，包括：JSON-RPC，XML-RPC，MessagePack-RPC.
    • Cap’n Proto RPC - Cap'n Proto C ++ RPC实现.
    • C++ Micro Services - 类似OSGi的C ++动态模块系统和服务注册表.
    • Enduro/X - 基于XATMI的GNU / Linux服务框架.
    • Pion - 用于构建轻量级HTTP接口的C ++框架.
    • Pistache - 用C ++编写的高性能REST工具包.
    • Poco - 用于构建基于网络的应用程序和服务器的C ++类库.
    • Restbed - 为C ++ 11应用程序带来异步RESTful功能.
    • Served - 用于构建高性能RESTful Web服务器的C ++库.
    • ULib - 用于编写C ++应用程序的高度优化的类框架.

    CSharp¶

    • Akka.NET - 用于在.NET和Mono上构建高度并发，分布式和容错的事件驱动应用程序的工具包和运行时.
    • Microdot - 开源.NET微服务框架.
    • Nancy - 轻量级Web框架.
    • Orleans - 提供构建分布式高规模计算应用程序的简单方法，无需学习和应用复杂的并发或其他扩展模式.

    D¶

    • Vibe.d - 用D语言写的不会妨碍你的异步I / O.

    Erlang VM¶

    Elixir¶

    • Phoenix - 构建HTML5应用程序，API后端和分布式系统的框架.
    • Plug - Web应用程序之间可组合模块的规范和便利.

    Erlang¶

    • Cowboy - Small, fast, modular HTTP server written in Erlang.
    • Mochiweb - 用于构建轻量级HTTP服务器的Erlang库.

    Go¶

    • Echo - Go的快速和不受欢迎的HTTP服务器框架. 比其他部分快10倍.
    • Gin - Gin是一个用Go（Golang）编写的HTTP Web框架. 它具有类似Martini的API，具有更好的性能 - 速度提高了40倍.
    • Gizmo - Microservices toolkit.
    • Goa - Go中基于设计的HTTP微服务.
    • Gocraft - 用于构建Web应用程序的工具包. 包括路由，中间件堆栈，日志记录和监控.
    • Goji - Go的简约灵活的请求多路复用器.
    • Go kit - 用于现代企业中微服务的分布式编程工具包.
    • Go-micro - 可插拔的RPC微服务框架.
    • Gopencils - 使用Go轻松使用REST API.
    • Gorilla - Go编程语言的Web工具包.
    • Iris - Go的快速，简单和高效的微型Web框架.
    • Kite - Go中的微服务框架.
    • KrakenD - 使用中间件构建超高性能API网关的框架.
    • Libchan - 超轻量级网络库，使网络服务的通信方式与goroutines使用通道进行通信的方式相同.
    • Martini - Martini是一个功能强大的软件包，用于在Golang中快速编写模块化Web应用程序/服务.
    • Micro - Go中的微服务工具链.
    • Nano - 简约的，与传输无关且对测试友好的微服务框架.
    • Negroni - Golang的惯用HTTP中间件.
    • Neutrino - 实时/ REST后端服务.
    • RPCX - 基于.NET / RPC的分布式RPC服务框架，如阿里巴巴Dubbo和微博莫坦.
    • Sleepy - REST for go.
    • Vamp-router - 服务路由，负载平衡和过滤应用程序.

    Haskell¶

    • Scotty - 使用WAI和Warp，受Ruby的Sinatra启发的微型Web框架.
    • Servant - 类型级Web DSL.
    • Yesod - Haskell RESTful Web框架.

    Java VM¶

    Clojure¶

    • Compojure - Ring / Clojure的简洁路由库.
    • Duct - 在Clojure中构建Web应用程序的最小框架，强调简单性.
    • Friboo - 用于在Clojure中编写微服务的实用程序库，支持Swagger和OAuth.
    • Liberator - 帮助您将数据公开为资源的库，同时自动遵守HTTP规范的所有相关要求.
    • Modularity - JUXT基于Clojure的模块化系统.
    • System - 建立在Stuart Sierra的组件库之上，提供一组现成组件.
    • Tesla - 一些Otto.de的Clojure微服务的共同基础.

    Java¶

    • Airlift - 用Java构建REST服务的框架.
    • Disruptor - 高性能的线程间消息传递库.
    • Dropwizard - 用于开发操作友好，高性能，RESTful Web服务的Java框架.
    • Dubbo - 由阿里巴巴开源的高性能，基于Java的RPC框架.
    • HTTP Remoting - 用于定义和创建基于Feign或Retrofit作为客户端的RESTish / RPC服务器和客户端的库以及具有JAX-RS服务定义作为服务器的Dropwizard / Jersey.
    • Ja-micro - 用于构建微服务的轻量级Java框架（与go-micro兼容）.
    • Jersey - Java中的RESTful服务. JAX-RS参考实现.
    • MSF4J - 高吞吐量和低内存占用Java微服务框架.
    • QBit - 用于构建微服务的反应式编程库.
    • Ratpack - 一组Java库，可以实现快速，高效，可演化和经过良好测试的HTTP应用程序. 提供了对Groovy语言的特定支持.
    • Restlet - 帮助Java开发人员构建遵循REST架构风格的Web API.
    • Spark - 用于在Java 8中以最小的努力创建Web应用程序的微框架.
    • Spring Boot - 使创建独立的，生产级的基于Spring的应用程序变得容易.

    Scala¶

    • Akka HTTP - 用于在Scala和Akka之上构建基于REST / HTTP的集成层的开源工具包.
    • Colossus - Scala的I / O和微服务库.
    • Finatra - 在Twitter-Server和Finagle上构建的快速，可测试的Scala HTTP服务.
    • Http4s - HTTP的最小，惯用的Scala接口
