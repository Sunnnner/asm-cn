    Mastodon

    Awesome Mastodon ¶

    >与乳齿象有关的东西的精选清单！

    Mastodon是世界上最大的免费，开源，分散的微博网络. 见 Official 和 Documentation 类别以了解有关网络的更多信息.

    Clients¶

    • Tusky - Android客户端.
    • Twidere - 适用于Twitter，GNU Social和Mastodon的Android应用.
    • Tooty - 实验性多帐户Mastodon Web客户端（Elm）.
    • toot - Mastodon CLI客户端（Python）.
    • madonctl - Mastodon CLI客户端（Go）.
    • naumanni - 专为Mastodon设计的Web用户界面.
    • Tooter - SailfishOS的原生客户端.
    • Tootdon - 适用于iOS和Android的全功能客户端.
    • Mastalab - Android客户端.
    • Pinafore - Mastodon的替代Web客户端，专注于速度和简单性.
    • Brutaldon - Brutaldon是Mastodon的野兽派Web 1.0界面.
    • Halcyon - Mastodon和Pleroma的替代网络客户端，具有类似Twitter的界面.
    • Whalebird - 适用于Mac OS的Mastodon客户端.
    • Planiverse - Mastodon的极简主义，无JS Web客户端.

    Tools¶

    • Mastodon instances - Mastodon实例列表.
    • Mastodon bridge - 在联合Mastodon网络中找到您的Twitter好友.
    • unmung.com/mastoview - 预览任何实例的本地或联合时间轴.
    • Toot scheduler - 现在安排，稍后嘟..
    • Mastodon Network Monitoring (MNM) - 跟踪和显示有关Mastodon网络的可浏览统计数据.
    • Last - 在提供Atom订阅源和epub（Perl）的网页上聚合toots.
    • Mastodon Tags Explorer - 探索Mastodon网络目前的趋势.
    • Forget - 在用户定义的时间段后删除toots（Python source code).
    • Mastodon Toot Bookmarklet - Bookmarklet to toot当前页面（source code)

    User styles¶

    • Variable width - 使用浏览器的宽度调整Mastodon比例.
    • Narrow drawer - 使用可变宽度样式，使抽屉更窄，其他列相应缩放.

    User scripts¶

    • NSFW Remover - 自动显示NSFW图像.
    • Customizable interface - 在网格上移动和调整列的大小.
    • BirdSite - 浏览器扩展，用于将Mastodon toots交叉发布到Twitter.

    Bots¶

    • feed2toot - 自动解析RSS提要，识别新帖子并在Mastodon（Python）上发布.
    • usercount - 将用户统计信息发布到Mastodon（Python）的Bot.
    • autofollow - 用于Mastodon（Python）的Autofollow bot.
    • hnbot - 用100多点（Python）发布黑客新闻故事.
    • translator - 使用@ translator @ toot.works [langcode]将任何toot翻译成所需的语言.
    • @TrendingBot@mastodon.social - 向您展示Mastodon的趋势.
    • Remindr - 从资源列表（Python）自动向Mastodon和Twitter发送提醒.
    • News Bot - 镜像Mastodon（ClojureScript）上的Twitter帐户，源代码可用 GitHub.
    • Welcome Bot - 自动向新用户发送欢迎DM（Python）.
