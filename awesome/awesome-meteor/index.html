    Collections¶

    收藏家和收藏品的费用

    • node-simple-schema - 一个JavaScript模式验证包，支持直接验证MongoDB更新修饰符对象
    • aldeed:collection2 - 自动验证客户端和服务器上的插入和更新操作.
    • dburles:collection-helpers - 使用您定义的帮助程序转换集合.
    • matb33:collection-hooks - 使用insert / update / remove / find / findOne的挂钩之前/之后扩展Mongo.Collection.
    • reywood:publish-composite - 使用反应式连接从各种集合中发布一组相关文档
    • jagi:astronomy - Meteor的模型层

    REST¶

    对Meteor 的REST支持

    • simple:rest - 自动使您的Meteor应用程序可通过HTTP和DDP访问.
    • nimble:restivus - 以极其轻松的方式为您的Meteor应用程序创建REST端点.

    Forms and Templates¶

    模板助手

    • aldeed:autoform - UI组件和帮助程序，可以轻松创建具有自动插入和更新事件的基本表单，以及自动反应验证.
    • aldeed:template-extension - Meteor包：替换已定义的模板，从其他模板继承帮助程序和事件.
    • kadira:blaze-layout - Blaze布局管理器（适用于Meteor FlowRouter）
    • dispatch:scrollview - 流星的高性能无限滚动视图
    • themeteorites:blaze-magic-events - 将事件处理程序绑定到Meteor的Blaze的html元素的新方法
    • manuel:viewmodel - 流星MVVM
    • webix:webix - Meteor.js - Webix UI集成
    • uniforms - 一堆React组件和帮助器，可以轻松生成和验证表单. Seamlessly integrate with aldeed:simple-schema and simpl-schema.

    Users and Authentication¶

    处理用户和身份验证的工具

    • accounts-base - Meteor的用户帐户系统.
    • alanning:roles - 角色支持内置帐户包.

    Payments¶

    处理Meteor付款的工具

    • Stripe - Stripe的支付网关
    • PayPal - PayPal的支付网关
    • BrainTree - BrainTree的支付网关
    • Instamojo - Instamojo的支付网关

    Deployment¶

    部署和维护Meteor应用程序的工具

    • When a Meteor finally hits production - 关于部署Meteor应用程序的博客文章
    • BulletProof Meteor
    • meteorhacks:kadira - 流星的性能监测.
    • meteor-up - 流星部署.
    • davidyaha:collection2-migrations - 在Meteor上使用collection2和简单模式进行自动数据库迁移
    • percolate:migrations - Meteor的简单迁移系统
    • meteorhacks:fast-render - Render you app even before the DDP connection is live
    • staringatlights:fast-render - 快速渲染的活动分支
    • meteorhacks:cluster - Meteor的集群解决方案，具有负载平衡和服务发现功能
    • yogiben:admin - 完整的管理仪表板解决方案
    • demeteorizer - 将Meteor应用程序转换为“标准”Node.js应用程序
    • houston:admin - Meteor的零配置，Django Admin式管理员
    • pm2-meteor - 使用PM2部署，扩展和运行Meteor应用程序的最简单方法.
    • kschingiz:meteor-elastic-apm - 基于弹性APM的流星性能监测

    Routers¶

    • Blaze的路由器*

    • iron:router - 专为Meteor设计的客户端和服务器端路由器.

    • kadira:flow-router - Meteor的客户端路由器.
    • ostrio:flow-router-extra - 仔细扩展flow-router包. 支持最新Meteor版本的最新版本.
    • meteorhacks:picker - Meteor的服务器端路由器.
    • msavin:parrot - 专为使用Meteor构建SPA而设计的Web路由器

    Offline¶

    • Meteor离线支持工具*

    • ground:db - GroundDB是一个薄层，提供Meteor离线数据库和方法.

    Testing¶

    测试工具

    • Meteor Testing Manual
    • mike:mocha - 在Meteor框架内运行mocha测试.
    • xolvio:chimp - 测试如此简单，灵长类动物可以做到！ 支持摩卡，黄瓜，茉莉和柴.
    • velocity:html-reporter - 流星速度测试框架的HTML记者.
    • Gagarin - 您的流星应用程序的另一个测试框架.

    SEO¶

    搜索引擎优化工具

    • ostrio:spiderable-middleware - 在ES6（ECMAScript2015）的支持下预呈现（ aka Spiderable ） - Meteor应用程序完全被搜索引擎抓取.

    Files¶

    处理Meteor中的文件

    • Meteor-CollectionFS - Meteor基于web的文件系统处理和下载.
    • ostrio:files - 通过DDP，HTTP和WebRTC / DC上传文件. 到Meteor服务器FS，AWS，GridFS，DropBox或Google Drive. 快速，安全和强大.
    • netanelgilad:excel - 解析和生成excel文件（xlsx，xls）.

    Search, sort and paginate¶

    搜索，排序和分页相关工具

    • tmeasday:publish-counts - 实时发布光标的计数.
    • percolate:find-from-publication - 允许查找给定出版物已发布的所有文档.
    • meteorhacks:search-source - 搜索的反应数据源
    • matteodem:easy-search - 使用Blaze组件轻松搜索（+弹性搜索支持）
    • alethes:pages - 开箱即用的Meteor分页
    • Discover Meteor Blog about pagination

    Mobile¶

    移动开发

    • meteoric:ionic - Meteor的离子组件.
    • driftyco:ionic - Meteor官方Ionic支持.
    • raix:push - 推送cordova（ios，android）浏览器（Chrome，Safari，Firefox）的通知.
    • martijnwalraven:meteor-ios - 通过DDP将本机iOS应用程序与Meteor平台集成.
    • delight-im/Android-DDP - 适用于Android的客户端的DDP.
    • okland:accounts-phone - 基于Meteor移动电话号码的登录服务.
    • okland:camera-ui - 用于使用用户界面拍照的Meteor包，桌面和移动设备上的一个功能调用. 允许在手机上选择相机到photoLibrary.
    • percolatestudio/cordova-plugin-safe-reload - 在破碎的流星热码推送后观看和恢复的Cordova插件.

    Data Visualization¶

    流星中的数据可视化：图表，地图，表格等

    Analytics¶

    Cron Jobs¶

    *流星中的Cron Jobs *

    • percolate:synced-cron - Meteor的Cron系统. 它支持在多个进程之间同步作业.
    • vsivsi:job-collection - Meteor的持久且反应性的作业队列，支持可在任何地方运行的分布式工作人员.
    • ostrio:cron-jobs - 具有与本机setTimeout和setInterval方法类似的API的包，但在所有正在运行的Meteor（NodeJS）实例之间同步.
