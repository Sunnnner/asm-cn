    Awesome Robot Operating System 2 (ROS 2) ¶

    >精选的机器人操作系统2.0版（ROS 2）资源和库的精选列表.

    机器人操作系统2（ROS 2）是一组软件库和工具，可帮助您构建机器人应用程序. 从驱动程序到最先进的算法，以及强大的开发人员工具，ROS 2拥有您下一个机器人项目所需的功能. 而且它都是开源的.

    Packages¶

    Demostrations¶

    • adlink_ddsbot - 基于ROS 2.0 / 1.0的机器人群体架构（opensplice DDS）.
    • adlink_neuronbot - 用于人类跟踪和群体的ROS2 / DDS机器人pkg.
    • turtlebot3 - 基于ROS2的TurtleBot3演示，包括Bringup，Teleop和Cartographer.

    Examples¶

    • turtlebot2_demo - 使用ROS 2的TurtleBot 2演示.
    • examples/rclcpp - C ++示例.
    • examples/rclpy - Python示例.
    • rcljava_examples - 包含如何使用rcljava API的示例的包.
    • ros2_talker_android, ros2_listener_android - 用于ROS2 Java绑定的示例Android应用程序.

    Benchmarking¶

    • ros2_benchmarking - ROS2基准测试框架. ROS2通信特性可以快速且自动地在多个轴上进行评估.
    • performance_test - 测试ROS 2，FastRTPS和Connext DDS Micro等各种通信方式的性能和延迟.

    Containerization¶

    • docker-ros2-ospl-ce - 用于构建ROS2 + OpenSplice CE容器的dockerfile.
    • ros2_java_docker - 用于使用OpenJDK和Android构建ros2_java的Dockerfiles.
    • microROS/docker - 与Docker相关的材料，用于设置，配置和开发微型ROS硬件.

    Ecosystem¶

    • rosbag2 - ROS2原生rosbag.
    • rviz - 3D Robot Visualizer.
    • urdfdom - URDF（U-Robot描述格式）库，提供核心数据结构和简单的XML解析器
    • urdfdom_headers - URDF解析器的标头.
    • ros2cli - ROS 2命令行工具.
    • orocos_kinematics_dynamics - Orocos运动学和动力学C ++库.
    • pydds - 适用于Vortex Lite和OpenSplice的简单DDS Python API.

    Application layer¶

    • Apex.Autonomy - Apex.Autonomy提供自治算法作为单独的构建块，并与Autoware.Auto兼容.
    • Autoware.Auto - Autoware.Auto提供基于ROS 2的开源软件堆栈，用于自驱动技术.
    • ros2_control - ros2_control是关于如何在机器人控制（ros2_controllers）环境中详细阐述和使用ROS 2中的新功能的概念证明.
    • ros2_controllers - ros_controllers的描述.
    • geometry2 - 一组用于跟踪坐标变换的ROS包.
    • cartographer - 跨多个平台和传感器配置的2D和3D实时同步定位和映射（SLAM）.
    • vision_opencv - 用于连接ROS2和OpenCV的软件包.
    • teleop_twist_keyboard - Generic Keyboard Teleop for ROS2.
    • teleop_twist_joy - 用于扭曲机器人的简单操纵杆遥控器.
    • navigation - ROS2导航堆栈.
    • diagnostics - ROS2原始ROS1诊断的分叉版本（仅限诊断_updater）.
    • robot_state_publisher - 原始ROS Robot State Publisher的分叉版本，所有修改都在ROS2生态系统中编译.
    • common_interfaces - 一组包含公共接口文件（.msg和.srv）的包.
    • ros2_object_map - “SLAM时在地图上标记对象标记”.
    • ros2_object_analytics - 对象分析（OA）是用于实时对象检测，定位和跟踪的ROS2包装器.
    • ros2_intel_movidius_ncs - 用于Movidius™神经计算棒（NCS）神经元计算API的ROS2包装.
    • ros2_moving_object - 根据Object Analytics“ros2_object_analytics”生成的消息寻址移动对象.
    • ros2_openvino_toolkit - 用于OpenVINO™CV API的ROS2包装（人类视觉仿真）.
    • ros2_grasp_library - 可能是掌握图书馆:).
    • apriltag2_node - 用于AprilTag检测的ROS2节点.
    • ros2-web-bridge - 将浏览器桥接到ROS 2.0.
    • ros2_message_filters - ros2_message_filters根据过滤器需要满足的条件混合各种消息，并从ROS message_filters的ROS2移植中获取.
    • ros2-tensorflow - Tensorflow中用于计算机视觉任务的ROS2节点.
    • pid - ROS2的PID控制器.
    • system-modes - ROS 2和微ROS的系统模式.

    Middleware¶

    • Micro XRCE-DDS Agent - Micro XRCE-DDS Agent充当DDS网络和Micro XRCE-DDS客户端之间的服务器.
    • Micro XRCE-DDS Agent docker - 包含Micro XRCE-DDS代理的Docker镜像.
    • Micro XRCE-DDS Client - Micro XRCE-DDS实现客户端 - 服务器协议，以使资源受限的设备（客户端）能够参与DDS通信.
    • micro-ROS-Agent - 使用Micro XRCE-DDS Agent的ROS 2包.

    "System" bindings¶

    • rclandroid - ROS2的Android API.
    • rclnodejs - Node.js版本的ROS2.0客户端.
    • riot-ros2 - 该项目使ROS2能够使用RIOT操作系统在微控制器上运行.

    Driver layer¶

    • Autoware.IO - Autoware.IO提供异构硬件参考平台，支持将成员公司的解决方案集成到支持Autoware.Auto和Autoware.AI软件堆栈的平台上.
    • ros2_xmlrpc - 带有xmlrpc的ros2接口包，使用Sesto API与Sesto服务器通信.
    • cozmo_driver_ros2 - ROS2的非官方Anki Cozmo节点.
    • sphero_ros2 - ROS2 sphero驱动程序.
    • flock2 - DJI Tello无人机的ROS2驱动程序.
    • ros2_raspicam_node - Raspberry Pi相机的ROS2节点.
    • joystick_drivers - 操纵杆的ROS2驱动程序.
    • joystick_drivers_from_scratch - ROS 2的操纵杆驱动程序包.
    • joystick_ros2 - ROS2的操纵杆驱动程序，支持所有平台：Linux，macOS，Windows.
    • ros2_teleop_keyboard - 用于ROS2的Teleop Twist键盘.
    • ros_astra_camera - 用于Astra相机的ROS2包装.
    • ros2_usb_camera - ROS2通用USB相机驱动程序.
    • ros2_android_drivers - 为几个Android传感器收集ROS2驱动程序.
    • ros2_intel_realsense - 用于英特尔®实感™设备的ROS2包装器.
    • raspicam2_node - Raspberry Pi相机模块的ROS2节点.
    • HRIM - 机器人模块的标准接口.
    • FIROS2 - ROS2可集成工具，专注于ROS2和FIWARE之间的相互通信.
    • lino2_upper - ROS2上的Linorobot.
    • RysROS2 - 用于MiniRys机器人的ROS2软件堆栈.
    • px4_to_ros - ROS2 / ROS包用于与ROS通信PX4.
    • multiwii_ros2 - 用于MultiWii和Cleanflight飞行控制器的ROS2节点.
    • ydlidar_ros2 - 用于ydlidar的ROS2包装.
    • zed-ros2-wrapper - ZED SDK的ROS 2包装测试版.
    • ros2_denso_radar - 用于ROS2的丰田/雷克萨斯2015-2017 Denso Radar驱动程序.
    • sick_scan2 - 用于SICK TiM系列激光扫描仪（TiM551 / TiM561 / TiM571）的ROS2驱动器.
    • micro-ROS/hardware - 有关微ROS项目中使用和支持的硬件平台的信息和文档.

    Client libraries¶

    • rclada - Ada的ROS客户端库.
    • rclcpp - 用于C ++的ROS客户端库.
    • rclgo - 用于Go的ROS客户端库.
    • rclpy - 用于Python的ROS客户端库.
    • rcljava - 用于Java的ROS客户端库.
    • rclnodejs - Node.js的ROS客户端库.
    • rclobjc - 用于Objective C的ROS客户端库（适用于iOS）.
    • rclc - 用于C的ROS客户端库.
    • ros2_rust - ROS2的Rust绑定.
    • ros2_dotnet - ROS2的.NET绑定.

    Client libraries common¶

    • rcl - 库，以支持特定语言的ROS客户端库的实现.
    • system_tests - 测试rclcpp和rclpy.
    • rcl_interfaces - ROS客户端库使用的消息和服务的存储库.

    IDL generators¶

    • rosidl_generator_java - 使用Java生成ROS接口.
    • rosidl_generator_objc - 在Objective C中生成ROS接口.
    • rosidl_generator_cpp - 在C ++中生成ROS接口.
    • rosidl_generator_c - 在C中生成ROS接口.
    • rosidl - 提供ROS IDL（.msg）定义和代码生成的包.
    • rosidl_dds - 为ROS接口生成DDS接口.

    RMW (ROS middleware)¶

    • rmw - 包含ROS中间件API.
    • rmw_connext_cpp - 使用C ++中的RTI Connext静态代码生成实现ROS中间件接口.
    • rmw_fastrtps_cpp - 使用C ++中的eProsima FastRTPS静态代码生成实现ROS中间件接口.
    • rmw_dps - 使用英特尔的分布式发布和订阅实现ROS中间件（rmw）接口.
    • rmw_opensplice_cpp - 使用C ++中的PrismTech OpenSplice静态代码生成实现ROS中间件接口.
    • rmw_coredx - ROS2的CoreDX DDS集成层.
    • rmw_freertps - 使用freertps实现RMW.
    • rcutils - ROS 2中使用的通用C函数和数据结构.
    • freertps - 免费，便携，极简主义，正在进行的RTPS实施.
    • rmw_zhe - 在Zhe之上的ROS2 RMW实现.
    • rmw_cyclonedds - 用于Eclipse Cyclone DDS的ROS2 RMW层.

    DDS communication mechanism implementations¶

    • Connext DDS - 用于开发和集成IIoT系统的连接软件. ：heavy_dollar_sign：
    • Fast-RTPS - RTPS标准的实施（RTPS是DDS的有线互操作性协议）.
    • OpenSplice - 实施OMG DDS标准. ：heavy_dollar_sign：
    • CoreDX DDS - Twin Oaks Computing，Inc ..的实施：heavy_dollar_sign：
    • freertps - 免费，便携，极简主义，正在进行中的RTPS实施.
    • cdds - Cyclone DDS完全开放，正在接受成为Eclipse IoT的一部分.
    • Micro-XRCE-DDS) - XRCE DDS实现（由microROS支持）.

    Build system (Linux)¶

    • meta-ros2 - 用于OpenEmbedded Linux的ROS 2层.

    Build system (ROS2)¶

    • ci - ROS 2 CI基础设施.
    • ament_cmake_export_jars - 能够将Java归档导出到CMake中的ament构建系统中的下游包.
    • rmw_implementation_cmake - CMake功能，可以发现和枚举可用的实现.
    • rmw_implementation - 用于rmw实现的CMake基础结构和依赖项.

    Operating systems¶

    • NuttX - NuttX正式用于微型ROS的叉子.
    • RIOT - RIOT是一种实时多线程操作系统（......）实时功能，占用内存小，（......）API提供部分POSIX兼容性.
    • eMCOS - 适用于多核处理器的POSIX兼容实时操作系统，有望在未来支持AUTOSAR.

    Forks¶

    • Apex.OS - Apex.OS是ROS 2的一个分支，它已经做得非常强大和可靠，可以用于安全关键应用程序.
