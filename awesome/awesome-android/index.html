    Awesome Android¶

    About¶

    精彩的Android精选列表 libraries 和 resources . 对于一般的Java库，请看一下 awesome-java.

    Supported with ❤️ by Instabug¶

    > Instabug SDK允许Android开发人员直接从应用程序内部接收来自用户和测试人员的详细错误报告. 它附带屏幕截图，屏幕录制，网络请求以及每个错误报告的重现步骤. 这最大限度地缩短了调试时间，并帮助开发人员追踪奇怪的错误. Learn more.🚀

    How to Use¶

    对于需要在其应用上使用某项功能的人来说，Awesome-Android是一个了不起的列表，所以最好的使用方法是： - 只需按下命令+ F即可搜索关键字 - 浏览我们的内容菜单

    Libraries¶

    Charts¶

    • AChartEngine - Charting Engine.
    • EazeGraph - 图表和图表库.
    • WilliamChart - 具有良好运动功能的图表库.
    • HelloCharts - 图表和图形库，支持缩放，滚动和动画.
    • MPAndroidChart - 支持按手势缩放和拖动的Android图表和图形库.
    • ArcChartView - 绘制创意统计弧图.
    • AnyChart - 数据可视化库，交互式图表.

    Cloud Services¶

    • CloudRail - 统一API库：云存储，社交个人资料，付款，电子邮件，短信和兴趣点.

    Data binding¶

    • Anvil - 一个小型库，用于创建反应式UI组件，灵感来自React. 提供数据绑定和事件侦听器绑定，非常适合MVVM.
    • Data Binding Library - 官方Android数据绑定库，用于编写声明性布局，并最小化绑定应用程序逻辑和布局所需的粘合代码.

    Dependency Injection¶

    • Dagger 2 - Android和Java的快速依赖注入器.
    • Butter Knife - 查看Android的“注入”库.
    • ActivityStarter - Android库，提供更简单的方法来启动具有多个参数的活动.
    • AndroidAnnotations - 在编译时使用依赖注入的Java注释.
    • Toothpick - 基于范围树的Java依赖注入（DI）库.

    Android Services¶

    • Remoter - 使用普通java接口替代Android AIDL for Android远程IPC服务.
    • Service Connector - 将Android服务和回调绑定到字段和方法.

    Game Development¶

    • Libgdx - 跨平台游戏引擎和SDK. Open Source
    • Vuforia - 增强现实库.
    • Unity - 跨平台游戏创建系统.
    • Rajawali - Android OpenGL ES 2.0 / 3.0引擎
    • Cocos2d-x - 跨平台的2D游戏框架.
    • JustWeEngine - 一个简单的开源Android Native Game FrameWork.

    Security¶

    • libsignal-protocol-java - 在同步和异步消息传递环境中工作的棘轮前向保密协议.
    • Themis - 使用典型加密方案易于使用的多语言框架：静态数据，经过身份验证的数据交换，传输保护，身份验证等.

    GUI¶

    • Pull to refresh - v4支持库中提供了一个刷卡刷新布局.
    • Cardslib - 用于构建UI卡的Android库.
    • AndroidStaggeredGrid - 网格视图，支持具有不同大小的行的多个列.
    • Flow - 有助于将应用程序描述为适度独立屏幕集合的库.
    • SortableTableView - 包含简单TableView和高级SortableTableView的Android库，提供了大量定制可能性以满足所有需求.
    • MaterialProgressBar - 材料设计ProgressBar具有一致的外观.
    • AndroidFillableLoaders - 使用SVG路径的可填写进度视图. 用于创建有趣的应用徽标的不错选项.
    • NexusDialog - 允许您使用少量代码轻松快速地在Android中创建表单.
    • Snap RecyclerView Utils - 在不创建适配器的情况下填充单个或多个布局RecyclerView.
    • MultiSnapRecyclerView - 用于多次捕捉RecyclerView的Android库
    • SwipeableCard - 实施像StreetView这样的刷卡！
    • ElasticProgressBar - 美丽的装载栏.
    • EntryScreenManager - 简介/进入/演练/开始屏幕.
    • EasyIntro - 适用于Android项目的灵活，易用，一体化应用程序介绍库.
    • Material-Calendar-View - 与API 8+兼容的Material Design Calendar
    • CrunchyCalendar - 具有无限滚动，日期范围选择和颜色自定义的材质日历窗口小部件.
    • SectionedRecyclerViewAdapter - 一个适配器，允许将RecyclerView拆分为带有页眉和/或页脚的章节.
    • DragListView - 拖放以重新排列列表，网格或板中的项目.
    • Animated Expanding ListView - 动画扩展ListView在展开或折叠listview项目的内容时提供了一个奇特的动画.
    • TastyToast - 图标和颜色的祝酒词.
    • DotLoader - 带Dots的可自定义加载动画.
    • PodSlider - 符合材料设计规格的可自定义滑块小部件.
    • TapTargetView - 从材料设计指南中实现功能发现的点击目标.
    • ShowCaseView - ShowcaseView库旨在通过引人注目的平面叠加层向用户突出显示和展示应用程序的特定部分.
    • MaterialIntroScreen - 具有易于扩展的API的Material Intro Screen实现.
    • FloatingView - FloatingView可以使目标视图浮动在锚视图上方，并带有很酷的动画.
    • Timecon - 易于使用的动画时钟图标
    • Audiogram - 轻量级音频进度条
    • Bubbles for Android - Facebook喜欢聊天泡泡库
    • Litho (By Facebook) - 用于在Android上构建高效UI的声明性框架.
    • MultiViewAdapter - Recyclerview适配器库，用于创建可组合的视图持有者.
    • LGSnackbar - 一个易于使用和可自定义的原生Android Snackbar包装器，可在多个活动中保持可见.
    • ShimmerLayout - 针对Android应用程序的内存高效闪烁效果.
    • CircleProgressBar - 一个用于为Android创建循环进度条的简单库.
    • Easy-Signature-Android - 一个简单的ui库，提供可插入的签名视图.
    • Flashbar - 适用于Android的高度可定制，功能强大且易于使用的警报库.
    • YuanaItemSettingView - Android的可自定义项目设置视图.
    • Gradients - 精选渐变的精选集合.

    Paginate¶

    • NoPaginate - 简单的Android分页库

    ActionBar¶

    • ActionBarSherlock - 旧版Android版的ActionBar.
    • FadingActionBar - 在新的Play音乐应用中可以看到淡入淡出的动作条效果.

    Navigation¶

    • SlidingMenu - 用于创建具有滑入式菜单的应用程序的库.
    • SlidingTutorial - 简单的库，有助于创建真棒滑动Android应用程序教程.
    • PagerSlidingTabStrip - 用于在ViewPager的不同页面之间导航的交互式指示器.
    • Page View indicator - 支持水平滚动ViewPager.
    • RecyclerTabLayout - 使用RecyclerView实现的高效TabLayout库.
    • MaterialDrawer - 简单介绍材料设计导航抽屉.
    • Debug-Artist - 调试菜单，轻松实现泄漏，手术刀等.
    • Floating-Navigation-View - 一个简单的浮动操作按钮，显示锚定的导航视图.

    Animations¶

    • Rebound - Rebound是一个模拟spring动力学的Java库.
    • Android View Animations - 可爱的视图动画集合.
    • Android-Transition - 允许轻松创建对用户输入做出反应的视图转换.
    • Android-View-Actions - 轻松创建复杂的动画视图.
    • Swipper - 用于可滑动手势的Android库，用于控制音量，亮度和搜索.
    • Spotlight - Android库，用于点亮教程或演练等项目......

    Images¶

    • Crescento - 通过在图像视图下方添加曲线，探索材料设计的新风格.
    • android-crop - 用于裁剪图像的图书馆项目.
    • CircularImageView - 自定义圆形图像视图，同时保持最佳绘图性能.
    • Android-Image-Filter - 轻松应用图像过滤器的库项目.
    • Compressor - Compressor是一个轻量级且功能强大的Android图像压缩库.
    • ShapeImageView - 库以显示不同形状的图像.

    Inputs¶

    • FloatingLabel - FloatingLabel允许您创建一种类型的EditText. 没有Gradle或Maven支持.
    • MaterialEditText - 支持浮动标签，单行省略号，最大/最小字符，帮助文本和自定义颜色的错误文本.
    • Emojicon - 为您的应用添加表情符号
    • MaterialSearchBar - Android的Material Design搜索栏
    • InputMask - 基于模式的用户输入格式化器，解析器和验证器.
    • SweetPassword - 密码EditText，允许自定义切换按钮
    • VoiceOverlay - 获取用户语音权限并在可自定义UI中作为文本输入的叠加层.

    View Pagers¶

    • Material Dots Indicators - 查看寻呼机的三个物料点指示器样式.

    Loading Images¶

    • Picasso - 适用于Android的强大图像下载和缓存库.
    • Universal Image Loader - 异步，开箱即用的图像加载和缓存.
    • Glide - 适用于Android的图像加载和缓存库，专注于平滑滚动，由Google推荐.
    • Fresco - 用于管理图像及其使用内存的Android库.
