    真棒飞镖 ¶

    精选的Dart框架，库和软件的精选列表. 列表中的项目在Dart社区中得到积极维护，记录良好且受欢迎. 灵感来自于 awesome 名单.

    Contributing¶

    请快速浏览一下 contribution guidelines 第一. 如果您在此处看到不再维护或不适合的软件包或项目，请提交拉取请求以改进此文件. 谢谢你们 contributors ; 你摇滚！

    Client Web App Frameworks¶

    • AngularDart - AngularDart是一个用于构建移动和桌面Web应用程序的开发平台
    • Flutter - Flutter是一个构建高性能，跨平台移动应用程序的框架，允许为Android和iOS编写应用程序.
    • MDL/Dart - Dart的Material Design Lite是基于Google Material Design理念的Web开发人员组件框架.
    • OverReact - 用于构建静态类型的React UI组件的库.
    • VueDart - 使用Dart创建Vue Web应用程序，Dart是一种用于构建Web UI的渐进式，可渐进式采用的框架

    Server Frameworks¶

    • Jaguar - 为快速，简单和可扩展性而构建的服务器框架.
    • Angel - Angel是一个服务器端框架，专为全栈开发而设计，重点是代码共享，可扩展性和低学习曲线.
    • Aqueduct - Aqueduct是一个功能齐全的服务器端框架，具有ORM，数据库迁移工具，OAuth 2.0实现，自动OpenAPI规范生成和多线程支持.
    • Redstone - Redstone是Dart的服务器端，元数据驱动的微框架.
    • Start - Sinatra启发Web框架以提供静态文件，处理动态请求，websockets和创建JSON响应.
    • Shelf - Shelf可以轻松创建和组合Web服务器和Web服务器的一部分. *为Shelf写了很多包. 按照惯例，他们开始 shelf_
    • Vane - 具有内置服务器运行时环境和中间件系统的框架
    • Rikulo Stream - 具有请求路由，过滤，模板引擎，WebSocket，MVC设计模式和基于文件的静态资源的轻量级Web服务器

    Game Development¶

    • Flame - 极简主义的Flutter游戏引擎
    • StageXL - StageXL提供易于使用且完整的API（基于Flash API），可用于令人印象深刻的2D内容，如游戏和其他丰富的应用程序.
    • DartRocket - DartRocket是一个用Dart编写的HTML5游戏框架，它使用StageXL渲染引擎.
    • Pixi Dart - pixi.js渲染引擎的一个端口.
    • Ranger - 以HTML5 Canvas为中心的游戏引擎和场景图.

    Animation¶

    • Universal Tween Engine - Aurelien Ribbon创建的原始java Universal Tween Engine的端口.
    • Spine Dart - Esoteric Software Spine运行时的实现.

    Template¶

    • mustache4dart - 一个简单的Mustache实现.
    • jaded - 优秀的玉视图引擎的端口.

    Database¶

    • Postgres - PostgreSQL数据库驱动程序，它使用扩展的二进制协议来实现更高效和安全的查询.
    • SQLJockey - MySQL连接器.
    • PostgreSQL - PostgreSQL数据库驱动程序
    • dartabase_model - Serverside数据库对象使用MySQL / PGSQL进行简单数据操作的模型，无需编写SQL
    • dartabase_migration - Serverside数据库迁移，用于使用MySQL / PGSQL进行简单版本控制的数据库结构操作，而无需编写SQL

    Package Managers¶

    • Pub - Pub用于管理包.

    Utilities¶

    • Archive - 用于编码和解码各种存档和压缩格式的库.
    • built_collection - 通过构建器模式的不可变集合.
    • built_value - 不可变值类型，枚举类和序列化.
    • Frappe - A functional reactive programming library for Dart. Frappé extends the functionality of Dart's streams, and introduces new concepts like properties/signals.
    • Quiver - 一组实用程序库，可以更轻松，更方便地使用许多库，或者添加其他功能.
    • route_hierarchical - Route是Dart的客户端路由库，可帮助构建单页Web应用程序.

    Dependency Injection¶

    • DI - Angular的依赖注入框架
    • Dependencies - 简单的模块化依赖注入系统，不使用镜像.

    Parsers¶

    • html - 用于处理HTML文档的库. 以前称为html5lib.
    • markdown - 在客户端和服务器上将markdown分解为HTML.
    • PetitParser - PetitParser将来自无扫描解析，解析器组合器，解析表达式语法和packrat解析器的思想组合成模型语法和解析器，作为可动态重新配置的对象.
    • XML - 用于解析，遍历，查询和构建XML文档的轻量级库.
    • xmlstream - 基于流事件的XML Parser.
    • YAML - YAML的解析器.
    • Dart Tags - 用于解析ID3标签的库，用纯Dart编写.

    Validation¶

    • Constrain - 提供基于约束的验证库，其灵感来自Java Bean验证，但利用了Dart的卓越语言功能.
    • validator.dart - Dart的字符串验证和清理.

    ORM¶

    • Objectory - Objectory提供类型化的，经过检查的环境，用于建模，保存和查询MongoDb上持久存储的数据.

    Image¶

    • image - 使服务器和Web应用程序能够使用各种图像文件格式加载，操作和保存图像，包括PNG，JPEG，GIF，WebP，TIFF，TGA，PSD，PVR和OpenEXR.

    Testing¶

    • Guinness - Jasmine图书馆的一个端口.
    • test - 提供在Dart中编写和运行测试的标准方法.

    Tools¶

    • Observatory - 天文台是用于分析和调试Dart应用程序的工具.
    • dart2js - 将Dart代码编译为JavaScript
    • js2dart - 将Javascript代码编译为Dart
    • Stagehand - 一个项目脚手架生成器，灵感来自Web Starter Kit和Yeoman等工具.
    • Crossdart - 来自Pub的包的交叉引用源代码.
    • Crossdart Github Chrome Extension - 在Github上的Dart项目中添加“转到声明”和“查找用法”功能（在树视图和拉取请求中）.
    • gulp-dart - 使用dart2js将Dart代码编译为JavaScript的gulp插件.
    • dev_compiler - Dart to JavaScript编译器，旨在创建惯用的，可读的JavaScript输出.
    • json2dart - 给定一个json，它生成dart类来解析并生成给定结构的json.

    • Darrrt - Web应用程序代码实验室.

    IDEs, Editors, and Plugins¶

    • IntelliJ Plugin - 来自JetBrains的Dart插件，适用于WebStorm，IntelliJ IDEA，PhpStorm，PyCharm和RubyMine.
    • Sublime Text Package - Sublime Text 3 Dart Package

    • Vim Plugin - 在Vim中为Dart突出显示语法
    • Atom Plugin - Dart对Atom的支持.
    • DartPad - 在线轻量级编辑器.
    • Dart Code - Dart支持Visual Studio Code.
    • Module Linker - Chrome扩展程序，可在GitHub上添加指向模块导入语句的直接链接.

    Everything Else¶

    有许多令人敬畏的库被添加到 Pub 每时每刻. 如果您在此列表中找不到符合您需求的库，请继续在Pub上搜索它. 如果你最终找到一个很棒的库，我们会喜欢带有信息的拉取请求，这样其他人也可以发现它. 请务必阅读 contributing guidlines 第一.
