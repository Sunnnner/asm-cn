    jQuery

    Awesome jQuery¶

    精彩的精选列表 jQuery 插件，资源和其他闪亮的东西. 灵感来自GitHub上的'awesome- *'趋势.

    jQuery 是一个快速，小巧，功能丰富 JavaScript 图书馆. 它通过易于使用的API在大量浏览器中运行，使得HTML文档遍历和操作，事件处理，动画和Ajax变得更加简单. 通过多功能性和可扩展性的结合，jQuery改变了数百万人编写JavaScript的方式.

    Resources¶

    很棒的学习材料，以了解jQuery和JavaScript或学习更高级的东西. 一些友好的建议：在深入了解jQuery之前，学习JavaScript.

    • API documentation - 官方jQuery文档.
    • Code School - 学习jQuery 2.0的基本构建模块，并为有相关编程挑战的初学者享受新的视频教程.
    • Codecademy jQuery track - 初学者跟踪学习jQuery.
    • Learning jQuery - jQuery JavaScript库的提示，技巧和教程.
    • Style guides - jQuery的代码样式指南在为jQuery项目做贡献时.
    • Tuts+ - 有关jQuery的有用和深入的文章.
    • jQuery guide for teams - 提示编写一致的jQuery代码，易于使用和团队理解.

    Tutorials¶

    非常棒的免费教程和学习jQuery的课程.

    • jQuery patterns - 各种jQuery插件模式，用于启动插件开发.
    • Learning jQuery Design Patterns - jQuery插件设计模式
    • Learn jQuery - jQuery学习中心.
    • Snoopcode - jQuery教程.
    • Try jQuery - 学习jQuery的基本构建块.
    • Extending jQuery - 扩展jQuery的选择器功能.

    推荐并没有太多过时的jQuery书籍.

    • jQuery fundamentals - 使用jQuery解决您需要解决的常见问题，从而轻松完成工作.

    • jQuery.eraser - 用鼠标或触摸动作擦除图像.
    • jQuery Mapael - 基于raphael.js的插件，允许您显示动态矢量地图.

    • Nivo Slider - 美观且易于使用的图像滑块.

    Menus¶

    菜单动画，可用性增强和美化.

    • jQuery contextMenu - 上下文菜单的管理工具.

    • jQuery-menu-aim - 当用户光标瞄准特定下拉菜单项时触发事件.
    • mmenu - 具有滑动子菜单的应用程序外观相似的画布外菜单.
    • Multi-level push menu - 允许无限嵌套导航元素.
    • Slidebars - 用于画布外菜单和侧边栏的jQuery框架.
    • stickUp - 在滚动浏览器窗口时将元素粘贴到浏览器窗口的顶部，始终将其保持在视图中.
    • Superfish - 为现有的多级下拉菜单添加可用性增强功能.
    • Yamm - Bootstrap 3的另一个megamenu.

    Modals and Popups¶

    模态插件（对话框/弹出窗口），工具提示和类似的弹出窗口.

    • animatedModal.js - 使用CSS3过渡创建全屏模态的插件. 您可以使用animate.css中的过渡或创建自己的过渡.
    • Avgrund - 为弹出窗口和页面之间显示深度的新概念插入模态框和弹出窗口.
    • Bootstrap Modal - 扩展默认的Bootstrap Modal类. 响应，可堆叠和ajax.
    • iziModal - 使用jQuery优雅，响应灵活，轻量级的模态插件.
    • jBox - 功能强大且灵活的插件，可以处理所有模态窗口，工具提示和通知.
    • jQuery Modal - 显示模态窗口的简单轻量级方法.
    • jQuery Popup Overlay - 响应式和可访问的模态窗口和工具提示.
    • Lightbox - 用于在当前页面上叠加图像的JavaScript库.
    • Magnific Popup - 快速，轻便，响应迅速的灯箱插件.
    • SweetAlert - JavaScript的警报的美丽替代品.
    • tipso - 轻量级响应式jQuery Tooltip插件.
    • vex - A modern dialog library which is highly configurable and easy to style.

    Tables¶

    网格插件和插件，用于以表格形式创建和展示数据.

    • Bootgrid - 专为引导程序设计的网格控件.
    • Bootstrap table - 为bootstrap设计的强大的表控件.
    • DataTables - HTML表的高级交互控件.
    • FancyGrid - 具有图表集成和服务器通信的JavaScript网格库.
    • Filter Table - 实时搜索/过滤HTML表格.
    • FloatThead - 粘性标题. 支持响应，窗口和溢出滚动.
    • FooTable - 使HTML表响应.
    • Isotope - 过滤和排序神奇的布局.
    • HighchartTable - 将HTML数据表转换为Highcharts图的简单方法.
    • jqGrid - 网格插件.
    • jQuery treetable - 在表格中显示树状结构.
    • jsGrid - 轻量级网格插件.
    • Stackable.js - 在小屏幕上堆叠表格.
    • Stupid Table Sort - 一个愚蠢的小而简单的jQuery表分类器插件.
    • tableExport.jquery.plugin - 将HTML表导出为JSON，XML，CSV，TXT，SQL，Word，Excel，PNG或PDF.

    Time and Date¶

    用于处理，操纵或格式化时间和日期的插件.

    • jQuery Timer Plugin - 启动/停止/恢复/删除任何HTML元素内的漂亮计时器.
    • Timeago - 轻松支持自动更新模糊时间戳（例如“4分钟前”）.

    Translations¶

    国际化（i18n）和本地化（l10n）.

    • i18n properties - 轻量级插件，为JavaScript提供国际化 .properties 档
    • jquery.i18n - 基于jQuery的Javascript国际化库，可轻松实现Web应用程序的国际化
    • jquery.localize.js - 轻松实现静态网站的国际化

    Validation¶

    验证插件.

    • FormValidation - 表单字段验证器，专为Bootstrap，Foundation，Pure，Semantic UI，UIKit和其他框架而设计.
    • h5Validate - jQuery的HTML5表单验证插件.
    • jQuery Form Validator - 在保持HTML标记清除JavaScript代码的同时验证用户输入.
    • jquery-validation - 对现有表单进行插入验证，同时使各种自定义适合您的应用程序非常简单.
    • jQuery.validationEngine - 验证浏览器中的表单字段.

    Miscellaneous Plugins¶

    尚未在上述类别中排序的插件.

    • Boilerplate - jQuery插件开发的快速入门.
    • Interlink - 事件互连插件.
    • Sisyphus - 类似Gmail的客户端草稿和更多内容.
    • Slugify - 在键入页面标题时创建URL slug（如Django slugify（））.
    • Readmore.js - 一个轻量级的jQuery插件，用于通过“阅读更多”和“关闭”链接来折叠和扩展长文本块.
    • Yahoo! Emoticons - 雅虎！ 表情符号JQuery插件.
    • autoNumberic - autoNumeric是一个独立的库，为国际号码和货币提供实时的自定义格式.

    Mobile¶

    移动部分.

    • jQuery mobile - 使用HTML5构建移动Web应用程序的轻量级框架.
    • Interdimensional - 在您的网页的移动设备上进行空间滚动.
    • OhSnap!.js - 一个简单的jQuery / Zepto通知库，旨在用于移动应用程序.

    UI¶

    用户界面部分.

    • Dragula - 框架无关的拖放库，支持jQuery集成.
    • jQuery UI - 在jQuery之上构建的一组用户界面交互，效果，小部件和主题.
    • tabulous.js - 今日网站的标签
    • X-editable - 使用Twitter Bootstrap，jQuery UI或纯jQuery进行就地编辑.

    Testing¶

    单元测试，测试部分.

    • Qunit - 一个JavaScript单元测试框架，由jQuery，jQuery UI和jQuery移动项目使用.
    • Mockjax - 提供简单且极其灵活的界面，用于模拟或模拟ajax请求和响应.

    Server-side Integrations¶

    将客户端jQuery与服务器端技术集成的最佳实践.

    • Node.js：
      • Bower - 针对前端优化的有用的包管理器.
      • Cheerio - 专为服务器设计的核心jQuery的快速，灵活和精益实现.
    • PHP：
      • PHP front end setup without Node - 教程如何将BowerPHP，Mini-Asset和Robo.li组合到PHP前端资产构建管道中.
      • symfony-collection - jQuery插件，管理从Symfony2集合添加，删除和移动元素.
    • Python：
      • django-static-jquery - jQuery打包在一个方便的Django应用程序中，以加快新的应用程序和部署.
    • Ruby：
      • jquery-rails - 一个使用Rails自动使用jQuery的gem.
      • jquery-ui-rails - 用于Rails资产管道的jQuery UI.
