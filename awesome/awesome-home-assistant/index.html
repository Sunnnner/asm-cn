    Home Assistant

    Awesome Home Assistant¶


    https://awesome-ha.com

    Home Assistant是一个开源的家庭自动化系统，可以实现本地控制 隐私第一. 由世界各地的修补匠和DIY社区提供支持 爱好者. 非常适合在Raspberry Pi或本地服务器上运行.

    如果你想获得外观和感觉， 你应该看看 Home Assistant online demo.

    令人敬畏的家庭助理是一个很棒的精选列表 Home Assistant 资源. 其他软件，教程，自定义组件，Hassio附加组件， 定制Lovelace面板，烹饪书，示例设置等等.

    该列表分为几类. 这些类别中的链接没有 预先建立的订单; 订单是为了贡献. 如果你想贡献， 请阅读 guide.

    How to Use¶

    令人敬畏的家庭助理是一个很棒的列表，人们试图自动化每一个 他们家的方面. 自动化你的家是一个漫长，艰难，永远不会完成 通常涉及大量修补的任务.

    您可以通过以下方式浏览列表：

    • 只需按下命令/ ctrl + F即可搜索关键字
    • 或者，使用我们网站上的搜索： https://www.awesome-ha.com

    Installing¶

    Home Assistant有几种安装/运行方法. 很多人都有 不同的意见和他们个人的最爱. 每种方法都有它 的优点和缺点. 重要的是要知道，没有错，或者就在这里， 每种技术都安装了 SAME Home Assistant.

    Home Assistant目前_recommends_ Hass.io方法.

    • Hass.io - 使用Docker托管环境进行安装（推荐方法）.
    • Docker - Installing on Docker.
    • Hassbian - 安装Hassbian.
    • Manually - 使用Python虚拟环境手动安装.

    Need Help?¶

    有多种方式可以与家庭助理社区联系. 如果您有问题，需要帮助，想要申请功能，这没关系， 或者只是说'嗨'.

    Official channels¶

    • Home Assistant Discord - 加入聊天，我们大多数人都在那里.
    • Home Assistant Community - 讨论论坛，也用于功能请求.
    • Home Assistant Subreddit - 如果你进入Reddit，请订阅.
    • Home Assistant Enthusiasts - 面向爱好者的Facebook群组.

    Other channels¶

    • Dr. ZZs - Zzs博士的Facebook小组.
    • Community Hassio Add-ons Discord - 获得社区Hassio附加组件的支持.

    Public Configurations¶

    有些人将完整的Home Assistant配置存储在GitHub上. 他们是 一个很棒的学习来源和灵感来源.

    • Carlo Costanzo - 可能是那里记录最多的配置.
    • DubhAd - 也称为Tinkerer共享他的配置文件.
    • geekofweek - 拥有超过300多个自动化.
    • Isabella Gross Alström - Hass.io，Intel NUC，Ubuntu，Docker，Lovelace UI.
    • Mahasri Kalavala - 令人印象深刻的设置，许多不同的硬件协同工作.
    • stanvx - 使用AppDaemon和HA Floorplan的完整设置.
    • Vasiley - 运行两个协同工作的实例.
    • Alok Saboo - 也称为arsaboo. 定期更新.
    • Aaron Bach - 也称为bachya. 定期更新并包含众多Dockerized服务.
    • James McCarthy - 记录良好，YAML和Node-RED中的3个实例和自动化.
    • Franck Nijhof - 基于Hass.io，与其他配置结构截然不同.
    • Andrea Donno - 基于Hass.io，专注于触摸屏使用.
    • Klaas Schoute - 基于Hass.io，Intel NUC，Ubuntu Server，Docker并定期更新.
    • Jason Hunter - 基于Hass.io，Intel NUC i5，TensorFlow和相机流.

    Hass.io¶

    Hass.io是一个完整的操作系统，将负责安装和 更新Home Assistant，并从前端进行管理.

    • Hass.io Beginner's Guide - ：tv：关于入门的优秀分步指南（基于HassOS）.

    Official Add-ons¶

    Add-ons是易于安装的服务，可以扩展功能 你的Hass.io实例.

    • DuckDNS - 更新您的Duck DNS IP地址并使用Let的加密生成SSL.
    • HASS Configurator - 基于浏览器的配置文件编辑器.
    • Mosquitto - 快速可靠的MQTT经纪人.
    • SSH Server - 允许使用SSH远程登录.
    • Samba - 使用Windows网络共享访问配置文件.
    • NGINX SSL proxy - 使用SSL终止的反向代理.
