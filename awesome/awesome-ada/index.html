    >阿达还活着踢！

    Competition¶

    Compilers¶

    Local¶

    • gnat-ce - Ada编程语言的Adacore编译器，它构成了GNU编译器集合的一部分. 它支持该语言的所有版本，即Ada 2012，Ada 2005，Ada 95和Ada 83.
    • fsf-gnat - 用于Ada编程语言的自由软件基础编译器，它构成GNU编译器集合的一部分. 它支持该语言的所有版本，即Ada 2012，Ada 2005，Ada 95和Ada 83.
    • janus-ada - 编译器支持该语言的Ada 83，Ada 95，Ada 2005和Ada 2012版本的所有主要功能.
    • object-ada - PTC ObjectAda是一系列广泛的本机和交叉开发工具和运行时环境.
    • apex-ada - PTC ApexAda是业界最受欢迎的Ada和混合Ada / C / C ++应用程序开发环境之一.
    • greenhills-ada - Green Hills Software提供业界首个经过全面验证的Ada Cross Compilers系列，用于实时目标.
    • hac - 实验在那里有一个完整的编译器.
    • byron - Byron项目是一个社区驱动的Ada编译器，工具链和IDE系统.

    Online¶

    • tutorials-point-ada-compiler - 基于CodingGround引擎构建的在线编译器.
    • jdoodle-ada-compiler - 在线编译器，提供多个版本的GNAT进行测试.

    Deployment¶

    Build and Package¶

    • alire - 即用型Ada库目录以及命令行工具（alr），用于获取，编译和合并到您自己的项目中. 它旨在履行与Rust的货物或OCaml的opam类似的角色.
    • gprbuild - Adacore多语言软件构建工具.
    • ravenadm - Ravenports的管理工具http://www.ravenports.com.
    • synth - 用于实时系统的Next D / Ports构建工具（Portmaster和Portupgrade工具的替代方案）.

    Continuous Integration¶

    • continuous-verification - 与Travis CI自动进行SPARK正式验证.

    Runtimes¶

    • bb-runtimes - GNAT裸机板支持包（BSP）.
    • avr-ada - 用于8位AVR微控制器的GNAT.
    • ada-runtime - 缩小的Ada运行时，可以适应不同的平台.

    OS and Kernels¶

    • marte-os - MaRTE OS是一种硬实时操作系统，适用于遵循最小实时POSIX.13子集的嵌入式应用程序. 它提供了易于使用和控制的环境来开发多线程实时应用程序.
    • muen - 用于高保证的x86 / 64分离内核.
    • ewok - EwoK是一种针对微控制器和嵌入式系统的微内核.
    • bare-bones - 阿达港口 osdev.org 最小的32位x86内核.

    Games¶

    • ada-gate - AdaGate是适用于Windows，OS-X和Linux的Stargate / Portal幻想设置中的第一人称3D推箱子益智游戏.

    • ada-venture - AdaVenture是一款儿童友好的复古点击游戏，包含迷宫，龙，蝙蝠和蛇.
    • terminal-apps - 在任何操作系统上的任何终端上运行的有趣益智游戏.
    • co-terminal-apps - 在任何平台上的终端中运行的复古游戏.

    • pasta - 意大利面条！ 是一个“匹配-3”类型的游戏.

    Frameworks¶

    Components¶

    • simple-components - B树，切比雪夫系列，密码学，双链网和列表，黑板（无锁），ELV / eQ-3 MAX！ 客户端，事件（普通，脉冲，数组），GNUTLS绑定，图形（有向，加权，非循环，树），HTTP实现，FIFO（无锁），IEEE 754，进程间通信，MODBUS客户端，MQTT客户端 - 服务器/代理，TCP，互斥体，对象，ODBC绑定，分析器，持久对象，持久存储，池，集，映射，SMTP客户端，SQLite绑定，堆栈，流，字符串编辑，表（字符串映射），无界数组.
    • ada-util - 一个接近Java log4j框架的日志框架，支持属性，XML / JSON / CSV的序列化/反序列化框架，Ada bean框架，编码/解码框架（Base16，Base64，SHA，HMAC-SHA），一个组合流框架（原始，文件，缓冲区，管道，套接字），几个并发工具（引用计数器，计数器，池，fifos，数组），进程创建和管道，支持加载共享库（在Windows或Unix上），HTTP客户端库在上面CURL或AWS.
    • gnatcoll-core - 这是GNAT Components Collection的核心模块.
    • gnatcoll-bindings - 这是GNAT Components Collection的绑定模块.
    • gnatcoll-db - 这是GNAT Components Collection的DB模块.
    • gnatcoll-json - 这是一组用于编写JSON接口的帮助程序，它包含大多数Ada运行时组件的JSON解析.
    • ada-interface - 这是一个与基于组件的系统的应用程序一起使用的接口集合. 它旨在轻松便携/平台独立，并与之兼容 ada-runtime.

    Distributed¶

    • poly-orb - PolyORB提供统一的解决方案，以依赖中间件标准构建分布式应用程序.

    Graphical User Interface¶

    • gnoga - 用于Ada的GNU Omnificent GUI.
    • gnavi - GNU Ada Visual Interface.
    • claw - CLAW，适用于Microsoft Windows的高级便携式Ada 95绑定.
    • gtkada - 基于Gtk3组件的Ada图形工具包.
    • qt-ada - Ada-2012端口到QT 5框架.
    • vtk-ada - Ada-2012端口到VTK 8.1（可视化工具包）.
