    Audio Related Packages¶

    • 包裹总数：72
    • Python版本兼容性： （²表示只有python 2的包，³表示只有python 3的包;剩下的包支持python 2 + 3）.

    Read-Write¶

    • audiolazy :octocat: - 用于Python的表达式数字信号处理（DSP）包.
    • audioread :octocat: - 跨库（GStreamer + Core Audio + MAD + FFmpeg）音频解码.
    • mutagen :octocat: - 读取和写入各种格式的各种音频元数据.
    • pyAV :octocat: - PyAV是FFmpeg或Libav的Pythonic绑定.
    • (Py)Soundfile :octocat: - 基于libsndfile，CFFI和NumPy的库.
    • pySox :octocat: - sox包装.
    • PySynth :octocat: - 简单的midi-to-wav合成器.
    • stempeg :octocat: - 读/写STEMS多流音频.
    • tinytag :octocat: - 阅读MP3，OGG，FLAC和Wave文件的音乐元数据.
    • audiomate³ :octocat: - 加载不同类型的音频数据集.

    Transformations - General DSP¶

    • acoustics :octocat: - 为声学家提供有用的工具.
    • AudioTK :octocat: - DSP滤波器工具箱（大量滤波器）.
    • AudioTSM :octocat: - 实时音频时标修改程序.
    • Gammatone :octocat: - Gammatone滤波器组实现.
    • pyFFTW :octocat: - 用于FFTW的包装器（3）.
    • NSGT :octocat: - 非定常gabor变换，常数q.
    • MDCT :octocat: - MDCT变换.
    • pydub :octocat: - 使用简单易用的高级界面处理音频.
    • pytftb :octocat: - MATLAB时频工具箱的实现.
    • pyroomacoustics :octocat: - 室内声学仿真（RIR发生器）
    • PyRubberband :octocat: - 包装纸 rubberband 进行音高变换和时间拉伸.
    • PyWavelets :octocat: - Python中的离散小波变换.
    • Resampy :octocat: - 采样率转换.
    • SFS-Python :octocat: - 声场合成工具箱.
    • STFT :octocat: - 用于短时傅里叶变换的独立包.

    Feature extraction¶

    • aubio :octocat: - 特征提取器，用C语言编写，Python接口.
    • audiolazy :octocat: - 实时音频处理库，通用.
    • essentia :octocat: - 音乐相关的低级和高级特征提取器，基于C ++，包括Python绑定.
    • muda :octocat: - 音乐数据增强.
    • python_speech_features :octocat: - ASR的常见语音功能.
    • pyYAAFE :octocat: - YAAFE功能提取器的Python绑定.
    • speechpy :octocat: - 语音处理和识别库，目前主要是特征提取.

    Speech Processing¶

    • aeneas :octocat: - 强制对准器，基于MFCC + DTW，35种以上语言.
    • Parselmouth :octocat: - Python界面 Praat 语音和语音分析，综合和操作软件.
    • pyAudioAnalysis² :octocat: - 特征提取，分类，Diarization.
    • py-webrtcvad :octocat: - WebRTC语音活动检测器的接口.
    • PyWorldVocoder :octocat: - Morise世界声码器的包装.
    • Montreal Forced Aligner :octocat: - 强制对齐器，基于Kaldi（HMM），英语（其他人可以训练）.
    • SIDEKIT³ - 演讲者和语言识别.
    • SpeechRecognition :octocat: - 适用于多个ASR引擎和API的在线和离线包装.

    Perceptial Models - Auditory Models¶

    • cochlea² :octocat: - 内耳模型.
    • Brian2 :octocat: - 尖峰神经网络模拟器，包括耳蜗模型.
    • Loudness³ :octocat: - 感知响度，包括Zwicker，Moore / Glasberg模型.
    • Sound Field Synthesis Toolbox :octocat: - 声场合成工具箱.

    Source Separation¶

    • beta_ntf² :octocat: - 使用PARAFAC的非负张量因子分解.
    • commonfate [：octocat：]（） - 共同的命运模型和变换.
    • NUSSL² :octocat: - 各种源分离算法+框架.
    • NTFLib :octocat: - 稀疏的Beta-发散张量分解.
    • NIMFA :octocat: - 几种NMF口味.
    • pyFASST² :octocat: - 用于灵活音频源分离工具箱的包装器.

    Music Information Retrieval¶

    • Catchy :octocat: - 用于计算钩子发现的语料库分析工具.
    • dejavu² :octocat: - 音频指纹识别和识别.
    • Madmom :octocat: - MIR包，重点关注节拍检测，起始检测和和弦识别.
    • mir_eval :octocat: - 各种MIR任务的共同分数. 还包括bss_eval实现.
    • msaf :octocat: - 音乐结构分析框架.
    • librosa :octocat: - 一般音频和音乐分析.
    • rp_extract² :octocat: - Rhythm Pattern音乐特色提取器.

    Deep Learning¶

    • Kapre :octocat: - 硬音频预处理器
    • TorchAudio :octocat: - PyTorch音频加载器

    Symbolic Music - MIDI - Musicology¶

    • Music21 :octocat: - 计算机辅助音乐学工具包.
    • Mido :octocat: - 实时MIDI包装.
    • mingus² :octocat: - 具有MIDI文件和播放支持的高级音乐理论和符号包.
    • Pretty-MIDI :octocat: - 实用功能，以美观/直观的方式处理MIDI数据.

    Realtime applications¶

    • PYO :octocat: - 实时音频dsp引擎.
    • python-sounddevice :octocat: - PortAudio包装器，提供NumPy的实时音频I / O.

    Web Audio¶

    • TimeSide² :octocat: - 高级音频分析，成像，转码，流媒体和标签.

    Audio related APIs and Datasets¶

    • beets :octocat: - 音乐图书馆经理和 MusicBrainz 标签.
    • dsdtools :octocat: - 解析并处理 demixing secrets dataset.
    • medleydb :octocat: - 解析 medleydb 音频+注释.
    • Soundcloud API :octocat: - 包装纸 Soundcloud API.
    • Youtube-Downloader :octocat: - 下载YouTube视频（和音频）.

    Wrappers for Audio Plugins¶

    • VamPy Host - 接口编译的vamp插件.

    Tutorials¶

    • Whirlwind Tour Of Python [：octocat：]（https://github.com/jakevdp/WhirlwindTourOfPython ） - 针对研究人员和开发人员快速介绍Python要点.
    • Introduction to Numpy and Scipy :octocat: - 强烈推荐的教程，涵盖科学Python生态系统的大部分内容.
    • Numpy for MATLAB® Users - 切换器的等效python函数的简短概述.
    • MIR Notebooks :octocat: - 收集用于音乐信息检索（MIR）的教学iPython笔记本.
    • Selected Topics in Audio Signal Processing - 作为iPython笔记本练习.

    Books¶

    • Python Data Science Handbook - Jake Vanderplas，优秀书籍和随附的教程笔记本.
    • Fundamentals of Music Processing - MeinardMüller，附带Python练习.

    Scientific Papers¶

    • Python for audio signal processing - John C. Glover，Victor Lazzarini和Joseph Timoney，2011年Linux音频会议.
    • librosa: Audio and Music Signal Analysis in Python, Video - Brian McFee，Colin Raffel，Dawen Liang，Daniel PW Ellis，Matt McVicar，Eric Battenberg，Oriol Nieto，Scipy 2015.

    Other Resources¶

    • Coursera Course - 音频信号处理，基于Python的课程，来自巴塞罗那和斯坦福大学的UPF.
    • Digital Signal Processing Course - 拥有许多Python示例的硕士课程材料（罗斯托克大学）.
    • Slack Channel - 音乐信息检索社区.
