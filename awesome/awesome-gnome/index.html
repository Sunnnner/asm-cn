    GNOME

    Awesome GNOME ¶

    > GNOME桌面环境的_awesome_应用程序，扩展，模块，主题和工具的精选列表.

    GNOME 是Linux和Unix的桌面环境，旨在让用户掌控并完成工作. 它将GTK用于其应用程序，将Clutter和JavaScript用于其官方桌面shell.

    >标有！[GNOME Core] [GNOME Core]的项目是由GNOME项目正式维护和分发的应用程序，作为核心应用程序和工具集的一部分.

    Applications¶

    Productivity¶

    • GNOME Todo - 最小的任务管理器. ！[GNOME核心] [GNOME核心]
    • GTimeLog - 用于跟踪时间的简单应用程序.
    • Gnome Pomodoro - 简单的番茄计时器.

    Internet¶

    • GNOME Web - GNOME的官方Web浏览器项目. ！[GNOME核心] [GNOME核心]
    • Eolie - 使用垂直选项卡的Web浏览器.
    • Geary - GNOME的现代邮件客户端，最初由Yorba创建.
    • FeedReader - RSS提要阅读器.
    • Corebird - Twitter客户端.
    • Something for Reddit - GNOME的Reddit客户端.
    • Ring-GTK - 用于Ring的GTK / GNOME客户端，一种安全的通信工具.
    • GNOME Twitch - 用于观看Twitch的GTK / GNOME客户端.
    • Haguichi - Linux上Hamachi的图形前端.
    • Transmission Remote GNOME - Transmission torrent守护程序的远程客户端.
    • Fragments - GTK3 BitTorrent客户端

    Editor¶

    • UberWriter - 使用GTK +制作的GNU / Linux的无分心Markdown编辑器

    Documents¶

    • NFO Viewer - NFO文件的简单查看器，具有预设字体和编码设置的文本编辑器以及可点击的超链接支持.
    • Paperwork - 扫描文档和PDF的个人文档管理器.
    • Easy Ebook Viewer - 现代GTK Python应用程序，可轻松读取ePub文件.
    • Bookworm - 简单，专注的电子书阅读器，在GNOME中看起来很棒.
    • Showdown - 简单的降价阅读器.
    • Pdftag - PDF的简单元数据编辑器

    Multimedia¶

    • Webmusic - 基于Web的音乐播放器，可将您喜爱的音乐服务集成到桌面中.
    • Nuvola Player - 基于Web的音乐流媒体服务的运行时，提供更加原生的用户体验并与Linux桌面环境集成.
    • Vocal - 功能强大，美观，简单的播客客户端，适用于现代免费桌面.
    • Celluloid - mpv的简单GTK +前端.
    • Lollypop - 美妙的音乐应用.
    • GRadio - 基于网络的无线电客户端.
    • Parlatype - 专门用于转录的音频播放器.
    • Pulseeffect - 为PulseAudio应用提供限制器，压缩器，混响，均衡器和自动音量效果.
    • Curlew - 用于Linux的多媒体转换器
    • Cozy - 适用于Linux的有声读物播放器

    Graphics¶

    • Coulr - 在RGB和十六进制代码之间转换颜色.
    • GColor 3 - GNOME的GTK3颜色选择器.
    • GThumb - 功能强大的高级应用程序，用于管理GNOME中的照片和图像.

    Games¶

    • Lutris - GNU / Linux的开源游戏平台
    • GNOME Games - 游戏启动器和libretro的仿真前端.

    Customization¶

    • GNOME Tweaks - 以前称为Tweak工具. 高级GNOME 3设置的图形界面.
    • GNOME AppFolders Manager - 管理GNOME应用程序覆盖中的应用程序文件夹的工具.
    • Menulibre - 如果使用Arc或应用程序菜单等传统应用程序菜单，请管理应用程序类别.
    • HydraPaper - 为GNOME上的每个监视器设置不同的背景
    • Dynamic Wallpaper Editor - 用于编辑GNOME的XML壁纸的实用程序

    Tools¶

    • Catfish - 为XFCE创建的搜索应用程序，但遵循GNOME HIG.
    • Tilix - 用于GNOME的平铺终端仿真器.
    • GPaste - GNOME的剪贴板管理器（由应用程序和扩展组成）.
    • Peek - 简单的屏幕录像机，可将您的录像保存为GIF.
    • Sequeler - 在Vala中构建的SQL客户端
    • GNOMIT - 简单的Git提交消息编辑器
    • Password Safe - GNOME的密码管理器

    System¶

    • systemd Manager - 控制和管理systemd的工具.

    Others¶

    • Luminance - 飞利浦Hue客户端.
    • razerCommander - 用于管理Razer外围设备的控制中心.

    GTK Modules and Systems¶

    • Plotinus - GTK3应用程序的命令调色板.

    Extensions¶

    Desktop¶

    • Places and Files Direct Access - 直接在桌面上访问您的地点和文件.
    • Notes - GNOME Shell扩展，提供可自定义的便签

    Menus¶

    • Application Menu - 应用程序的基于类别的菜单.
    • Places Indicator - 提供快速访问文件夹的菜单.
    • Arc Menu - 类似Windows7的应用程序菜单，灵感来自Zorin OS中的菜单.

    Applets¶

    • Cafeine - Applet that let you deactivate the GNOME lock screen.
    • Removable Drive Menu - 状态菜单显示可移动驱动器.
    • OpenWeather - 天气小程序.
    • Focusli - 环境噪音小程序.
    • Clipboard Indicator - 简单的剪贴板小程序.
    • Desk Changer - 壁纸幻灯片小程序.
    • Emoji Selector - 表情符号选择器小程序.
    • Media Player Indicator - GNOME的高级MPRIS小程序.
    • Todo.txt - 用于todo.txt的GNOME Shell接口.
    • Time ++ - todo.txt管理器，时间跟踪器，计时器，秒表，番茄钟和闹钟.
    • GSConnect - Gnome Shell 3.24+的KDE Con​​nect实现
    • Docker Integration - 管理docker容器的扩展

    Status Area¶

    • TopIcons Plus - GNOME状态栏的传统系统.
    • KStatusNotifiers/AppIndicator Support - GNOME的Appindicator系统.
    • Argos - 将可执行文件的标准输出转换为面板下拉菜单; 完全兼容macOS的BitBar应用程序.
    • Extend Panel Menu - 将状态菜单分成几个小程序.

    Windows¶

    • Window Corner Preview - 创建窗口预览并将其预览到屏幕的核心.
    • Cascade Windows - 快速安排级联窗户.
    • No Title Bar - 合并活动栏和最大化窗口的标题栏.
    • gTile - 为GNOME Shell带来更高级的平铺.
    • Shellshape - GNOME Shell的平铺窗口扩展.

    Docks and Panels¶

    • Dash-to-Dock - 将GNOME Dash转换为功能齐全的Dock.
    • Workspace-to-Dock - 现代化的码头融合了工作空间和码头的功能.
    • Dash-to-Panel - 现代任务栏/面板与Windows 7+和KDE上的任务栏/面板不同.

    Services integration¶

    • GMail Message Tray - 将GMail与您的GNOME桌面集成.

    Themes¶

    • Oomox - 生成Numix和Materia主题（GTK2，GTK3），Gnome-Colors和Archdroid图标的不同颜色变化，以及匹配它们的终端颜色方案.

    Adwaita-based¶

    • Minwaita - Adwaita紧凑型叉子. （GTK，壳牌）
    • Adwaita Tweaks - 紧凑型Adwaita，比minwaita更忠实，但不太紧凑. （GTK，壳牌）

    Material¶

    • Materia - 基于GNOME / GTK +的桌面环境的类似Material Design的主题. （GTK，壳牌）
    • Adapta - 基于材料设计指南的自适应GTK +主题. （GTK，壳牌）
    • Vimix - GTK 3，GTK 2和Gnome-Shell的平面材料设计主题.

    Flat¶

    • Arc - 平面主题与透明元素. （GTK，壳牌）
    • Pop - System76的官方主题Pop！_OS，使用双色蓝色和橙色方案. （GTK，壳牌）
    • United - 针对GNOME的Unity8灵感主题. （GTK，壳牌）
    • Numix - 流行的平面半暗主题，橙色触感. （GTK）
    • Zukitre - 平灰色主题，zuki主题套房的一部分. （GTK，与Zuki-Shell合作开发shell主题）
    • Yaru - Ubuntu社区主题（GTK，Shell）

    Metallic¶

    • Greybird - 官方Xubuntu主题. （GTK）
    • Vertex - 由Arc的创造者horst3180创建的金属主题. （GTK，壳牌）
    • GNOME OSC - 模仿macOS外观和感觉的主题. （GTK，壳牌）
    • Zukitwo - Skeumorphic灰色主题，zuki主题套件的一部分. （GTK，与Zuki-Shell合作开发shell主题）.

    Icons¶

    Material Icons¶

    • Paper - 材料图标主题.
    • Papirus - 材质图标主题，最初基于纸张.
    • Pop! - 基于Papirus的System76官方图标主题Pop！_OS.

    Flat Icons¶

    • La Capitaine - 图标灵感来自macOS和Material Design指南.

    Homogeneous Icons¶

    • Moka - 简单的方形图标主题，松散地基于Tango规范.
    • Obsidian - 延续着名的法恩莎主题.
    • Numix Circle - 扁圆形图标主题.
    • Numix Square - 扁方形图标主题.

    Skeumorphic¶

    • Elementary XFCE - 基本操作系统图标的桌面无关版本.

    Cursors¶

    • ComixCursors - X11鼠标主题，带有漫画感. 提供六种颜色和三种颜色（常规，纤细和不透明）.
    • Bibata - 基于Silm材质的光标主题.
    • Capitaine Cursors - 受macOS启发并基于KDE Breeze的x光标主题. 旨在与La Capitaine图标配对.
