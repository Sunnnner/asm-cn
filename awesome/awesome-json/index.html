    JSON

    Awesome JSON ¶

    精选的JSON库和资源的精选列表.

    灵感来自于 awesome 名单.

    Applications¶

    OS X * Visual JSON (github） - 适用于Mac OS X的简单JSON漂亮查看器. * JSONExport - 将对象转换为当前支持的语言之一的类.

    Binary Serialization¶

    • BSON - 二进制JSON.
    • MessagePack - 一个非常有效的对象序列化库.
    • UBJSON - 二进制JSON的通用兼容格式规范.
    • CBOR - 简明的二进制对象表示.
    • PSON - Protocol JSON，超高效的二进制序列化格式.

    Browser Extensions¶

    Chrome * JSON Formatter (github ） - 使JSON易于阅读. 开源. * JSON Viewer (github） - 这是一个用于打印JSON和JSONP的Chrome扩展程序. * JSON Browser (github） - 在JSON模式的帮助下浏览JSON Web. * JSON Finder (github） - 像在Finder中一样浏览.

    Firefox * JSONView (github） - 在浏览器中查看JSON文档. * JSON-DataView (github） - 以具有语法高亮的可折叠树结构显示数据.

    Safari * JSONAce (github） - 格式和语法使用ACE编辑器突出显示在Web浏览器内部查看的JSON. * JSONView (github） - JSONView Firefox扩展的一个端口，其格式和语法突出显示在浏览器内查看的JSON

    Command-line tools¶

    • fx - 交互式终端工具.
    • jsoncat - 漂亮的印刷Json终端，颜色和调整标签大小.
    • jq - 轻量级且灵活的命令行JSON处理器.
    • json - 用于在Unix命令行上按摩JSON的“json”命令.
    • jshon - 一个解析器，旨在最大限度地方便shell.
    • jarg - shell中的速记JSON和表单编码语法.
    • jsawk - 像awk一样，但对于JSON.
    • gron - 将JSON文件转换为可迁移的离散分配.
    • jid - 增量挖掘机. 使用jq等过滤查询以交互方式深入研究JSON.
    • jiq - 这是`jq和jq. 您可以使用jq过滤查询以交互方式向下钻取.
    • jv - jv（用于jsonviewer）可以帮助您查看JSON.
    • jl - Functional sed for JSON.

    Databases¶

    • MongoDB - 开源文档数据库和领先的NoSQL数据库.
    • RethinkDB - 一个开源的分布式文档数据库，具有令人愉快和强大的查询语言.
    • EJDB - 嵌入式JSON数据库引擎. （C）
    • lowdb - 基于lodash API构建的平面文件数据库. （JavaScript）
    • Lawnchair - 轻量级的客户端文档存储. （JavaScript）
    • JSON ODM - 用于在服务器或浏览器中使用的JavaScript的对象文档映射器. （JavaScript）
    • JSON Server - 在不到30秒的时间内获得零编码的完整虚假REST API.
    • Kinto - 具有同步和共享功能的轻量级JSON存储服务.
    • CouchDB - 无缝多主同步，可从大数据扩展到移动，具有直观的HTTP / JSON API，专为可靠性而设计.
    • RxDB - 具有JSON-Schema，mango-Query和CouchDB-sync的事件驱动的JSON数据库. （JavaScript）
    • JSONlite - 一个简单，独立，无服务器，零配置的json文档存储. （击）

    Datasets¶

    • country.io - 各种国家相关数据集，如JSON inc货币，国家代码，名称等
    • countries - 世界国家.
    • vat-rates - 所有欧盟国家的增值税税率.
    • MTG JSON - 最新的Magic the Gathering卡数据.
    • Heartstone JSON - 最新的炉石信息卡数据.
    • getCountries() - 自定义国家数据的生成器.

    Data modeling¶

    • JSONModel - 神奇的数据建模框架. （目标C）

    Data generation¶

    • jsonymize - 从标准输入读取数据，匿名，然后写入标准输出.
    • dyson - 用于动态，伪造JSON的服务器. （的node.js）

    Differencing¶

    • JSONPatch - 用于描述文档更改的格式.
    • JSON-Patch - 精益和JSON-Patch标准（RFC 6902）的Javascript实现. （JavaScript）
    • jiff - 基于rfc6902的JSON Patch和diff. （JavaScript）
    • json-patch-php - JSON-patch的实现（IETF RFC 6902）（PHP）
    • dffptch - 使用紧凑的diff格式进行差异和修补的微型库. （JavaScript）
    • jsondiffpatch - JavaScript对象的Diff和补丁. （JavaScript）

    Editors¶

    • JSONEdit - 用户友好的可视化编辑器，构建为AngularJS指令.

    Format Extensions¶

    • GeoJSON - 地理空间数据交换格式.
    • JSON-LD - 轻量级关联数据格式.
    • JSON-RPC - 无状态，轻量级远程过程调用（RPC）协议.
    • JSONP - 使用JSON-P / JSONP更安全的跨域Ajax.
    • JsonML - 用于将基于XML的标记作为JSON传输的紧凑格式，允许将其无损地转换回其原始形式.
    • JSON5 - 旨在使人们更容易手工编写和维护的扩展.
    • JSON6 - 人类JSON（ES6）.
    • JSON Resume - 为简历创建标准的开源计划.
    • JSON Web Tokens - 一种紧凑的URL安全手段，代表双方之间转让的索赔.
    • JSON API - 构建API的标准.
    • Collection+JSON - 读/写超媒体类型，旨在支持管理和查询简单集合.
    • hal-json - 用于表达JSON或XML中的超链接的一组约定.
    • JSON Activity Streams - 用于在网络上联合社交活动的格式.
    • JSON-stat - 用于数据传播的简单轻量级格式.
    • /contribute.json - 跨项目更容易访问开源贡献信息.
    • JSON Table Schema - 表格数据的简单模式
    • NDJSON （换行符分隔JSON） - 用于在流协议中分隔JSON的标准.
    • survey.js - 基于JSON的调查库.
    • JSON Meta Application Protocol (JMAP) - 用于有效同步基于JSON的数据对象的协议，支持推送和带外二进制数据上载/下载.
    • JackSON: JSON secret keeper - 在配置文件中存储秘密的JSONic方式.

    Frontend components¶

    • JSON editor jQuery plugin - 适用于您的网络应用/网页的组件. （jQuery的）
    • jqTree - 用于在html中显示树结构的小部件. （jQuery的）
    • jsTree - jquery插件，提供交互式树. （jQuery的）
    • Dynatable.js - 一个更有趣，语义，HTML5 + JSON，交互式表插件. （jQuery的）
    • JSON Formatter - HTML中可折叠JSON的Angular指令. （AngularJS）
    • react-jsonschema-form - 用于从JSON Schema构建Web表单的React组件. （反应）

    Libraries¶

    C * Jansson - 用于编码，解码和操作数据的AC库. * jsmn - C中的简约解析器.它可以轻松集成到资源有限的项目或嵌入式系统中.

    C++ * ArduinoJson - 嵌入式系统的高效库. * JSON++ - 用于C ++ 11的自包含Flex / Bison解析器. * json11 - 一个用于C ++ 11的小型库. * RapidJSON - 一个快速的JSON解析器/生成器，用于C ++，同时具有SAX / DOM样式API

    Clojure * data.json - 来自Clojure数据结构的解析器/生成器.

    Fortran * JSON-Fortran - 用于编写，读取和操作JSON文件和数据结构的Fortran库.

    Haskell * aeson-qq - Haskell的JSON quasiquoter. * json-schema - Haskell的JSON Schema库 * hjsonschema - Haskell的JSON Schema Draft 4库

    Java * JSON-java - 参考实施. * Fast JSON Processor * Gson - 将JSON转换为Java对象的Java库，反之亦然. * Jackson - 用于处理JSON数据格式的多用途Java库. * moshi - 适用于Android和Java的现代JSON库.

    Javascript * JSON-js - JavaScript中的JSON. * JSON 3 - 现代化的实施. * oboe.js - 流式处理方法，通过在响应完成之前提供已解析的对象来加速Web应用程序.

    Objective-C * JSONKit - Objective-C库. * SBJson - 解析一个或多个数据块.

    Perl * JSON::Tiny - 用于以简约方式编码和解码JSON的Perl模块.

    PL/SQL * PL/JSON - 用PL / SQL编写的通用JSON对象.

    PHP * Webmozart JSON - 强大的解码器/编码器，支持模式验证.

    Python * simplejson - 简单，快速，可扩展的编码器/解码器 * jsonpickle - 用于序列化任意对象图的库. * metamagic.json - JSON编码器的超快速Python 3实现.

    Ruby * oj - 快速JSON解析器和Object marshaller作为Ruby gem. * MultiJSON - 用于JSON处理的通用可交换后端.

    React * json2react - 使用JSON创建React无状态组件.

    .NET * jsonfx - .NET的序列化框架.

    Scala * spray-json - Scala中的轻量级，干净且简单的实现. * circe - 另一个Scala的JSON库. * scala-jsonapi - 支持库，用于将JSON：API规范与Play，Spray和/或Circe后端集成. * jsoniter-scala - 用于编译时生成超快JSON编解码器的Scala宏.

    Swift * SwiftyJSON - 在Swift中处理数据的更好方法.

    • HanSON - 人类的JSON - 带有不带引号的标识符，多行字符串和注释.
    • μson （uson） - JSON的简写.
    • HOCON - 人类优化的配置对象表示法.
    • ASON - 一个语义完整的JSON（草案）超集.
    • TOML - 由于明显的语义而易于阅读的最小配置文件格式.
    • HCL - 结构化配置语言，既人性化又适合机器使用.
