    Packages¶

    Mad science¶

    • webtorrent - 为Node.js和浏览器流式传输torrent客户端.

    • turf - 模块化地理空间处理和分析引擎.
    • webcat - 使用WebRTC使用您的GitHub私钥/公钥进行身份验证的网络上的p2p管道.
    • NodeOS - 第一个由npm驱动的操作系统.
    • YodaOS - AI操作系统.
    • Brain.js - 机器学习框架.

    • xlsx - 纯JS Excel电子表格阅读器和编写器.

    Functional programming¶

    • lodash - 实用程序库提供一致性，定制，性能和附加功能. 更好更快的Underscore.js.
    • immutable - 不可变数据收集.
    • Ramda - 实用程序库，侧重于通过自动currying和反向参数顺序启用的灵活功能组合. 避免变异数据.
    • Folktale - 用于JavaScript中通用函数编程的库套件，允许您编写具有更少错误的优雅模块化应用程序，以及更多重用.
    • Mout - 其他现有解决方案之间最大区别的实用程序库是您可以选择仅加载所需的模块/功能，而无需额外开销.
    • Bacon.js - 功能反应式编程.
    • RxJS - 用于转换，组合和查询各种数据的功能反应库.
    • Lazy.js - 类似于lodash / Underscore的实用程序库，但具有惰性评估，在许多情况下可以转化为卓越的性能.
    • Kefir.js - 反应型库，专注于高性能和低内存使用.

    • gh-got - got与GitHub API交互的便捷包装器.
    • axios - 基于Promise的HTTP客户端（也适用于浏览器）.
    • request - 简化的HTTP请求客户端.
    • wreck - HTTP客户端实用程序.
    • download - 轻松下载和提取文件.
    • http-proxy - HTTP代理.
    • superagent - HTTP请求库.
    • node-fetch - Node.js的window.fetch.
    • flashheart - REST客户端.
    • http-fake-backend - 通过可配置的路由提供JSON文件或JavaScript对象的内容，构建虚假的后端.
    • cacheable-request - 使用符合RFC的缓存支持包装本机HTTP请求.
    • gotql - 构建的GraphQL请求库 got.

    Debugging / Profiling¶

    • ndb - Chrome DevTools支持改进的调试体验.
    • ironNode - 开箱即用的支持ES2015的Node.js调试器.
    • node-inspector - 基于Blink Developer Tools的调试器.
    • debug - 微小的调试实用程序.
    • why-is-node-running - Node.js正在运行，但你不知道为什么？
    • njsTrace - 检测并跟踪代码，查看所有函数调用，参数，返回值以及每个函数所花费的时间.
    • vstream - 可检测的流混合来检查流的管道.
    • stackman - 使用代码摘录和其他好东西增强错误堆栈跟踪.
    • locus - 在运行时启动可以访问所有变量的REPL.
    • 0x - Flamegraph分析.
    • ctrace - 格式良好且改进的跟踪系统调用和信号.
    • leakage - 写内存泄漏测试.
    • llnode - 事后分析工具，允许您检查对象并从崩溃的Node.js进程中获取见解.
