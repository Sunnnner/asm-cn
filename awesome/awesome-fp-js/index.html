    Functional Programming

    Awesome FP JS ¶

    这是一个很棒的精选列表 functional programming JavaScript的代码和学习资源. 作为一种多范式编程语言，JavaScript可以用很多样式编写. 有了这些资源，我们希望帮助您更好地利用JavaScript支持以_functional_方式编写程序.

    功能编程是一个 style of programming 将计算模型化为表达式的评估. 将此与命令式编程进行对比，其中程序由在执行时改变全局状态的语句组成. 函数式编程通常避免使用可变状态，而是支持_side-effect free_function和_immutable_ data. 这鼓励编写易于推理的可组合和声明性程序.

    Libraries¶

    • Ramda - 一个实用的JavaScript函数库，专为函数式编程风格而设计. 一种样式，可以轻松创建功能管道，从不改变用户数据.
    • Ramda Adjunct 是Ramda的社区维护扩展.
    • Ramda-extension 是一组仅由Ramda函数组成的实用程序无点函数.
    • Rambda - 在10kB以下更快的替代Ramda.
    • Folktale – A standard library for functional programming in JavaScript.
    • lodash/fp - 一个实例 Lodash 将其方法包装起来，以生成不可变的，自动curried，iteratee-first，data-last方法.
    • functional.js - 轻量级功能JavaScript库，便于currying和无点/默认编程.
    • 101 - 一个现代的模块化JavaScript实用程序库，可以很好地与vanilla JavaScript方法配合使用.
    • fnuc - CoffeeScript（和JavaScript）的功能库，用于促进功能组合和更高阶的功能.
    • barely-functional - 使用本机ES5 / 6操作的小型（2.7kb）函数编程库.
    • bluebird-promisell - 承诺的实用函数编程库.
    • prelude.ls - 功能导向的实用程序库，有点基于Haskell的Prelude模块.
    • preludejs - JavaScript的硬核功能编程.
    • 1-liners - 功能性工具，不能简单. 这款简约实用多功能腰带采用手工精制而成，充满爱与关怀.
    • fn-curry - 一个简单的函数来理解函数.
    • curry - 使用函数绑定语法来调整函数.
    • compose-function - 从较小的功能中编写新功能.
    • functionize - 有助于使非功能库功能化的一系列功能.
    • lambdajs - 完整的ECMAScript API以功能方式完成.
    • fp-dom - 使DOM功能化.
    • trifl - 具有单向数据流和虚拟dom的功能用户界面库.
    • funcy - 在JavaScript中添加功能模式匹配的实验. Experimental：triangular_flag_on_post：
    • date-fp - 用于处理JavaScript日期的功能实用程序库. date-fp中的所有函数都是纯粹的，自动调整的，不会改变它们应用的日期对象.
    • js-joda - 一个不可变的日期和时间库，它提供基于ISO8601日历的简单，域驱动和干净的API.
    • _part_ - 通过将本机方法作为部分应用函数提供来鼓励函数式编程的微型库.
    • claire - 基于属性的测试库，用于清楚地指定代码不变量和行为.
    • FPO.js - Kyle Simpson的JavaScript库（又名getify）. 支持命名参数样式方法.
    • Javascript Parser Combinator - 受Haskell的Parsec启发的Javascript解析器组合器实现
    • fun-task - 用于管理JS中的异步代码的抽象. 任务类似于Promises，主要区别在于Tasks可以表示计算，而Promise只能表示计算结果. 具有Flow类型定义的船舶.
    • most - 超高性能的反应式编程，可帮助您在值和事件流上组成异步操作，而不会产生许多副作用和可变共享状态的危险.
    • zen-signals - ☯最简单的信号库
    • funfix - Funfix是一个用于JavaScript，TypeScript和Flow中的函数编程的类型类和数据类型的库.
    • pico-lambda - 数组，字符串和东西的功能方式. 基于本机方法的640b功能库.
    • pareto-js - 用于JavaScript的极小，直观且快速的功能实用程序库
    • fpEs - 一个小型库提供简单的基本FP和模式匹配/ sumtype和MonadIO / Rx＆可选的Javascript用法
    • Creed - 具有高级功能的复杂和功能强大的异步：协程，承诺，ES2015迭代，幻想 - 土地.

    Data Structures¶

    Write performant functional code by using the right data structures for the task.

    • Icepick 用于将冻结的JavaScript对象视为持久不可变集合的实用程序
    • Immutable.js - 不可变的持久数据收集.
    • Immer - Immer是一个基于写时复制机制的不可变状态的小包.
    • Mori - ClojureScript的持久数据结构和支持API来自舒适的vanilla JavaScript.
    • Baobab - 具有游标的持久性和可选的不可变数据树.
    • immutable-sequence.js - 基于JavaScript的JavaScript中不可变序列的高性能实现 Finger Trees.
    • Timm - 具有快速读取和可接受写入的不变性助手.
    • Lazy.js - 一个带有懒惰引擎的实用程序库，尽可能灵活地尽可能少地工作.
    • DerivableJS - JavaScript和TypeScript的功能反应状态. DerivableJS使您能够对关于状态位的相关性做出优雅的声明性陈述.
    • imlazy - 使用ES2015迭代协议创建和操作惰性迭代的库.
    • collectable - 适用于现代JavaScript和TypeScript应用程序的超高性能不可变数据结构.
    • List - 具有全面功能API的极其快速的不可变列表.
    • prelude.ts - 不可变的持久集合，功能构造，如Option和Either，以及组合器. 在Typescript中实现，但也支持javascript.
    • TSTL - C ++ STL（标准模板库）在TypeScript中实现. 提供遵循函数编程规则的STL容器，迭代器，算法和仿函数.

    Algebraic Data Types¶

    使用数学定律而不是总是重塑自己的事物. 代数！

    • Fantasy Land - 不是图书馆，而是图书馆要遵循的Monad法律规范.
    • Static Land - 与Fantasy Land类似的规范，但基于静态方法而非实例方法.
    • immutable-ext - FantasyLand扩展 Immutable.js.
    • daggy - 用于创建标记构造函数的库.
    • Sanctuary - Sanctuary可以编写安全代码而无需空检查.
    • monet.js - 通过提供丰富的Monad和其他有用功能来协助功能编程的库.
    • union-type - 用于定义和使用联合类型的小型JavaScript库.
    • freeky - 免费monad的集合.
    • Fluture - 包含控制实用程序，高性能和出色错误消息的Future库.
    • fantasy-combinators - 常见的组合器.
    • fantasy-birds - Haskell包的端口Data.Aviary.Birds. 一切都满足您的组合需求.
    • crocks - 一系列流行的代数数据类型，其主要目标是策划并提供每种类型之间的通用接口，以及实现所需的所有辅助功能.
    • kudo-js - 一个小实用程序库，带有一组代数数据类型和辅助函数，可帮助您在Javascript函数编程风格中编写代码
    • purify - 针对ADT的TypeScript功能编程库
    • MojiScript- 一种异步优先，自以为是，功能强大的语言，旨在与JavaScript引擎100％兼容

    Lenses¶

    • lenses - 可组合 kmett 风格镜片.
    • optics - Profunctor光学器件（透镜，棱镜，iso）.
    • ramda-lens – Lens library built on Ramda.
    • fantasy-lenses - 可组合，不可变的getter和setter. （Profunctor镜头WIP）
    • nanoscope - 镜头支持.
    • partial.lenses - 部分镜头是一个全面的，高性能的JavaScript光学库.
    • shades - 用于Javascript的lodash风格的镜头式库.

    Functional Languages that Compile to JavaScript¶

    • ClojureScript - 编译 Clojure，一个带有不可变持久数据结构的托管Lisp到JavaScript.
    • Elm - 一种类型安全的函数式编程语言，用于声明性地创建基于Web浏览器的图形用户界面. 在Haskell中实现.
    • Fable - 编译 F# 可读的JavaScript.
    • PureScript - 一种编译为JavaScript的小型强类型编程语言.
    • Idris - 具有依赖类型的通用纯函数编程语言.
    • GHCJS – Haskell 到JavaScript编译器，基于GHC.
    • ElixirScript - 编译的子集 Elixir，一种动态的，功能性的语言，旨在为JavaScript构建可伸缩和可维护的应用程序.
    • Js_of_ocaml - 编译 OCaml 字节码到JavaScript，可以在浏览器中运行OCaml程序.
    • BuckleScript - JavaScript后端 the OCaml compiler.
    • Reason - Reason是OCaml的新界面，OCaml是一种高度表达的ML语言方言，具有类型推断和静态类型检查功能.
    • Scala.js - 编译 Scala 到JavaScript.
    • LiveScript - LiveScript可以直接映射到JavaScript，并允许您编写没有重复样板的富有表现力的代码.
    • Quack - 一种多范式编程语言，具有针对PHP和JS的渐进式和鸭式打字.
    • ion - 没有BS JS
    • RamdaScript - 一个Lisp，它编译在RamdaJS之上的自以为是的JavaScript
    • RacketScript - RacketScript旨在利用JavaScript和Racket的生态系统，并使它们之间的互操作性干净顺畅.
    • Koka - Koka是一种面向函数的编程语言，它使用熟悉的类似JavaScript的语法将副作用计算中的纯值分离出来.

    Programming Tools¶

    • eslint-plugin-fp - 函数式编程的ESLint规则
    • 4.5 - 功能断言库. 如果您更喜欢JavaScript中的函数式API和实践，那么这个库旨在通过面向函数的断言来解决这个问题，这些断言是懒惰的和一元的.
    • hm-def - 使用Hindley Milner签名检查JS的运行时类型.
    • ftor - 可插入的运行时类型检查器和功能调试工具，通过Scott Encoding支持参数和行多态，隐式rank-2类型和代数数据类型.

    Resources¶

    Books¶

    • Professor Frisby’s Mostly Adequate Guide to Functional Programming - 这是一本关于功能范例的书，通常使用世界上最流行的函数式编程语言：JavaScript. 这是一个实用的介绍，通过现实世界的例子建立直觉. 强力推荐. 通过 Brian Lonsdorf (2016)
    • Functional-Light JavaScript - 本书探讨了可应用于JavaScript的函数式编程（FP）的核心原理. 但是，这本书的不同之处在于，它没有使用繁琐的术语来接近这些原则.
    • JavaScript Allongé ，“六”版. 从尽可能少的功能开始 - 但不能少！ - 并建立强大的组合器和装饰器. 一本基础书. 通过 Reginald Braithwaite (2016)
    • Functional Programming in JavaScript 教授JavaScript开发人员的功能技术，这些技术将提高可扩展性，模块性，可重用性，可测试性和性能. 通过具体示例和无术语解释，本书将教您如何将函数式编程应用于实际开发任务. 作者：Luis Atencio（2016）
    • Eloquent JavaScript . 使用JavaScript编程的现代介绍. 作者：Marijn Haverbeke（2014）
    • Functional JavaScript 通过使用JavaScript的函数式编程支持，教授如何创建美观，安全，易于理解和测试的代码. 通过 Michael Fogus (2013)

    Articles¶

    • FP Concepts in JavaScript - JavaScript中的函数式编程概念简介. 通过最简单的示例，使用Ramda库来教授合成，无点样式和仿函数的概念.
    • Functional programming with JavaScript - JavaScript中的函数式编程的另一个介绍，重点关注三个关键主题：计算作为函数的应用，无状态，避免副作用.
    • A gentle introduction to functional JavaScript - 由JavaScript组成的四部分系列介绍函数式编程，可帮助您快速了解函数式编程的所有宣传内容.
