    无需登录web应用

    Awesome Web Apps (that work without login)¶

    精选的令人敬畏的网络应用程序列表，无需您创建帐户即可使用.

    Inspiration¶

    为了拯救世界，不为每个该死的东西创建用户帐户和安装软件应用程序.

    Some pointers¶

    此列表中链接的Web应用程序无需登录即可提供其大部分核心功能. 由于缺乏适当的替代方案，一些需要注册的网站也被列入此列表. 他们的列表中标有[Account]. 此处提到的大多数应用程序都提供某种用户帐户功能来保存历史记录和进度. 但如前所述，用户帐户不必使用这些应用程序的主要功能.

    Audio and Video¶

    • bfxr - 为您的游戏制作声音效果.
    • MP3 Cut - 在线音频切割机.
    • TwistedWeb Online - 强大的音频编辑器. 文件未在线保存.
    • Filelab Audio Editor - 音频编辑器，具有修剪，合并，剪切音频等基本功能.
    • Until AM - 在线音乐混音器，DJ软件.
    • YOU.DJ - 在线音乐混音器，DJ软件从SoundCloud采购音乐.
    • VideoToolbox [帐户] - 具有所有常见编辑功能的视频编辑器. 尺寸限制在600mb.
    • Stupeflix [帐户] - 用图像，音频和视频剪辑制作电影.

    Business and Finance¶

    • ShapeShift.io - 即时加密货币转换.
    • Cryptrader.com - 加密货币价格图表和指标. 保存设置需要一个帐户.
    • TradingView.com - 来自各种交易所的实时信息和市场见解. 需要帐户才能保存设置.
    • ICOStats.com - 跟踪和比较ICO的表现. 显示自ICO以来的ROI，自ICO以来的ROI与ETH等详细统计数据，以及用于比较ICO历史绩效的图表.
    • InvoiceToMe - 使用您公司的详细信息从各种模板生成专业发票.

    Communication¶

    • Tlk.io - 使用您希望的任何URL（例如tlk.io / some-room）创建一个房间并分组聊天.
    • Discord - 在线语音和文字会议. 在开始时询问用户名以创建临时化身.
    • Gruveo - 在线视频会议. 就像Tlk.io一样，易于记住自定义URL.
    • Appear.in - 在线视频会议. 创建任何房间，如appear.in / some-room并开始聊天.
    • Videolink2 - 在线视频会议. 链接生成，共享链接以邀请其他参与者.
    • GetARoom.io - 在线视频会议. 有移动浏览器支持，只需点击开始新呼叫并共享链接.
    • FaxZero - 免费发送在线传真.

    Document Editors¶

    Latex¶

    • Papeeria - 乳胶预览，可选择下载PDF格式而无需注册.
    • Overleaf [帐户] - 无限制的协作者和项目，免费帐户中的git访问权限.
    • Sharelatex [帐户] - 具有无限私人项目的Latex编辑器. 1个免费帐户合作者.

    Markdown¶

    • Dillinger - 强大的Markdown WYSIWYG，可导出为HTML和PDF. 用户帐户保存文档.
    • Github.md - 带打印功能的简单GitHub风格Markdown（GFM）预览.
    • HackMD - 具有预览功能的协作，强大，实时的在线降价编辑器.
    • Markdown Live Preview - 即时降价预览.
    • Markdownify - Minimal Markdown编辑器.
    • StackEdit.io - 丰富的浏览器降价编辑器，可脱机工作.

    Others¶

    • rst.ninjs - WYSIWYG RST预览，可选择在线保存文档以供日后编辑.
    • EtherCalc - 在线电子表格编辑器，导出到Excel.
    • PdfEscape - 在浏览器中编辑或创建PDF.
    • Browserpad - 浏览器中的无服务器纯文本编辑器. 允许您打开并保存纯文本文件.
    • WriteURL - 协作实时在线文本编辑器.

    Drawing¶

    • SVG-Edit - 功能强大的SVG编辑器，具有许多功能. 开源.
    • Method Draw - 干净且功能丰富的SVG编辑器. 出口到巴布亚新几内亚.
    • A Web Whiteboard - 触摸友好的在线协作白板应用程序.
    • Witeboard - 为您的团队提供实时白板. 无需注册.
    • Draw.io - 流程图和UML绘图工具. 有很多绘画风格可供选择. 导出到PNG并与Google云端硬盘集成.
    • AutoDraw - 借助谷歌的机器学习，绘制精美的草图. 图纸可以下载为PNG.
    • kleki - MS Paint在浏览器中的界面.
    • bubbl - 创建思维导图，在浏览器中集思广益.
    • ScribbleMaps - 创建自定义谷歌地图.
    • Sketchpad - 创建数字艺术作品并导出为流行的图像格式JPEG，PNG，SVG和PDF.

    File Converters¶

    • Cloud Convert - 从任何东西转换为任何东西，包括音频，视频，字体，文档等等. 与Google Drive和Dropbox等云存储服务实现出色的集成. 直接链接.
    • Online Convert - 收集音频，视频，图像，文档，电子书和存档转换器. 转换时没有文件大小限制和直接下载链接.
    • PDF <=> EPUB - PDF到EPUB转换器，反之亦然.
    • Pandoc Try - 使用Pandoc进行帧间标记转换. 支持 - Latex，Markdown，HTML，RST等等.
    • Print Friendly - 将任何网页转换为简单格式. 然后下载或打印为PDF.
    • OnlineOCR - 将扫描的PDF转换为DOC，XLS或TXT.
    • GifDeck - 将幻灯片从幻灯片转换为GIF.
    • favicon-generator - 通过上传所需图片，为您的Android或iOS应用生成网络应用或图标的图标.

    File Hosting/Sharing¶

    • EFShare - 点对点安全文件共享.
    • Ge.tt - 文件托管服务. 最大限制2gb.
    • RGhost - 文件托管服务，Max限制100mb无需登录.
    • ImgSafe - 图像托管服务与小直接链接.
    • Clyp - 无限制的音频共享，丰富的API.
    • SendVid - 视频托管服务.
    • Archive.org [帐户] - 任何类型的无限文件托管，带宽和上传大小没有限制.
    • imgh - 图像托管服务. 支持所有主要格式. 文件大小限制为16mb.
    • MultCloud - 云服务，用于在多个云服务之间管理，移动，复制和迁移数据. 支持所有主要的云服务. 无需注册，2TB云存储，直接将大文件下载到云端，没有大小限制.
    • Firefox Send - Mozilla的免费文件共享程序. 最多1GB无帐户，2.5gb帐户.

    Games¶

    • Typeracer - 在与其他人比赛时评估您的打字速度.
    • Spark Chess - 学习下棋，与全球用户一起玩或根据您的技能水平玩AI.
    • Miniclip - 收集各种免费游戏，无需发送电子邮件即可播放. 需要Flash.
    • CrossCode - 在遥远的未来设置复古风格的2D游戏. 配有精彩的游戏机制，如连击，拼图，技能树，任务，物品等.

    Graphics, Image and Design¶

    • PhotoPea - 图像编辑器，看起来像Photoshop，具有大部分功能.
    • Pixlr Editor - Pixlr的图像编辑器. 很多功能.
    • Photo Editor | Befunky - 具有大量滤镜和效果的图像编辑器.
    • Logo Makr - 徽标制造商，收藏了大量的SVG. 需要付款才能将徽标导出为SVG（PNG是免费的）.
    • Online Logo Maker - 在线制作徽标. 额外的用户帐户功能，以保存创作.
    • Coolors - 调色板发生器.
    • Tiny PNG - 智能PNG和JPEG压缩.
    • Realtime Board [帐户] - 具有大面积，屏幕共享，视频和语音通信的协作黑板.
    • ezGIF - GIF图像编辑器（裁剪，缩放，优化等）.
    • Adobe Color CC - 配色方案设计师.
    • Burner Bonanza [帐户] - 删除图像背景并立即在Facebook或Twitter上分享的工具，登录以PNG或JPEG格式下载编辑后的图像.
    • BG patterns - 图案创建工具，允许您从各种形状中进行选择，然后自定义大小，旋转，位置，颜色，纹理等.
    • IcoMoon - 一个平台，可以帮助您找到矢量图标包，托管图标作为SVG或字体，并生成所选图标的SVG + PNG +图标字体.
    • Flaming Text - 在线徽标或名称生成器/创建者，根据您的意愿修改您的徽标，并获得包括PNG和JPEG在内的多种格式.

    Internet Downloaders¶

    • savefrom.net - 从YouTube，Dailymotion，Vimeo等下载.
    • Youtube Converter & Downloader - Download YouTube video in any audio/video format.
    • keepvid - YouTube，SoundCloud，Facebook下载器.
    • Bitport [帐户] - Torrent下载器. 免费帐户限制为每天1洪流1gb.
    • Vubey - 以各种音频质量（比特率）从YouTube，SoundCloud等下载MP3.
    • Seedr [Account] - 基于云的torrent下载器. 限制为免费帐户中总容量为2gb的多个种子. 无限带宽. 贡献新用户增加500mb.
    • ZbigZ - Torrent下载器，可让您选择要从torrent下载的文件. 文件大小限制为2GB，免费版下载限制为50kbps.
    • Instagram Story Downloader - Instagram Story Downloader让您只需输入用户名即可下载公共帐户的故事.
    • Instagram Media Downloader - 允许您下载Instagram图像和视频. 支持多个图像/视频帖（Sidecar）.

    Music, Radio and Podcasts¶

    • Podbay - 在线收听任何播客，可以下载个人剧集.
    • Internet Radio - 收听超过40k的广播电台.
    • Stitcher - 在线收听播客.
    • AnyAudio - 流式传输和下载音乐.
    • Muki - 基于WebAudio的chiptune和MIDI播放器，带有很多视频游戏音乐.
    • MusixHub - 在线播放艺术家和专辑视频歌曲，由YouTube提供支持.
    • plug.dj - 收听成千上万的DJ，由YouTube提供支持，登录成为DJ.
    • Jango Radio - 免费的网络收音机，收听数百个流派电台或用您喜欢的音乐创建自己的电台.
    • Anime Openings - 免费动漫电台，收听或下载动漫开场和结束视频.
    • NoteFlight [帐户] - 打印音乐表，在线编写自己的音乐（评论）.
    • ongaku - 在线动漫音乐电台，与 desktop port.
