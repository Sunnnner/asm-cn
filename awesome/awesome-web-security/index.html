    Awesome Web Security ¶

    >策划的Web安全材料和资源列表.

    毋庸置疑，大多数网站都存在各种类型的错误，最终可能导致漏洞. 为什么这会经常发生？ 可能涉及许多因素，包括配置错误，工程师的安全技能不足等.为了解决这个问题，这里列出了用于学习尖端渗透技术的Web安全材料和资源，我强烈建议您阅读本文“So you want to be a web security researcher?“先.

    请阅读 contribution guidelines 在贡献之前.

    🌈 Want to strengthen your penetration skills?
    I would recommend playing some awesome-ctfs.

    如果您喜欢这个很棒的列表并想支持它，请查看我 Patreon 页面:)
