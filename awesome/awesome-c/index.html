    awesome-c ¶

    C好东西的精选列表. 此列表仅包含* free software 代码和卖家谁不是物质资源的邪恶.

    寻找维护者：此列表目前非常不活跃. 我正在寻找可以让它再次成为伟大的维护者. 看到 #26 了解更多信息，但过程很简单：在此列表中添加或修复一些内容，您将成为850多星级列表的贡献者.

    此列表以前由以下人员维护 @kozross . 他决定将名单改为a new platform所以我决定将它分叉，以便我们可以将它保存在GitHub上.

    Contents¶

    Build Systems¶

    这些是在C中自动构建和测试项目的工具.

    • aimake - 构建工具旨在避免复杂的配置. GNU GPL3 or later
    • Autoconf - 可扩展的M4宏包，可生成用于自动配置软件源代码包的shell脚本. 部分Autotools. GNU GPL3 or later
    • Automake - 用于自动生成符合GNU编码标准的Makefile.in文件的工具. 需要使用Autoconf. 部分Autotools. GNU GPL3 or later
    • CMake - 用于跨平台构建，测试和打包的工具. 3-clause BSD
    • Jam - 构建系统，旨在比制作更容易. 隐式理解C构建规则. Jam License
    • Libtool - 通用库支持脚本. 部分Autotools. GNU GPL3 or later
    • Meson - 极其快速，用户友好的构建系统. 基于忍者. Apache 2.0
    • Premake - 为Visual Studio，Xcode或GNU Make等工具集构建脚本生成器. 3-clause BSD
    • Ninja - 小巧，简单的构建系统，专注于速度. Apache 2.0
    • Bazel - 为各种操作系统和目标构建系统. Apache 2.0
    • Buck - 构建由Facebook创建和使用的系统. Apache 2.0
    • Qbs - 用于软件项目的现代构建工具. LGPL3
    • qmake - 构建Qt框架附带的系统. GNU GPL3 with Qt Exception
    • xmake - 基于lua的跨平台构建实用程序. Apache 2.0

    Compilers¶

    • Clang - 用于LLVM的C编译器. 支持C11. NCSA
    • CompCert - 完全验证的C编译器. 支持几乎所有的C89. GNU GPL2.1 or later
    • GCC - Provides a C compiler as part of its compiler set. Supports C11 and OpenMP. GNU GPL3 or later
    • PCC - 尊敬的C编译器. 支持C99. Various free licenses
    • TCC - Tiny C编译器; 一个小而快速的C编译器. 支持C99（复杂类型除外）. GNU LGPL2.1
    • Intel SPMD - 编译器用于C语言的变体，用于单个程序，多个数据编程. Various licenses
    • ccache - 编译器缓存旨在加速重新编译. GNU GPL3 or later

    Compiler libraries¶

    用于编译器，解释器和基于低级项目的库.

    • yasm - Yasm模块化汇编项目. 2-clause BSD

    Compression¶

    • libzip - 用于读取，创建和修改zip存档的AC库. 3-clause BSD
    • lzo - 非常快速的数据压缩库. GNU GPL2.1
    • zlib - 大量漂亮但精巧不引人注目的压缩库. 3-clause BSD
    • bzip2 - 免专利，高质量的数据压缩器. 3-clause BSD
    • lz4 - 快速压缩算法.
    • snappy - 快速压缩库（在C ++中实现，与C的本机绑定）.
    • zstd - 快速实时压缩算法 3-clause BSD
    • quicklz - 快速压缩库. GNU GPL2.1
    • pixz - Pixz（发音为pixie）是xz的并行索引版本. 2-clause BSD

    Crypto¶

    • GnuTLS - 安全通信库，实现SSL，TLS和DTLS. GNU LGPL2.1 or later
    • libgcrypt - 通用加密库，具有一系列可用密码. GNU LGPL2.1or later (code) 和 GNU GPL2.1 or later (manual 和 tools)
    • OpenSSL - SSL和TLS协议的实现，还包括加密库. Dual Licensed under the OpenSSL License and the SSLeay License
    • libsodium - 现代且易于使用的加密库. Expat
    • libtomcrypt - 相当全面的模块化和便携式加密工具包. Public Domain
    • mbed TLS - C的另一个加密实现 Apache 2.0
    • libressl - OpenSSL的现代化分支. Various Licenses

    Database¶

    这列出了使用C API的数据库和数据存储.

    • BerkeleyDB - 用于键值数据的高性能嵌入式数据库的库. GNU AGPLv3
    • Hiredis - Redis的简约客户端库. 3-clause BSD
    • LMDB - 超快速，超紧凑的键值嵌入式数据存储. OpenLDAP License
    • MariaDB - 强大，可扩展且可靠的SQL服务器，旨在成为MySQL的直接替代品. 3-clause BSD
    • mongo-c-driver - 高性能客户端库 MongoDB. Apache 2.0
    • PostgreSQL - 强大的对象关系数据库系统. PostgreSQL licence
    • recutils - 一组工具和一个C库，用于访问称为recfiles的人类可编辑的纯文本数据库文件. GNU GPL3 or later
    • Redis - 高级键值存储. 3-clause BSD
    • sophia - 现代可嵌入的键值数据库. FreeBSD
    • SQLite - 具有C接口的独立，无服务器，零配置，事务性SQL数据库引擎. Public Domain
    • UnQLite - 具有C接口的独立，无服务器，零配置，事务性NoSQL引擎. FreeBSD

    Deep Learning¶

    • Darknet - 用C和CUDA编写的开源神经网络框架. 它快速，易于安装，并支持CPU和GPU计算.

    Documentation Generation¶

    • Cxref - 以LaTeX，HTML，RTF或SGML生成C程序的文档. GNU GPL2.1
    • DocOnce - 适度标记的标记语言，可用于生成一系列格式. 3-clause BSD
    • Doxygen - The de-facto standard tool for generating C documentation from annotated sources. Can generate a large range of formats. GNU GPL2.1
    • GTK-Doc - 用于从带注释的源生成C文档的工具. 支持Autotools. GNU GPL2.1 (code) 要么 GNU FDL1.1

    Editors¶

    这些是特别高级的IDE类编辑器. 如果您想要程序员的文本编辑器，请查看其他地方. 此外，无论你做出什么样的选择，最有可能支持C.

    • Anjuta DevStudio - GNOME IDE. GNU GPL2.1
    • Code::Blocks - 可扩展，可配置的IDE支持C. GNU GPL3
    • CodeLite - 跨平台IDE. GNU GPL2.1
    • Eclipse - 用Java编写的IDE. EPL
    • Geany - 非常小巧，快速的IDE. GNU GPL2.1 or later
    • KDevelop - KDE IDE. GNU GPL2.1
    • Qt Creator - 使用C ++和Qt编写的跨平台IDE，Qt SDK的一部分. 支持Clang代码模型. GNU GPL3 with Qt exception
    • Visual Studio Code - 代码编辑，重新定义. Visual Studio代码.

    Embedded Systems¶

    这些项目允许人们使用微控制器和其他资源受限的架构.

    RTOS¶

    实时操作系统（RTOS）旨在为以低延迟延迟处理数据的实时应用程序提供服务.

    • Amazon FreeRTOS - 用于微控制器的RTOS，可使小型低功耗边缘器件易于编程. MIT
    • ChibiOS - 具有完整开发环境的RTOS（HAL，驱动程序，支持文件和工具）. Various Licenses
    • Contiki - 将低成本，低功耗微控制器连接到Internet. 3-clause BSD
    • Zephyr Project - 可扩展的实时RTOS，支持多种硬件架构，针对资源受限设备进行了优化. Apache 2.0
    • RIOT OS - 实时多线程操作系统，支持通常在物联网中找到的一系列设备. LGPL-2.1

    Environments¶

    这是一系列技术，旨在将Windows带入21世纪，以支持C语言.

    • Cygwin - 旨在在Windows下广泛模拟POSIX兼容环境. Various Licenses
    • MinGW-w64 - Windows上具有64位支持的C开发的极简环境. Various Licenses

    Frameworks¶

    本节包含大型库，可提供数据结构以及您对“现代”标准库所期望的其他内容.

    • APR - Apache Portable Runtime; 另一个跨平台实用程序函数库. Apache 2.0
    • C Algorithms - C的通用算法和数据结构的集合 ISC
    • CPL - 公共管道图书馆; 一组库，旨在成为一个全面，高效和强大的软件工具包. GNU GPL2.1
    • EFL - 大量有用的数据结构和功能.
    • GLib - 实用功能和结构库，旨在便携，高效和强大. GNU LGPL2.1
    • GIO - 现代且易于使用的VFS API. GNU LGPL2.1
    • GObject - Object-oriented system and object model for C. GNU LGPL2.1
    • libnih - C函数和结构的轻量级库. GNU GPL2.1
    • libU - 基本实用程序的小型库，包括内存分配，字符串操作和日志记录. 3-clause BSD
    • PBL - 大型实用程序库，包括数据结构等. GNU LGPL2.1 要么 later (library) 要么 GNU GPL2.1 要么 later (test code)
    • qlibc - 简单而强大的C库，旨在替代GLib，同时专注于小巧轻便. qLib license （相近 FreeBSD)
    • stb - C的单文件库范围. Public Domain
    • libsrt - C的安全实时库. 3-clause BSD

    Game Programming¶

    Engines¶

    这些是作为C游戏编程代码的示例提供的.

    • Corange - 纯C中的游戏引擎 FreeBSD
    • Darkplaces - Quake2引擎的修改版本. GNU GPL2.1
    • ioquake3 - Quake3引擎，最后解放了. GNU GPL2.1
    • Orx - 便携，轻量，基于插件，数据驱动，面向2D的游戏引擎. zlib
    • Quake - 地震引擎. GNU GPL2.1
    • Quake2 - Quake2引擎. GNU GPL2.1
    • Spearmint - 为FPS游戏设计的引擎. GNU GPL3 or later

    Resources¶

    这些是对游戏编程有用的各种库.

    • Allegro - 跨平台，视频游戏开发和多媒体库. zlib
    • Chipmunk2D - 快速轻巧的2D游戏物理库. Expat
    • CSFML - 绑定 SFML 在C. zlib
    • FreeGLUT - OpenGL Utility Toolkit的替代品. 允许使用OpenGL上下文创建和管理窗口. X11
    • GLFW - 用于使用OpenGL上下文创建窗口的多平台库. zlib
    • libao - 具有各种输出的跨平台音频库. GNU GPL2.1 or later
    • RetroArch - 参考前端 libretro. GNU GPL3
    • SDL and SDL2 - 跨平台开发库，旨在通过OpenGL提供对音频，键盘，鼠标，操纵杆和图形硬件的低级访问. SDL2是最新版本. zlib
    • raylib - 简单易用的库，用于学习视频游戏编程. zlib

    Generic Programming¶

    • klib - 通用算法和数据结构的小型轻量级实现. MIT

    Graphics¶

    • Cairo -2D图形库. GNU LGPL2.1 要么 MPLv1.1
    • Cogl - GPU图形和实用程序API. Expat
    • Clutter - 基于OpenGL的UI库. GNU LGPL2.1
    • giflib - 用于读取和写入gif图像的库. MIT
    • heman - 处理高度图，法线贴图，距离场等的图像实用程序的小型库. Expat
    • libcaca - 基于终端的接口的ASCII渲染器. WTFPLv2
    • libimagequant - 小型便携式库，用于将RGBA图像高质量转换为8位索引彩色图像. FreeBSD
    • libjpeg-turbo - 更快的库，用于读取和写入JPEG文件. Various Licenses
    • libpng - 官方PNG参考图书馆. libpng license
    • libxmi - 用于光栅化2D矢量图形的函数库. GNU GPL3 or later
    • mozjpeg - 改进的JPEG编码器. 3-clause BSD

    Graphic APIs¶

    • OpenGL - 行业采用2D和3D图形API. 更多资源 awesome-opengl.
    • OpenGL ES - 行业采用2D和3D图形API，用于移动和嵌入式设备.
    • OpenGL SC - 符合行业要求的系统安全认证的图形和计算标准.
    • Vulkan - 用于现代跨平台开发的显式图形和计算API. 更多资源 awesome-vulkan.

    Graphical User Interface¶

    这些是具体 widget toolkits.

    • GTK+ - 跨平台小部件工具包. GNU LGPL2.1
    • IUP - 另一个跨平台的小部件工具包. Expat
    • Tk - 基本小部件工具包. Tcl / Tk的一部分. Tcl/Tk License
