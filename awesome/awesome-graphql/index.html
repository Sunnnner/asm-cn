    Libraries¶

    JavaScript Libraries¶

    Clients¶

    • relay - Relay是一个用于构建数据驱动的React应用程序的JavaScript框架.
    • Apollo Client - 适用于每个UI框架和GraphQL服务器的功能齐全的生产就绪缓存GraphQL客户端.
    • aws-amplify - 由Amazon开发的用于缓存，分析等的客户端库，其中包括一种获取GraphQL查询的方法.
    • graphql-hooks - 最小的钩子 - 第一个具有缓存和服务器端渲染支持的GraphQL客户端.
    • graphql-request - 用于节点和浏览器的最小GraphQL客户端.
    • FetchQL - 使用Fetch的简单GraphQL查询客户端.
    • urql - React的简单缓存GraphQL客户端.
    • micro-graphql-react - 用于将GraphQL添加到React的轻量级实用程序. 组件. 包括简单缓存并使用GET请求，这些请求还可以通过服务工作者缓存.
    • Lokka - GraphQL的简单JavaScript客户端，您可以在任何地方使用它.
    • react-reach - 通过Redux与Graphql通信的库.
    • Grafoo - 一个小型但完全成熟的基于缓存的GraphQL客户端

    HTTP Server Bindings¶

    • express-graphql - GraphQL Express中间件.
    • hapi-graphql - 使用Hapi创建GraphQL HTTP服务器.
    • hapi-plugin-graphiql - 用于GraphiQL集成的HAPI插件.
    • graphql-api-koa - GraphQL Koa中间件，从头开始实现GraphQL.js并支持本机ESM.
    • koa-graphql -GraphQL Koa Middleware.

    Database & ORM¶

    • graphql-bookshelf - 一些帮助定义BookshelfJS模型周围的GraphQL架构.
    • graphql-sequelize - GraphQL的Sequelize助手.
    • graphql-thinky - 从Thinky RethinkDB模型构建优化的GraphQL架构.
    • graphql-tools-sequelize - 使用Sequelize ORM的GraphQL-Tools的OO样式的模式定义函数和模式解析器函数
    • graysql-orm-loader - 从ORM加载GraphQL架构的GraysQL扩展.
    • GRelDAL - 一个简单的微框架，用于关系数据存储和GraphQL API之间的双向映射.

    Miscellaneous¶

    • GraphQL.js - GraphQL for JavaScript的参考实现.
    • codemirror-graphql - CodeMirror的GraphQL模式和帮助程序.
    • graphql-schema - 使用流畅/可链接的界面创建GraphQL模式.
    • graphql-tools-types - 用于GraphQL-Tools的自定义GraphQL类型（Void，Int，Float，String，Date，UUID，JSON）
    • graphql-normalizr - 规范化GraphQL响应以持久保存在客户端缓存/状态中
    • babel-plugin-graphql - 编译GraphQL标记模板字符串的Babel插件.
    • adrenaline - 考虑Redux和Relay的绑定.
    • graph.ql - 用于创建和查询GraphQL模式的更快更简单的技术.
    • Strapi - 开源即时支持“GraphQL”的开源Node.js框架.
    • GraysQL - GraphQL管理器和加载器.
    • Annotated GraphQL - GraphQL中注释的概念证明（即：将现有REST API转换为GraphQL端点）.
    • graphql-tools - 用于构建和维护GraphQL-JS服务器的工具库.
    • graphql-anywhere - 在没有架构的情况下，针对任何数据在任何地方运行GraphQL查询.
    • graphql-tag - 解析GraphQL查询的JavaScript模板文字标记.
    • modelizr - 一个库，用于简化编写GraphQL查询，模拟它们并规范化响应的过程.
    • vue-apollo - 阿波罗的Vue集成.
    • graphql-pouch - 由GraphQL简写表示法创建的PouchDB之上的GraphQL-API运行时作为具有CouchDB同步的自包含服务.
    • gql-tools - 带有CLI的工具库，用于模式生成和操作.
    • graphql-iso-date - 与GraphQL.js一起使用的GraphQL日期标量类型. 此标量表示ISO 8601格式YYYY-MM-DD的日期.
    • graphql-compose - 工具，允许您通过插件从不同的数据源构建灵活的graphql架构.
    • node-graphjoiner - 使用连接，SQL或其他方式创建GraphQL API.
    • Join Monster - 用于批量数据提取的GraphQL-to-SQL查询执行层.
    • graphql-factory - 从JSON定义创建GraphQL类型
    • type-o-rama - JS类型系统的交互性.
    • GraphiteJS - GraphQl的框架NodeJS.

    Relay Related¶

    • graphql-relay-js - 一个帮助构建支持react-relay的graphql-js服务器的库.
    • sequelize-relay - 将sequelize和graphql-relay-js连接在一起的Serverside库.
    • babel-relay-plugin - 用于转换用于Relay的GraphQL查询的Babel Relay插件.
    • react-router-relay - React Router的中继集成.
    • relay-local-schema - 使用没有GraphQL服务器的中继.
    • relay-sink - 使用Relay在React组件之外获取和存储数据.
    • recompose-relay - 为Relay重新组合助手.
    • Graylay - GraysQL扩展，用于创建与中继兼容的模式.
    • react-relay-network-layer - 用于Relay的网络层，具有查询批处理和中间件支持（urlThunk，retryThunk，auth，defer等）.
    • relay-subscriptions - 对Relay的订阅支持.
    • Portfolio Relay Example - 一个示例网站，从各种api获取数据，并使用Relay和GraphQL将数据提供给React组件！

    Ruby Libraries¶

    • graphql-ruby - Ruby的GraphQL实现.
    • graphql-client - 用于声明，编写和执行GraphQL查询的Ruby库.

    PHP Libraries¶

    • graphql-php - GraphQL参考实现的PHP端口.

    • laravel-graphql （废弃） - Laravel 5的Facebook GraphQL.
    • laravel-graphql-relay - Laravel库，帮助构建支持反应中继的服务器.
    • graphql-mapper （ abandoned ） - 该库允许基于您的模型构建GraphQL架构.
    • graphql-bundle - Symfony 2的GraphQL Bundle.
    • overblog/graphql-bundle - 此捆绑包提供了在Symfony App中构建完整GraphQL服务器的工具. 支持反应中继.
    • GraphQL Symfony Bundle - Symfony 3的GraphQL Bundle（支持2​​.6+）.
    • graphql-wp - 公开GraphQL端点的WordPress插件.
    • eZ Platform GraphQL Bundle - 用于eZ Platform Symfony CMS的GraphQL Bundle.
    • GraphQL Middleware - GraphQL Psr7中间件
    • Zend Expressive GraphiQL Extension - zend表达式的GraphiQL扩展
    • GraphQLite - 框架无关的库，允许您通过注释PHP类来编写GraphQL服务器.

    Python Libraries¶

    • graphql-parser - 用于Python的GraphQL解析器.

    • flask-graphql - 为Flask应用程序添加GraphQL支持.

    • python-graphjoiner - 使用连接，SQL或其他方式创建GraphQL API.

    • Flask-GraphQL-Auth - 来自flask-jwt-extended的Flask认证库.

    Java Libraries¶

    • graphql-java - GraphQL Java实现.
    • gaphql-java-type-generator - 自动生成用于GraphQL Java的类型

    • graphql-spring-boot - GraphQL和GraphiQL Spring Framework启动程序.
    • vertx-graphql-service-discovery - 异步GraphQL服务发现和查询您的微服务.
