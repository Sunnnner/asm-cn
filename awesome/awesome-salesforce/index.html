    IDE¶

    • C9.IO - 在线编辑器，快速连接到任何Salesforce组织.
    • Illuminated Cloud - JetBrains IntelliJ IDEA中托管的强大的Force.com开发工具.

    Packages and Libraries supporting salesforce¶

    • Apex mocks - ApexMocks是Force.com Apex语言的模拟框架.它源于着名的Java模拟框架Mockito的灵感.

    • Apex Commons - 支持Apex企业模式的Common Apex Library等等！ 必须部署 ApexMocks 在部署这个库之前.另见 MavensMate Templates

    • Apex Commons Sample Code - 示例应用程序，说明Apex企业模式库 必须部署 Apex Common 在部署此库之前

    • Apex Metadata API - Salesforce Metadata API的Apex Wrapper

    • Apex Core HttpCallout - Apex HTTP标注框架.

    该库已由五个库扩展
    * Google Cloud Print * Google Drive * Google Mirror * Google Sheets * Dropbox

    该库的主要功能 + HTTP标注的框架. + JSON序列化和反序列化基类. + OAuth Client - 扩展基础AbstractClient以提供OAuth身份验证所需的标注.

    • Polyforce - PolyForce是一个以创造聚合物应用/组件的动机创建的项目，从Salesforce获取数据.该想法创建了一组强大的聚合物组件，可以相互共享，另一方面提供工作样本开发人员学习并获得灵感来构建更多组件

    • Salesforce to Github - Apex for Force.com的Github API客户端

    • Apex UML Digram Maker - 使用Tooling API从Apex代码创建UML图表

    • SfApexDoc - 功能齐全的Apex文档生成器，类似于JavaDoc.

    • Visualforce TableGrid - TableGrid是一个免费的开源Force.com库，为用户和开发人员提供高度可定制，本机外观，可排序，可过滤，可编辑的Grid Visualforce组件. 此组件可用作高级，高度可配置（由开发人员和用户）替换apex：pageBlockTables和标准相关列表

    • Force.com ESAPI - Force.com平台上的Apex语言的Enterprise Security API. 该库由三个模块组成 - 验证，输出编码和访问控制（CRUD，FLS和共享）.

    • R.apex - Apex的功能库. 该库受Ramda.js的启发，提供了一系列实用程序函数来操作Apex集合，以及对函数式编程的广泛支持.

    • Force DI - 通用 dependency injection （DI）库，支持在运行时注入Apex，Visualforce，Lightning和Flows.

    • Query.apex - Salesforce.com平台上的SOQL查询构建器，它支持复杂条件，字段和命名空间管理，子关系查询和函数式编程样式编码.

    • Apex Chainable Batch - 可以通过从上一批的finish（）方法调用后续批处理来链接Apex批处理. 但是这种硬编码使得这种模式不灵活. 很难从外部构建链，从中央类构建更直接，也不依赖于业务逻辑构建运行时.

    Developer Utilities¶

    • Weave - 用于生成样板文件Salesforce代码模板的Sublime Text Plugin

    • GitHub Salesforce Deploy Tool - 自动将GitHub存储库部署到Salesforce ORG 和“部署到Salesforce按钮”到您的README.md文件

    • Cinnamon-Cinnamon是一个Force.com应用程序，使您能够构建和运行Selenium测试，以在Salesforce组织中使用Visualforce / Javascript验证自定义UI页面.

    • Push Notification Plugin for iOS and Android- 适用于Android，iOS，WP8和Amazon Fire OS的Cordova Push Notifications Plugin

    • Salesforce1-Dev-Guide-Setup-Package-Salesforce1-Dev-Guide-Setup-Package

    • Analytics-Cloud-Dataset-Utils- 用于将外部数据加载到Analytics Cloud数据集中以及转换现有数据集的实用程序.

    • Workbench- Workbench是一个功能强大的基于Web的工具套件，专为管理员和开发人员设计，可通过Force.com API与Salesforce.com组织进行交互.

    • Playground- Adminbooster Playground是一个运行SOQL / SOSL查询或Apex代码的在线工具，为分析和管理工作效率提供高级功能.

    • CassaCloud - 良好的salesforce调试可帮助您了解Salesforce开发人员控制台的日志. 例如，如果您有带有列表的地图地图，则很难找到其确切位置的值.

    • SoqlXplorer- SoqlXplorer对于使用Salesforce.com平台的开发人员来说是一个很棒的工具.

    • force-dev-tool - 支持Force.com开发生命周期的命令行工具.

    • JSON to Apex- 从JSON字符串生成Apex类.

    • Salesforce IDs converter- 将15个字符的ID转换为18个字符的ID.

    • SOQooLio- 用于查询，排序和导出Salesforce数据的Web应用程序.

    • Sweet.apex - Sweet.apex是Apex开发的下一代产品，它将Apex类的Sweet Apex（功能丰富的类Apex代码）转换为.

    • soql-parser-js - 用于解析SOQL查询，编写SOQL查询和格式化SOQL查询的JavaScript库.

    Open Source Projects Repositories from Salesforce¶

    • Salesforce Git Repo - 在这里获取所有最新的salesforce开源项目，你会发现很棒的项目，框架，salesforce并为世界开放的库
    • ForcedotCom Labs - 从Salesforce Team查看开源和实验项目. 及早访问可以获得功能并提供您的想法的功能

    • Salesforce Mobile UI - Polymer Elements for Salesforce - Mobile UI Elements是一个免费的开源Force.com（不支持）库，用于简化移动应用程序的开发. 该库基于Google的Polymer框架，为创建在智能手机和平板电脑上运行良好的HTML5应用提供了基本构建模块
    • ForceServer - 它支持代理和Web服务器，ForceServer是一个简单的开发服务器，旨在在构建使用Salesforce OAuth和REST服务的应用程序时提供简单和集成的开发人员体验.

    Javascript Libraries for Salesforce¶

    • JSForce - JSforce (f.k.a. Node-Salesforce) - 它封装了对Salesforce在异步JavaScript函数调用中提供的各种API的访问.与其他Salesforce API库不同，它旨在为服务器端和客户端应用程序提供集成接口，因此您不必重写类似的逻辑不同的库只用于在不同的环境中运行.此外，它还有一个有用的命令行界面（CLI），它提供交互式控制台（REPL），因此您可以轻松学习使用.

    • ForceSniffer - Device Detection Library for Salesforce - 简化的库，可检测移动设备，Salesforce1应用程序和其他Saleforce上下文.
    • ForceJS - 现在支持ES6，这个微库设计用于在JavaScript应用程序中使用Salesforce REST API
    • nForce - nforce是force.com，database.com和salesforce.com的node.js salesforce REST API包装器
    • ngForce - 一组Angular.js模块，可在Force.com平台上实现快速，可持续的Angular.js应用程序开发.
    • ForceTK- ForceTK - JavaScript应用程序的最小Force.com REST API
    • ForceEng - Micro-Library在AngularJS应用程序中使用Salesforce REST API
    • Formulon - 在ES6中完全实现的公式解析器. See Demo
    • Scripting Toolkit - 基于Windows的工具包，用于运行本机Javascript以访问Salesforce数据.

    UX Libraries for Salesforce¶

    • Salesforce Lightning Design System - Office UI library based on Lightning Experience Design Language - 使用Lightning设计系统，您可以构建与Salesforce核心功能一致的外观和感觉的自定义应用程序 - 无需对我们的样式进行逆向工程！ 只需下载我们的平台无关的CSS框架，立即开始使用. 展示，使用 Google Material Design on 销售队伍

    • Feather - Google Material Design Mockup for Salesforce - 图书馆
    • React Components for Saleforce Lightning Design System - 使用React构建的Salesforce Lightning Design System组件
    • Salesforce ReactJS SPA Starter - 在Salesforce上创建基于ReactJS的单页面应用程序的模板项目，具有自动构建脚本（Gulp.js）
    • ForceSpinner - Loaders / Spinner集合，以利用Salesforce项目的强大用户体验
    • Avonni Creator - Salesforce原型制作工具，支持复杂的Lightning Design System.

    Mobile Development with Salesforce¶

    Mobile SDK¶

    • Salesfoce MobileSDK for iOS - 如果您想使用SDK本身的源代码，那么您来对地方了！ 您可以浏览示例应用程序源代码并通过层调试，以了解一切如何在幕后工作.

    • Salesfoce MobileSDK for Andriod- 适用于Android的Salesforce Mobile SDK的源存储库.
