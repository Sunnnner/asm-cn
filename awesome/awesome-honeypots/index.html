    Honeypots¶

    • 数据库蜜罐

      • Delilah - 用Python编写的Elasticsearch Honeypot.
      • ESPot - 用NodeJS编写的Elasticsearch蜜罐，捕获每次利用CVE-2014-3120的尝试.
      • Elastic honey - 简单的Elasticsearch蜜罐.
      • HoneyMysql - 简单的Mysql蜜罐项目.
      • MongoDB-HoneyProxy - MongoDB蜜罐代理.
      • NoSQLpot - 在NoSQL风格的数据库上构建的蜜罐框架.
      • mysql-honeypotd - 用C.编写的低交互MySQL蜜罐.
      • MysqlPot - MySQL蜜罐，还处于早期阶段.
      • pghoney - 低交互Postgres蜜罐.
      • sticky_elephant - 中等互动postgresql蜜罐.

    • 网络蜜罐

      • Bukkit Honeypot - Bukkit的Honeypot插件.
      • EoHoneypotBundle - Symfony2表格的蜜罐类型.
      • Glastopf - Web应用程序蜜罐.
      • Google Hack Honeypot - 旨在为使用搜索引擎作为攻击资源的黑客工具的攻击者提供侦察.
      • Laravel Application Honeypot - Laravel应用程序的简单垃圾邮件防护包.
      • Nodepot - NodeJS Web应用程序蜜罐.
      • Servletpot - Web应用程序Honeypot.
      • Shadow Daemon - 用于PHP，Perl和Python应用程序的模块化Web应用程序防火墙/高交互蜜罐.
      • StrutsHoneypot - Struts Apache 2 based honeypot as well as a detection module for Apache 2 servers.
      • WebTrap - 旨在创建欺骗性网页，以欺骗和重定向攻击者远离真实网站.
      • basic-auth-pot (bap) - HTTP基本身份验证蜜罐.
      • django-admin-honeypot - 假Django管理员登录屏幕通知管理员未经授权的访问尝试.
      • honeyhttpd - 基于Python的Web服务器honeypot构建器.
      • phpmyadmin_honeypot - 简单有效的phpMyAdmin honeypot.
      • shockpot - 用于检测Shell Shock漏洞尝试的WebApp Honeypot.
      • smart-honeypot - PHP脚本演示智能蜜罐.
      • Snare / Tanner - Glastopf的继任者
        • Snare - 超级下一代高级活性蜜罐.
        • Tanner - 评估SNARE事件.
      • stack-honeypot - 为垃圾邮件机器人插入陷阱进入响应.
      • tomcat-manager-honeypot - 模仿Tomcat管理器端点的蜜罐. 记录请求并保存攻击者的WAR文件以供以后学习
      • WordPress蜜罐
        • HonnyPotter - WordPress登录蜜罐，用于收集和分析失败的登录尝试.
        • HoneyPress - Docker容器中基于Python的WordPress蜜罐.
        • wp-smart-honeypot - 使用更智能的蜜罐减少垃圾评论的WordPress插件.
        • wordpot - WordPress蜜罐.

    • 服务蜜罐

      • AMTHoneypot - 针对英特尔AMT固件漏洞CVE-2017-5689的蜜罐.
      • Ensnare - 易于部署Ruby honeypot.
      • HoneyPy - 低交互蜜罐.
      • Honeygrove - 基于Twisted的多功能模块化蜜罐.
      • Honeyport - 用Bash和Python编写的简单honeyport.
      • Honeyprint - 打印机蜜罐.
      • Lyrebird - 现代高交互蜜罐框架.
      • MICROS honeypot - 低交互蜜罐，用于检测Oracle Hospitality Applications（MICROS）的Oracle Hospitality Simphony组件中的CVE-2018-2636.
      • RDPy - 用Python实现的Microsoft远程桌面协议（RDP）蜜罐.
      • SMB Honeypot - 高交互SMB服务蜜罐，能够捕获类似于类似恶意软件的恶意软件.
      • Tom's Honeypot - 低交互Python蜜罐.
      • WebLogic honeypot - 低交互蜜罐，用于检测Oracle融合中间件的Oracle WebLogic Server组件中的CVE-2017-10271.
      • WhiteFace Honeypot - WhiteFace的基于Twisted的蜜罐.
      • honeycomb_plugins - Plugin repository for Honeycomb, the honeypot framework by Cymmetria.
      • honeyntp - NTP记录器/蜜罐.
      • honeypot-camera - 观察相机蜜罐.
      • honeypot-ftp - FTP蜜罐.
      • honeytrap - 用Go编写的高级Honeypot框架，可与其他蜜罐软件连接.
      • pyrdp - RDP man-in-the-middle和Python 3库，能够实时或事后观看连接.
      • troje - 在单独的LXC容器内运行与服务的每个连接的蜜罐.

    • 分布式蜜罐

      • DemonHunter - 低交互蜜罐服务器.

    • 反蜜罐的东西

      • kippo_detect - 攻击组件，检测kippo蜜罐的存在.

    • ICS / SCADA蜜罐

      • Conpot - ICS / SCADA蜜罐.
      • GasPot - Veeder Root Gaurdian AST，在石油和天然气行业中很常见.
      • SCADA honeynet - 为工业网络构建蜜罐.
      • gridpot - 用于逼真表现电网蜜网的开源工具.
      • scada-honeynet - 模仿来自流行PLC的许多服务，更好地帮助SCADA研究人员了解暴露控制系统设备的潜在风险.

    • 其他/随机

      • Damn Simple Honeypot (DSHP) - 具有可插拔处理程序的蜜罐框架.
      • NOVA - 使用蜜罐作为探测器，看起来像一个完整的系统.
      • OpenFlow Honeypot (OFPot) - 将未使用的IP的流量重定向到基于POX构建的蜜罐.
      • OpenCanary - 模块化和分散式蜜罐守护程序，运行多个服务版本的服务，在服务（ab）使用时发出警报.

    • 僵尸网络C2工具

      • Hale - 僵尸网络命令和控制监视器.
      • dnsMole - 分析DNS流量并潜在地检测僵尸网络命令和控制服务器活动以及受感染的主机.

    • IPv6攻击检测工具

      • ipv6-attack-detector - Google Summer of Code 2012项目，由Honeynet项目组织提供支持.

    • 动态代码检测工具包

      • Frida - 注入JavaScript以探索Windows，Mac，Linux，iOS和Android上的本机应用程序.

    • 将网站转换为服务器蜜罐的工具

      • HIHAT - 将任意PHP应用程序转换为基于Web的高交互Honeypots.

    • 恶意软件收集器

      • Kippo-Malware - Python脚本，将下载存储为Kippo SSH蜜罐数据库中的URL的所有恶意文件.

    • 分布式传感器部署

      • Active Defense Harbinger Distribution (ADHD) - 基于Ubuntu LTS的GNU / Linux发行版，附带许多旨在预先安装和配置主动防御的工具.
      • Modern Honey Network - 多snort和蜜罐传感器管理，使用VM网络，小型SNORT安装，隐身dionaeas和集中式服务器进行管理.

    • 网络分析工具

      • Tracexploit - 重播网络数据包.

    • 记录匿名者

      • LogAnon - 记录匿名库，帮助匿名日志在日志和网络捕获之间保持一致.

    • 低交互蜜罐（路由器后门）

      • Honeypot-32764 - 路由器后门的蜜罐（TCP 32764）.
      • WAPot - 可用于观察指向家用路由器的流量的蜜罐.

    • 蜜网农场交通重定向器

      • Honeymole - 部署多个传感器，将流量重定向到集中的蜜罐集合.

    • HTTPS代理

      • mitmproxy - 允许拦截，检查，修改和重放流量流.

    • 系统仪表

      • Sysdig - 开源，系统级探索允许从正在运行的GNU / Linux实例捕获系统状态和活动，然后保存，过滤和分析结果.
      • Fibratus - 用于探索和跟踪Windows内核的工具.

    • 用于USB传播恶意软件的蜜罐

      • Ghost-usb - 通过USB存储设备传播的恶意软件的蜜罐.
      • Honeystick - USB棒上的低交互蜜罐.

    • 数据采集

      • Kippo2MySQL - 从Kippo基于文本的日志文件中提取一些非常基本的统计信息，并将它们插入MySQL数据库中.
      • Kippo2ElasticSearch - 用于将数据从Kippo SSH蜜罐MySQL数据库传输到ElasticSearch实例（服务器或集群）的Python脚本.

    • 被动网络审计框架解析器

      • Passive Network Audit Framework (pnaf) - 结合多种被动和自动分析技术的框架，以提供对网络平台的安全评估.

    • VM监控和工具

      • Antivmdetect - 用于创建与VirtualBox一起使用的模板的脚本，以使VM检测更加困难.
      • VMCloak - 为Cuckoo Sandbox自动生成虚拟机和隐藏真实内容.
      • vmitools - 带有Python绑定的C库，可以轻松监视正在运行的虚拟机的底层细节.

    • 二进制调试器

      • Hexgolems - Pint Debugger Backend - 用于PIN的调试器后端和LUA包装器.
      • Hexgolems - Schem Debugger Frontend - 调试器前端.

    • 移动分析工具

      • Androguard - Android应用程序的逆向工程，恶意软件和良好软件分析等.
      • APKinspector - 强大的GUI工具，供分析师分析Android应用程序.

    • 低交互蜜罐

      • Honeyperl - 基于Perl的蜜罐软件，具有为许多功能开发的插件：wingates，telnet，squid，smtp等.
      • T-Pot - 来自电信提供商T-Mobile的一体化蜜罐设备

    • Honeynet数据融合

      • HFlow2 - 用于蜜网/网络分析的数据合并工具.

    • 服务器

      • Amun - 漏洞仿真蜜罐.
      • Artillery - 开源蓝色团队工具，旨在通过多种方法保护Linux和Windows操作系统.
      • Bait and Switch - 将所有恶意流量重定向到蜜罐，该蜜罐部分镜像您的生产系统.
      • Bifrozt - 使用ansible自动部署bifrozt.
      • Conpot - 低交互式服务器端工业控制系统蜜罐.
      • Heralding - 凭据捕获蜜罐.
      • HoneyWRT - 低交互Python蜜罐，旨在模仿可能被攻击者攻击的服务或端口.
      • Honeyd - 见 honeyd tools.
      • Honeysink - 开源网络sinkhole，提供检测和防止给定网络上的恶意流量的机制.
      • Hontel - Telnet Honeypot.
      • KFSensor - 基于Windows的蜜罐入侵检测系统（IDS）.
      • LaBrea - 接管未使用的IP地址，并创建对蠕虫，黑客和其他互联网居民具有吸引力的虚拟服务器.
      • MTPot - 开源Telnet Honeypot，专注于Mirai恶意软件.
      • SIREN - 半智能HoneyPot网络 - HoneyNet智能虚拟环境.
      • TelnetHoney - 简单的telnet蜜罐.
      • UDPot Honeypot - 简单的UDP / DNS蜜罐脚本.
      • Yet Another Fake Honeypot (YAFH) - 用Go编写的简单蜜罐.
      • arctic-swallow - 低交互蜜罐.
      • glutton - 所有吃蜜罐.
      • go-HoneyPot - 用Go编写的蜜罐服务器.
      • go-emulators - 蜜罐Golang模拟器.
      • honeymail - 用Golang编写的SMTP蜜罐.
      • honeytrap - 低交互蜜罐和网络安全工具，用于捕获针对TCP和UDP服务的攻击.
      • imap-honey - 用Golang编写的IMAP蜜罐.
      • mwcollectd - 多功能恶意软件收集守护程序，结合了猪笼草和蜜蜂的最佳功能.
      • potd - 高度可扩展的低到中交互SSH / TCP蜜罐，专为OpenWrt / IoT设备而设计，利用多种Linux内核功能，如命名空间，seccomp和线程功能.
      • portlurker - Rust中的端口侦听器，具有协议猜测和安全字符串显示.
      • slipm-honeypot - 简单的低交互端口监控蜜罐.
      • telnet-iot-honeypot - 用于捕获僵尸网络二进制文件的Python telnet蜜罐.
      • telnetlogger - 用于跟踪Mirai僵尸网络的Telnet蜜罐.
      • vnclowpot - 低交互VNC蜜罐.
