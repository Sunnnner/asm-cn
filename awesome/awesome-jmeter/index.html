    JMeter

    Awesome JMeter¶

    >策划的资源集合 Apache JMeter 和相关的资源和闪亮的东西：插件，集成，测试技术，devops实践等.

    The Apache JMeter 是开源的，纯Java应用程序，旨在加载测试功能行为和测量性能.

    这个名单从此长大 this answer on Stack Exchange 和个人JMeter相关的链接收藏，得到了进一步的灵感 awesome 项目并通过这些改进 amazing contributors.
