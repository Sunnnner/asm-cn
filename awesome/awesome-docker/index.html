    What is Docker¶

    > Docker是开发人员和系统管理员构建，发布和运行分布式应用程序的开放平台. Docker Engine是一种便携式轻量级运行时和打包工具，Docker Hub是一种用于共享应用程序和自动化工作流程的云服务，它可以通过组件快速组装应用程序，消除开发，QA和生产环境之间的摩擦. 因此，IT可以更快地发货并在笔记本电脑，数据中心虚拟机和任何云上运行相同的应用程序，不变.

    Source: What is Docker
