    令人敬畏的W .. ¶

    > Wagtail社区精选软件包，文章和其他很酷资源的精选列表.

    Wagtail 是由Django提供支持的Python CMS，专注于灵活性和用户体验.

    你可能还喜欢 Awesome Django 和 Awesome Python . ：蛇：

    General resources¶

    • Official site
    • GitHub repository
    • Twitter account
    • Roadmap
    • PyPI classifiers for Wagtail

    Apps¶

    Blogging/news¶

    • Puput - Puput是一个功能强大且简单的Django应用程序来管理博客. 它使用了令人敬畏的Wagtail CMS作为内容管理系统.
    • wagtail_blog - Wagtail实现的类似WordPress的博客应用程序.
    • wagtailnews - Wagtail的插件，提供新闻/博客功能.
    • wagtail-blog-app - Wagtail Django CMS的博客应用程序.
    • Django Wagtail Feeds - 为您的Wagtail CMS项目添加对RSS源，Facebook Instant Articles和Apple News Publisher的支持.
    • CodeRed CMS - 专业支持的W要么dPress替代品，用于构建营销网站. 在开箱即用的wagtail管理员中创建页面，博客，表单和每个Bootstrap 4组件！ Learn m要么e 要么 watch the lightning talk.
    • Snotra_RSS - Snotra_RSS是Wagtail的Atom和RSS新闻聚合器应用程序.

    Rich text editor extensions¶

    • wagtail-readability - 测试你输入Wagtail的内容的可读性.
    • wagtailembedder - 用于Wagtail richtext字段的Snippets嵌入器.
    • Wagtail TinyMCE - Winytail的TinyMCE编辑器集成.
    • Wagtail Froala - 扩展Wagtail以在RichTextField / RichTextBlock中使用Froala WYSIWYG编辑器.
    • Wagtail Medium Editor - Wagtail的可自定义媒体编辑器，支持链接锚点.
    • WagtailDraftail - 基于Wagtail的Draft.js编辑器 Draftail 和 draftjs_exporter.
    • wagtail-readinglevel - Wagtail插件，用于确定输入到富文本字段的文本的阅读级别.

    Widgets¶

    • wagtailgmaps - Wagtail字段的简单Google地址格式化程序.
    • Wagtail-Geo-Widget - Wagtail中GeoDjango PointField字段的Google Maps小部件.
    • wagtail-markdown - Markdown字段和Wagtail块.
    • wagtail-autocomplete - 为“ForeignKey”，“ParentalKey”和“ManyToMany”字段自动填充选择器.

    StreamField¶

    • Wagtail FontAwesome - 将FontAwesome图标添加到StreamField.
    • Wagtail Commonblocks - 用于Wagtail的常见StreamField块.
    • Wagtail SVGmap - Wagtail通过内联SVG的ImageMap功能.
    • Wagtail ClearStream - 使Wagtail的StreamField更加模块化的应用程序.
    • UWKM Streamfields - 基本的Wagtail StreamField模块可以获得乐趣和利润.
    • wagtail-inventory - 通过它们包含的StreamField块搜索Wagtail页面.
    • Wagtail Code Block - 具有实时PrismJS语法突出显示功能的Wagtail CMS的StreamField代码块.
    • Wagtail Blocks - 一系列令人敬畏的Wagtail CMS流场块和图表.

    Static site generation¶

    • Wagtail-bakery - 一组用于将Django Wagtail网站烘焙为平面文件的帮助程序.
    • Wagtail-Netlify - Easilly将您静态呈现的Wagtail网站发布到Netlify.

    Settings management¶

    • Wagtail-Constance - Wagtail CMS的django-constance集成.
    • Wagtail-Flags - Wagtail网站的功能标志.

    E-commerce¶

    • wagtailinvoices - 用于创建发票的Wagtail模块.
    • longclaw - Wagtail CMS的商店模板.
    • django-oscar-wagtail - 为Oscar Commerce（或Wagtail的Oscar Commerce整合？）提供Wagtail集成.

    SEO and SMO¶

    • wagtail-metadata - 帮助社交媒体和搜索引擎的元数据的工具.
    • wagtail-metadata-mixin - Wagtail CMS页面的OpenGraph，Twitter Card和Google+代码段标签.
    • wagtail-schema.org - Wagtail网站的Schema.org JSON-LD标签.

    Analytics¶

    • Wagtail Analytics - Wagtail管理员中的Google Analytics信息中心.

    Customer experience¶

    • Wagtail Experiments - Wagtail的A / B测试.
    • Wagtail Personalisation - 个性化模块，使编辑人员能够根据直接在管理界面中配置规则的段创建自定义页面或页面的一部分.

    Security¶

    • wagtailenforcer - 如果您需要在Wagtail网站上强制执行安全协议，那么您来对地方了.
    • wagtail-yubikey - 在Wagtail管理面板上启用YubiKey双因素身份验证.
    • wagtail-2fa - 通过将其与django-otp集成，为Wagtail添加双因素身份验证.

    Media¶

    • wagtailmedia - A Wagtail module for managing video and audio files within the admin.
    • wagtail-embedvideos - 简单的应用程序，类似于wagtailimages，但用于嵌入来自SoundCloud的YouTube和Vimeo视频和音乐. 这是一个整合 django-embed-video.
    • Wagtail Alt Generator - 用于基于计算机视觉生成图像描述和标签的模块.
    • Wagtail FilePreviews - 使用FilePreviews.io中的图像预览和元数据扩展Wagtail的文档.
    • Wagtail-Textract - 制作Wagtail搜索文档内容（PDF，Excel和Word等）.

    Translations¶

    • Wagtail Modeltranslation - 包含集成的mixin模型的简单应用程序 django-modeltranslation 进入Wagtail面板系统.
    • wagtailtrans - 用于支持多语言网站的Wagtail插件.

    Forms¶

    • wagtailpolls - 用于向Wagtail CMS添加轮询功能的插件.
    • Wagtailsurveys - A module for Wagtail which provides the ability to build polls and surveys.
    • Wagtail ReCaptcha - wagtail-django-captcha提供了一种简单的方法来集成 django-recaptcha 使用Wagtail formbuilder时的字段.
    • wagtailstreamforms - 在Wagtail的管理员中构建表单，以便在流场中使用.

    Testing¶

    • wagtail-linkchecker - 帮助您在Wagtail网站上查找损坏链接的工具.
    • Wagtail Accessibility - 在Wagtail开发时帮助实现可访问性的插件.
    • Wagtail Factories - Wagtail的工厂男孩课程.
    • Wagtail Foliage - 以Wagtail以编程方式构建页面树的实用程序.

    Misc¶

    • Wagtail Plus - Wagtail CMS的模块化附加装置.
    • wagtailmenus - Torchbox Wagtail CMS的扩展，可帮助您以一致，灵活的方式管理和呈现多级导航和简单的平面菜单.
    • Wagtail Error Pages - Wagtail漂亮，智能，可定制的错误页面.
    • Wagtail Themes - Wagtail的特定于站点的主题加载器.
    • Wagtail Sharing - 更容易分享Wagtail草稿.
    • Wagtail Gridder - 与Google图像搜索结果类似的网格卡布局，扩展了卡片详细信息区域.
    • Wagtail Condensed Inline Panel - 适用于大量内联的Wagtail InlinePanel的直接替换（可与拖放支撑折叠）.
    • Joyous - Wagtail的日历应用程序.
    • Wagtail App Pages - 使用实际的URL配置和django视图扩展Wagtail页面.
    • Wagtail Import Export - 在Wagtail实例之间导入/导出页面.
    • Wagtail Tag Manager - Wagtail插件，允许更容易和GDPR兼容的脚本和标签管理.
    • Wagtail Cache - 使用Django缓存中间件的Wagtail的简单页面缓存.
    • Wagtail GraphQL - 应用程序自动添加GraphQL支持到Wagtail网站.
    • Wagtail Orderable - Mixin支持管理面板中的拖放排序.

    Tools¶

    • Wagtail Cookiecutter Foundation - 使用Zurb Foundation 6的Wagtail CMS的Cookiecutter模板.
    • Beginner Wagtail Cookiecutter - Wagtail CMS的超级简单实现.
    • Wagtail Starter Kit - 一个cookieingter完成wagtail，django布局，vagrant，供应scrips，前端构建系统等等！
    • Wagtail Boilerplate - 一个Django / Wagtail cookiecutter，内置支持docker，部署（带有ansible），Reactjs，pytest和S3.
