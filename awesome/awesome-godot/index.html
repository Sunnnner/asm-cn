    Godot

    Awesome Godot ¶

    精选的清单 free/libre 游戏，插件，附加组件和脚本 Godot.

    Games¶

    使用戈多的游戏.

    2D¶

    • Blipshift - 简单的自上而下移动平台游戏（Godot 3.1）.
    • BlockPop - 一个简单的突破克隆.
    • Bombs - 简单的手机游戏. 你需要触摸炸弹来化解它们.
    • Breakable - 类似突破的游戏. 与Godot 3.0兼容.
    • Drone Commander - 命令成群的无人机对抗外星人. 专为CGA JAM而制造.
    • Captain Holetooth - 详细世界中的探索性平台游戏. 面向年轻球员.
    • Dolphin Island 2 - 平台游戏为A Game By It Cover 2015 Game Jam制作.
    • DynaDungeons - 炸弹人克隆在幻想世界.
    • Kraken-Rampage - Ludum Dare 33小时入场.
    • Libre Pinball - 弹球游戏.
    • Memory Game - 简单记忆游戏.
    • Minehunter - 扫雷克隆.
    • Minilens - 免费益智平台游戏.
    • Mountain of Hope - 剧场游戏果酱游戏（PGA 2015）
    • Ninja-Training - 小型跑步游戏，1小时挑战.
    • No hope - Ludum Dare 33小时入场.
    • Planet Rider - 在外星环境中的2D物理跑酷游戏.
    • Platshoot - 2D射击游戏.
    • Ringed - 自上而下的射击游戏，具有广泛的灯光效果.
    • Tanks of Freedom - 免费回合制等距战略游戏.
    • Three Hungry Mice - 为2016年3月的Godot Engine Jam制作的老鼠突围游戏.

    3D¶

    • Cheese Hunter - 2016年3月Godot Engine Jam游戏.
    • Meteorite - 由Metroid Prime启发的低端都市电视FPS，为LOWREZJAM 2018制作.
    • MouseBoat - 作为鼠标玩耍并在池塘中航行. 扔你的杆，并尝试捕捉一些鱼.
    • Super Tux Party - 免费/自由和开源派对游戏，旨在复制马里奥派对等游戏的感觉.
    • Veraball - 免费“滚球”游戏. 收集硬币并到达终点，不会掉出水平.

    Demos¶

    演示学习GDScript，其概念和各种游戏功能.

    • Cave generator - 利用细胞自动机方法的简单洞穴发生器.
    • FPS Test - 单人FPS测试. 物理工作，但没有敌人，没有适当的射击.
    • HTTP Server - HTTP 1.0文件服务器.
    • Open Source A-RPG Demo - RPG演示（Godot 3.0）.
    • OpenRPG - JRPG风格的游戏演示和模板（Godot 3.1）.
    • Pathfinding 2d - 使用tilemap演示寻路的演示项目.
    • UDP Snapshot interpolation - 允许您连接到服务器并查看联网物理.
    • UDP State synchronization - 比快照插值更有效的网络演示.
    • Voxel Game - 类似Minecraft的体素地形演示.
    • voxelgame - 使用godot_voxel模块的类似Minecraft的体素地形演示.

    Modules¶

    • godot-enet-better - 使用Godot进行高性能多人游戏的更好的ENet模块.
    • godot-voxel - 用于在Godot Engine中创建体积世界的模块.
    • Gdnet - 戈多的ENet包装.

    • RawPacker - RawArray的简单二进制打包/解包.
    • spine - Spine 动画支持模块.
    • WaterSplash 2D node

    Plugins and scripts¶

    脚本，让你做新的东西，或增强戈多功能.

    • Atlas importer - 从中​​导入打包的地图集的插件 TexturePacker, Attila 和 Kenney Assets.
    • Auto Tile Layer - 允许使用RPG Maker的Autotile格式自动进行2D平铺.
    • Camera2D Screen Shake - Camera2D的屏幕抖动效果.
    • CRT Shader - 模拟CRT显示的Godot着色器.
    • Easing Script - 罗伯特·彭纳（Robert Penner）缓和方程式的一个端口.
    • Escoria - 点击冒险游戏框架.
    • FontAwesome - 戈多的Font Awesome插件.
    • gdstats - 用于常见统计分布的伪随机数生成器库.
    • gdutils - 用GDScript编写的Godot实用程序.
    • Godot NExt - Godot的一组基本节点扩展.
    • GodotNotificationCenter - 戈多的通知中心.
    • GodotTIE - 用于控制文本输出的文本界面引擎（如在RPG对话框中）.
    • Grass Plugin - 处理大量草，叶和其他植被的插件.
    • gterm - GUI控件，绘制类似于 * nix终端仿真器的东西.
    • GUT - 在GDScript中编写单元测试的实用程序.
    • Polygon Merge - 合并多边形.
    • Simplex 2D/3D - 单纯形确定性噪声函数.
    • Scrolling Background - A scrolling background node for Godot.
    • SMRT-Godot - 一个可以定制的对话系统和编辑器，非常适合故事驱动的游戏.
    • SUTjoystick - 简单的游戏手柄支持GNU / Linux和Windows.
    • Tiled importer - 从中​​导入地图 Tiled.
    • Tiled importer - 从Tiled for Godot 2.1导入tilemaps的插件.
    • Tileset Builder - 构建tilesets.
    • TileSet Builder - 快速构建风格的瓷砖组.

    Other¶

    其他的东西.

    • gd-com npm package - 使用Node.js与Godot客户端通信.
    • godot-ci - Docker图像通过CI导出Godot游戏. 包括GitLab CI脚本示例.
    • godot.sh - 自动下载并启动Godot最新版本的脚本，或者编译并启动Gitmaster分支.
    • godot-wrapper - 帮助Debian和Ubuntu用户安装和使用Godot的脚本.
    • joystick-mapping - 为所有支持的平台生成游戏手柄映射的工具.
    • replicate-file.sh - 使用一个主文件更新文件副本的脚本.
    • strip-to-frames.pl - Perl脚本将网格spritesheet图像拆分为编号的单个帧文件.
