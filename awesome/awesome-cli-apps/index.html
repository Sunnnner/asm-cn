    寻找新的维护者

    Entertainment¶

    • football-cli - 获得终端中几乎所有足球比赛/联赛的现场比分，赛程，积分榜.
    • pockyt - 阅读，管理和自动化您 Pocket 采集.
    • newsboat - 用于文本终端的可扩展RSS提要阅读器.

    Music¶

    • cmus - 适用于类Unix操作系统的小型，快速且功能强大的控制台音乐播放器.
    • Instant-Music-Downloader - 即时音乐下载器.
    • itunes-remote - 这是关于在不离开终端的情况下听音乐. 仅限macOS.
    • pianobar - 潘多拉客户端.
    • somafm-cli - 在您的终端收听SomaFM.
    • nehm - 控制台工具，下载，设置IDv3标签并添加到您的iTunes（如果您使用macOS）您的SoundCloud喜欢方便的方式.
    • mpd - 音乐播放器守护进程.
    • ncmpcpp - 由ncmpc启发的基于ncurses的音乐播放器守护进程客户端.
    • moc - 适用于Linux / UNIX的控制台音频播放器.
    • musikcube - 跨平台，基于终端的音乐播放器，音频引擎，元数据索引器和服务器.

    Social Media¶

    • facebook-cli - Facebook命令行界面.
    • TTYtter - Twitter客户端 - （original).
    • Rainbowstream - 用Python编写的终端上一个聪明而漂亮的Twitter客户端.
    • RTV - RTV提供了一个界面，用于查看终端上的reddit并与之交互.
    • WeeChat - WeeChat是一个快速，轻便且可扩展的聊天客户端. 它运行在许多平台上，如Linux，Unix，BSD，GNU Hurd，Mac OS X和Windows（Bash / Ubuntu和Cygwin）.

    Video¶

    • youtube-dl - 一个小型命令行程序，用于从YouTube.com和更多网站下载视频. 您可以使用brew install youtube-dl进行安装.
    • open-pip-cli - 从终端观看macOS原生画中画播放器中的电影.
    • streamlink - 一种命令行实用程序，可将来自各种服务的视频流传输到视频播放器（如VLC）中. Streamlink的主要目的是允许用户避免错误和CPU重闪存插件，但仍然能够享受各种流内容.
    • mps-youtube - 命令行Youtube播放器.

    Movies¶

    • moviemon - 命令行中有关您电影的所有内容.
    • movie - 获取电影信息或比较终端中的电影.

    Games¶

    • Dwarf Fortress - 矮人要塞是部分建筑和管理模拟，部分roguelike，独立视频游戏.

    Development¶

    • bcal - 用于存储转换和计算的字节计算器.
    • boilr - 无依赖关系的CLI，用于从样板模板创建项目.
    • cgasm - 提供x86组装文档的工具. 它发音为“SeekAzzem”.
    • cookiecutter - 从cookiecutters（项目模板）创建项目.
    • dockly - 用于管理Docker容器的交互式CLI.
    • legit - 从许多开源许可证中进行选择以生成LICENSE文件或向文件添加许可证头.
    • mklicense - 使用自定义信息轻松创建自定义LICENSE文件. 忙碌的人和初学者友好.
    • rebound - 当您遇到编译器错误时，立即在终端中获取Stack Overflow结果.
    • yo - 用于运行Yeoman生成器的CLI脚手架工具. 有成千上万的即用型发电机，创建一个自己的发电机非常容易.
    • grip - 在提交之前使用GitHub样式预览markdown文件.

    Text Editors¶

    • emacs - 可扩展，可定制，免费/自由的文本编辑器 - 等等.
    • vim - Vim - 无处不在的文本编辑器.
    • neovim - Vim fork专注于可扩展性和可用性.

    Web Development¶

    • bower-name-cli - 检查凉亭上是否有包裹名称.
    • caniuse-cmd - 搜索 https://github.com/agarrharr/awesome-cli-apps/blob/master/caniuse.com 关于功能的浏览器支持.
    • is-up-cli - 检查网站是上升还是下降.
    • pageres-cli - 捕获网站截图.
    • strip-css-comments-cli - 从CSS中删除评论.
    • viewport-list-cli - 返回设备列表及其视口.
    • surge - 将目录部署到Web.
    • acmetool - 易于使用的命令行工具，用于自动从ACME服务器获取证书.
    • loadtest - 对选定的URL运行负载测试. 易于扩展至少为您自己的目的.
    • wp-cli - 无需使用Web浏览器即可管理WordPress安装，更新插件，设置多站点安装等等.
    • diff2html-cli - 命令行工具，将git diff解析为JSON并生成漂亮的HTML.
    • HTTPie - 用户友好的HTTP客户端.
    • serve - 易于使用的静态文件服务器，具有https，CORS和GZIP压缩等功能.
    • add-gitignore - 交互式CLI，可根据您的需要为项目生成.gitignore.
    • mevn-cli - MEVN堆栈的CLI工具.
    • scaffold-static - vanila JS的静态站点生成器.
    • Discharge - 将静态网站部署到Amazon S3的简单方便的方法.
    • reachable - Reachable是一个CLI工具，用于检查域是否已启动.
    • simplehttp - 从CLI开始通过HTTP提供本地目录的最简单直接的方法.
    • shell2http - 用于执行shell命令的HTTP服务器. 专为开发，原型设计或远程控制而设计.
    • HTTP Prompt - 具有自动完成和语法突出显示功能的交互式命令行HTTP客户端.

    Mobile Development¶

    • mobicon-cli - 移动应用图标生成器.
    • mobisplash-cli - 移动应用程序启动画面生成器
    • deviceframe - 将设备框架放在您的移动/网络/渐进式应用屏幕截图周围.

    Database¶

    • cycli - Neo4j的终端客户端，具有自动完成和语法突出显示功能.
    • mycli - 具有AutoCompletion和语法突出显示的MySQL终端客户端.
    • pgcli - Postgre的终端客户端，具有自动完成和语法突出显示功能.

    Devops¶

    • htconvert - 将.htaccess重定向转换为nginx.conf重定向.
    • aws-cli - Amazon Web Services的通用命令行界面.
    • azure-cli - Azure的命令行工具.
    • SAWS - 增压的AWS CLI.
    • s3cmd - 全功能S3客户端.
    • lstags - 跨不同的注册表同步Docker镜像.
    • pm2 - Node.js的生产过程管理器

    Release¶

    • release-it - 自动发布Git存储库和/或npm软件包. 更改日志生成，GitHub / GitLab版本等.
    • clog - 我们其他人的传统更改日志.
    • np - 更好的'npm发布'.
    • release - 使用单个命令生成更改日志.

    Npm¶

    • npm-name-cli - 检查npm上是否有可用的包名.
    • npm-user-cli - 获取npm用户的用户信息.
    • npm-home - 在当前目录中打开包的npm页面.
    • pkg-dir-cli - 找到npm包的根目录.
    • npm-check-updates - 查找比package.json允许的更新版本的软件包依赖项.

    Productivity¶

    • AutoScreenshotUploader - 即时捕获并上传截图 imgur.
    • bitly-client - 使用此命令行实用程序创建和管理Bitly短链接.
    • doing - 一个命令行工具，用于跟踪您正在做的事情并跟踪您已完成的工作.
    • ffscreencast - 具有视频覆盖和多显示器支持的ffmpeg截屏视频.
    • imgur-uploader-cli - 上传图片到imgur.
    • meetup-cli - Meetup.com命令行界面.
    • NeoMutt - 邮件客户端.
    • terjira - Jira的命令行电动工具.
    • ipt - 关键跟踪器的命令行界面.
    • g3l - Git很简单，github cli很容易，但是在w0rld中g3l最简单的git cli！
    • uber-cli - 优步，触手可及.
    • Buku - 强大的命令行书签管理器.
    • googler - 来自终端的Google搜索，Google Site Search，Google新闻.
    • calcurse - 计算，命令行的日历和计划应用程序.
    • papis - 功能强大且可扩展的文档和参考书目管理器.
    • mdp - 基于命令行的降价演示工具.
    • sent - 简单的纯文本演示工具.

    Time Tracking¶

    • Timetrap - 简单的命令行timetracker.
    • moro - 使用单个命令的时间跟踪器.

    Note Taking and Lists¶

    • idea - 轻量级的CLI工具和模块，可以快速，轻松地将想法保存在安全的地方.
    • geeknote - Evernote的控制台客户端.
    • Taskwarrior - 免费和开源软件，从您的命令行管理您的TODO列表.
    • Terminal velocity - UNIX终端的快速笔记应用程序.
    • eureka - 无需离开终端即可存储您的想法.
    • sncli - 一个简单的Python应用程序，可让您通过命令行访问Simplenote帐户.
    • td-cli - 一个todo命令行管理器，用于组织和管理多个项目中的待办事项.

    Finance¶

    • ledger - 强大的复式会计系统，可从UNIX命令行访问.
    • moeda - 使用CLI进行外汇汇率和货币兑换.
    • cash-cli - 从命令行转换32种货币！
    • cointop - 用于跟踪加密货币的最快且最具交互性的终端UI应用程序.

    Utilities¶

    • archive-type-cli - 检测Buffer / Uint8Array的归档类型.
    • battery-level - 获取当前的电池电量.
    • brightness-cli - 改变屏幕亮度 - windows version.
    • screensaver - 启动屏幕保护程序.
    • yank - Yank终端输出到剪贴板.
    • text-meme - 生成文本模因.
