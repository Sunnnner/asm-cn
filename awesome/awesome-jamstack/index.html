    JAMstack









    JAMstack ：名词'jam-stak'\
    基于客户端JavaScript，可重用API和预构建标记的现代Web开发体系结构.



    这是一份精心策划的杰出JAMstack资源列表.



    您可能也喜欢Awesome Static Generators .

    想要贡献一些资源？ 大！ 请在提交之前阅读贡献指南 .



    General¶

    • JAMstack
    • JAMstack resources - 关于JAMstack的视频和文章.
    • the New Dynamic - 在我们的目录中选择工具以使用JAMstack. 浏览我们的展示并获得灵感.

    Chats¶

    • Appernetic Gitter Chat
    • JAMstack Gitter Chat
    • Netlify Gitter Chat
    • NetlifyCMS Gitter Chat
    • the New Dynamic Slack

    Platforms¶

    • Appernetic - 使用Hugo和GitHub Pages对静态网站进行可视内容管理.
    • Netlify - 用于自动化现代Web项目的一体化平台.

    JAMstack Sites Showcase¶

    • React - 以盖茨比为基础.
    • Squoosh.app - 在Netlify上托管，演示了现代Web应用程序的高级功能.
    • Hopper - 建立在Gatsby上并在Netlify上托管.

    Static Site Generators¶

    • Gatsby - React的快速静态站点生成器.
    • Next.js - 用于静态和服务器呈现的应用程序的轻量级框架.
    • Hugo - Hugo是一个用Go编写的静态站点生成器.
    • Jekyll - Jekyll是一个简单的，支持博客的静态网站生成器，非常适合个人，项目或组织网站.
    • Hexo - 一个快速，简单和强大的博客框架，由Node.js提供支持.
    • Metalsmith - 一个非常简单，可插拔的静态站点生成器.
    • Phenomic - 模块化网站编译器.
    • Gridsome - Vue支持的静态站点生成器，用于为任何无头CMS，API或Markdown文件构建CDN就绪网站.
    • Nuxt.js - 用于服务器端呈现，静态和传统单页面应用程序的多功能基于vue的框架.

    有关更完整的列表，请参阅 StaticGen.

    CMS¶

    • Contentful - 数字团队的内容基础架构.
    • NetlifyCMS - 着名的数字艺术博客.
    • ButterCMS - 无头CMS和内容API.
    • Scrivito - 为数字代理商和中型到大型企业构建的基于云的JavaScript CMS.
    • GraphCMS - GraphQL无头CMS.
    • Prismic - 适用于开发人员和营销人员的无头API CMS.
    • Siteleaf - 适用于您的静态站点的友好CMS.
    • DatoCMS - 您的编辑喜欢的基于API的CMS.
    • Prose - Prose是GitHub的内容编辑器，专为管理网站而设计.
    • Tipe - 下一代API-first CMS.
    • Forestry - 使用Hugo和Jekyll构建的静态站点的无头CMS.
    • Coisas - 用于编辑GitHub Markdown（和其他）文件的客户端CMS.
    • Cockpit - 自我托管的无头和api驱动的CMS.
    • HeadlessCMS - JAMstack站点的内容管理系统列表.
    • TakeShape - 无头GraphQL API CMS，只需单击一次即可部署到Netlify.

    API¶

    Authentication¶

    • Auth0 - 单点登录和基于令牌的身份验证.
    • Netlify Identity - 提供全套身份验证功能，由 GoTrue API.

    Comments¶

    • Disqus - 全球评论系统，可改善网站讨论并连接网络上的对话.
    • Facebook Comments - 评论插件可让人们使用他们的Facebook帐户评论您网站上的内容.

    Forms¶

    • Netlify Forms - 通过在部署时直接解析HTML文件，对构建时间进行内置表单处理.

    E-commerce¶

    • Flatmarket - Flatmarket是一个免费的开源电子商务平台，适用于静态网站.
    • GoCommerce - JAMstack网站的无头电子商务.
    • Snipcart - 面向开发人员的强大购物车平台.
    • Moltin - 面向开发人员的电子商务API.
    • Trolley - 为JAMstack设计的购物车.

    Search¶

    • Algolia - 构建搜索业务的最可靠平台.
    • Lunr - 搜索变得简单（在前端）.
    • CloudSh - 使用几行JavaScript进行强大的搜索.

    Database¶

    • GraphQL - API的查询语言和用于使用现有数据完成这些查询的运行时.
    • Graphcool - Open-source and self-hosted backend-as-a-service to develop serverless GraphQL backends.
    • FaunaDB - 关系NoSQL数据库，在云或内部部署中具有联接，索引和多区域ACID事务.

    Automation¶

    • Zapier - 触发将超过1000个应用程序连接在一起的操作.
    • DataFire - DataFire是一个用于构建和集成API的开源框架.

    Serverless¶

    • Netlify Functions - Netlify允许您在没有AWS账户的情况下部署Lambda功能，并且可以直接在Netlify中处理功能管理.
    • Amazon Lambda - 允许您在不配置或管理服务器的情况下运行代码.
    • Microsoft Azure
    • Azure Functions - 无服务器计算服务，使您能够按需运行代码，而无需显式配置或管理基础架构.
    • Azure Logic Apps - 简化构建自动化可扩展工作流，以跨云服务和本地系统集成应用程序和数据.
    • Google Cloud
    • App Engine - 无服务器应用程序，完全抽象出基础架构，因此您只关注代码.
    • Cloud Functions - 无服务器环境，用于构建和连接云服务.
    • Cloud Datastore - 具有自动分片和复制功能的高度可扩展的NoSQL数据库.
    • Cloud Storage - 满足高QPS需求的地理冗余对象存储.
    • Cloud Pub/Sub - 针对所有消息大小和速度的地理冗余实时消息传递.
    • Apigee - 针对多云环境的企业API管理.
    • Endpoints - 基于Google Cloud构建的API管理应用程序.
    • Cloud Dataflow - 无服务器流和批量数据处理服务.
    • BigQuery - 无服务器数据仓库服务，可帮助您为企业部署高级云数据仓库解决方案.
    • Cloud ML Engine - 无服务器机器学习服务，可自动扩展自定义Google硬件（Tensor Processing Units）.
    • Serverless - 用于部署和运行无服务器架构的工具包.

    有关更完整的列表，请参阅 Awesome Serverless.
