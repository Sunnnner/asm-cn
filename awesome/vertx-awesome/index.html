    Awesome Vert.x ¶

    • Awesome Vert.x *是一个很棒的框架，库或其他组件的列表，供使用或使用 Vert.x 版本3.

    如果您希望组件显示在此处，请向此存储库发送拉取请求以添加它.

    请注意，除非有，否则我们无法保证此列表中所有内容的稳定性或生产性 图标 在它的旁边. 此图标表示该组件是官方的一部分 Vert.x stack.

    对于Vert.x版本2检查 this page.

    • Vert.x Jersey - 创建JAX-RS Jersey Vert.x中的资源.

    • Knot.x - 基于Vert.x 3构建的现代网站的高效和高性能集成平台.

    • Irked - Vert.x 3 Web和控制器框架的基于注释的配置.

    • Atmosphere Vert.x - JVM的实时客户端服务器框架，支持跨浏览器回退的WebSockets和服务器发送事件.
    • Vert.x Vaadin - 在Vert.x上运行应用程序.
    • Serverx - 允许您仅使用路径处理程序注释快速轻松地设置Vert.x驱动的服务器.

    Database Clients¶

    连接数据库的客户

    • NoSQL数据库
    • MongoDB - 用于与MongoDB数据库交互的异步客户端.
    • Redis - 与Redis交互的异步API.
    • Cassandra - Vert.x客户端，允许应用程序与Cassandra服务进行交互.
    • Cassandra - 与Cassandra和Cassandra Mapping交互的异步API.
    • OrientDB - 非阻塞OrientDB服务器集成.
    • Bitsy - 非阻塞Bitsy Graph服务器集成.
    • MarkLogic - Marklogic数据库服务器的异步客户端.

    • vertx-pojo-mapper - MySQL和MongoDB的非阻塞POJO映射.

    • vertx-mysql-binlog-client - 用于访问MySQL复制流的Vert.x客户端.
    • vertx-mongo-streams - 帮助使用MongoDB异步驱动程序将Vert.x流传输到MongoDB GridFS或从MongoDB GridFS传输.

    *服务器发送的事件 * jEaSSE - Java Easy SSE. 一个简单，轻量级的SSE实现. * vertx-sse - Vert.x SSE实现+事件总线SSE桥.

    • Vert.x REST Client - Vert.x的REST客户端，支持RxJava和请求缓存.

    • JCA adaptor - 用于Vert.x事件总线的Java Connector Architecture Adapter.
    • Weld - 将CDI编程模型引入Vert.x生态系统（将CDI观察者方法注册为Vert.x消息使用者，CDI驱动的Verticle，以声明方式定义路由等）.

    *流星 * Meteor - 通过Vert.x事件总线支持流星集成.

    *指标 * Hawkular metrics - Hawkular Vert.x Metrics SPI的实现. * DropWizard metrics - 使用DropWizard指标实现指标. * Micrometer metrics - 使用Micrometer指标实施指标. * OpenTsDb Metrics - OpenTsDb Vert.x的指标客户端. * Bosun Monitoring - Bosun Vert.x的客户端库.

    • Hystrix Metrics Stream - 使用Vert.x应用程序发送Hystrix仪表板的指标 Hystrix.
