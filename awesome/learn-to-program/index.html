    Learn to Program ¶

    Foundation in Web Development¶

    互联网上充斥着越来越多的课程，书籍和指南，供各种技能水平的程序员使用，以提高他们的技能. 不幸的是，这些资源要么难以找到，要么质量低劣.

    该清单旨在成为一系列高质量的教育资源. 该平台的成本与所涵盖的主要主题一起突出显示.

    Beginner¶

    Mozilla Webmaker¶

    免费社区和工具集，学习创建网页和应用程序\ （HTML，CSS，JavaScript？）

    Codecademy¶

    免费平台，用于学习编写Web技术的代码\ （HTML，CSS，JavaScript，jQuery，Python，Ruby，Rails，PHP）

    Khan Academy's Hour of Code¶

    免费互动1小时课程，学习网络开发的基础知识 （HTML，CSS，JavaScript，SQL）

    Treehouse¶

    用于如何构建网站和应用程序的商业平台 （网页设计，前端Web开发，Rails，iOS，Android，PHP）

    Learn CSS Layout¶

    关于如何使用CSS进行布局的免费教程 (CSS)

    Udemy Programming, Development¶

    免费增值来自第三方提供商的课程市场. 质量可能会有所不同. （HTML，CSS，JavaScript，Ruby，Rails，Python，iOS，Android）

    Code Avengers¶

    基础网络和应用程序开发课程的免费增值平台\ （HTML，CSS，JavaScript）

    learn.shayhowe.com¶

    免费初学者到网络开发的中间指南\ （HTML，CSS，JavaScript）

    HTML Dog¶

    网站开发免费初学者和中级指南\ （HTML，CSS，JavaScript）

    Gibbon¶

    用于计算机编程的混合策划学习轨道\ （网站开发，JavaScript，AngularJS，Ruby，Rails，iOS，Haskell，计算机科学）

    Degreed Web Development, Programming¶

    混合目录的课程，视频和其他学习资源，用于Web开发和编程. 质量可能会有所不同. （HTML，CSS，JavaScript，AngularJS，Rails）

    Platzi¶

    免费平台设计，营销，启动和代码课程. 了解网络的未来. （RethinkDB，SailsJs，NodeJS，Git，Startup Class等）

    Free Code Camp¶

    免费学习编码并帮助公益组织. 一个开源社区的人，他们学习编码和帮助非营利组织. (HTML, CSS, Javascript, Databases, Git & GitHub, Node.js, React.js, D3.js)

    Vertabelo Academy¶

    免费 SQL课程，互动练习和测验\ （SQL，数据库概念）

    GitHub Learning Lab¶

    免费自学，互动项目，学习Git和GitHub. 由GitHub的培训团队创建和维护. （Git，GitHub）

    Intermediate¶

    Khan Academy Computer Programming, Computer Science¶

    关于如何编程绘图，动画，游戏和网页以及更高级的计算机科学主题的免费中级到高级课程 （HTML，CSS，JavaScript，算法，加密）

    Udacity¶

    免费的计算机科学和网络开发课程平台\ （HTML，CSS，JavaScript，数据科学，Python，计算机科学专题）

    Learn Python the Hard Way¶

    免费 （仅限图书）适合初学者通过中级Python编程的书籍和课程 （Python，面向对象的编程，Web开发）

    Michael Hartl's Ruby on Rails Tutorial¶

    免费的在线书籍，涵盖了创建Ruby on Rails应用程序的所有阶段 （HTML，CSS，JavaScript，Ruby，Rails）

    Lynda.com Developer, Web, IT¶

    免费增加平台，用于网络开发和中级编程的视频课程 （HTML，CSS，JavaScript，编程基础，系统管理......）

    Thinkful¶

    来自行业专家的商业平台，用于指导网络和移动开发课程 （网站开发，前端Web开发，AngularJS，Android，iOS）

    exercism.io¶

    免费众包导师平台的编程练习和代码审查\ (Clojure, CoffeeScript, C++, C#, Elixir, Erlang, F#, Go, Haskell, JavaScript, Common Lisp, Lua, Objective-C, OCaml, Perl 5, PL/SQL, Python, Ruby, Scala, Swift)

    University of Reddit¶

    免费众包的视频和课程列表，用于学习编程和网站开发\ （HTML，CSS，JavaScript，Java，Haskell，Python）

    Stanford on iTunes U¶

    来自斯坦福大学的免费编程和移动开发课程 （编程，初创公司，iOS）

    PluralSight¶

    用于网站开发课程的商业平台\ （JavaScript，AngularJS，Java）

    CodeChef Problems¶

    免费中级到高级编程问题\ (programming)

    CodingBat¶

    Python和Java中的免费练习题 （Python，Java）

    Codewars¶

    免费代码挑战. 将您的解决方案与其他解决方案进行比较 （JavaScript，CoffeeScript，Ruby，Python，Clojure，Haskell，Java）

    CodinGame¶

    免费同时学习编码和游戏. (C#, C++, Java, Javascript, Python, Bash, C, Clojure, Dart, F#, Go, Groovy, Haskell, Lua, ObjectiveC, Pascal, Perl, PHP, Ruby, Rust, Scala, Swift, VB.NET)

    1 Million Women To Tech Summer of Code¶

    适用于初级，中级和高级的免费编程课程资料 （Python，JavaScript，数据科学，人工智能，机器学习，AR和VR）

    Advanced¶

    MIT Courseware¶

    麻省理工学院关于高级计算机科学专题的免费课程 （各种广泛的计算机科学主题，C，C ++，..）

    Coursera¶

    免费平台，适用于全球顶尖大学和组织的课程\ （各种计算机科学科目，包括理论和编程，数据科学，算法......）

    prakhar1989/awesome-courses¶

    免费大学级课程从互联网上搜索. （各种各样的计算机科学主题，...）

    Metacademy Roadmaps, Course Guides¶

    掌握概念所需的互联主题的免费图表 （编程，机器学习）

    HackerRank¶

    免费编程挑战和竞赛\ （人工智能，算法，函数编程，机器学习）

    HackerEarth¶

    免费编程挑战，黑客马拉松和竞赛\ （动态编程，人工智能，算法，函数编程，机器学习）

    Project Euler¶

    免费数学/计算机编程问题 （编程，数学）

    CodeSignal¶

    免费编程挑战 (Java, C++, Python, Javascript, Ruby, C#, PHP and Perl)
