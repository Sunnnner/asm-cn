    CakePHP

    Awesome CakePHP ¶

    精选的CakePHP 3.x **插件，资源和闪亮的东西.

    如果您正在寻找CakePHP 2.x资源，请访问： - CakePHP 2.x version 这个很棒的清单 - 这个wiki有一个 list of not-yet upgraded plugins

    您可能会发现有用的其他列表： - CakePHP Plugins - Awesome PHP - Awesome Awesomeness

    >对于那些想知道的人; 这个列表与plugins.cakephp.org的不同之处在于支持 >插件子部分（而不仅仅是整个插件/ repo），更精细 >分组，主要关注任务特定功能.

    Plugins¶

    APM¶

    应用程序性能监控插件.

    • NewRelic plugin - 一个完整的插件，可以为CakePHP应用程序实现完整的New Relic集成，包括CLI命名，异常发送，自定义计时等.它不依赖于New Relic代理.

    • Brunitto/NewRelic plugin - 一个简单的插件，它使用New Relic PHP代理为CakePHP 3应用程序启用名称事务和浏览器计时.

    Asset Management¶

    用于管理，压缩和缩小网站资产的工具.

    • AssetCompress plugin - CakePHP的完整资产管理器.
    • Assets plugin - 动态和“动态”资产文件.
    • Less plugin - CakePHP的解析器插件较少.
    • MinifyHtml plugin - 压缩HTML输出.

    Auditing / Logging¶

    用于审计和记录的插件.

    • AuditStash plugin - 灵活且坚如磐石的审计日志跟踪.
    • DatabaseLog plugin - 简单且独立的日志记录到数据库而不是文件.
    • Muffin/Footprint plugin - 允许将当前登录用户传递给模型层的插件.
    • Version plugin - 一个便于版本化数据库实体的插件.

    Authentication and Authorization¶

    用于实现身份验证和授权的插件和库.

    • Acl plugin - 将ACL作为数据库方法管理.
    • ADmad/HybridAuth plugin - 允许使用的插件 HybridAuth 与CakePHP的图书馆的社会标志.
    • ADmad/JwtAuth plugin - 使用JSON Web令牌进行身份验证的插件.
    • Authentication plugin - 官方CakePHP身份验证中间件插件.
    • Authorization plugin - 官方CakePHP授权堆栈.
    • CakeDC/NavAuth plugin - 使用SOAP或OData服务对Navision®服务进行身份验证的插件. 它包括NTLM身份验证等.
    • CakeDC/Users plugin - 完整的用户管理（管理面板，记住我等），社交登录（FB，Twitter，LinkedIn，谷歌，Instagram），RBAC，API等.
    • CookieAuth plugin - 一个简单的Cake 3插件，可以使用Cookie自动验证用户身份.
    • HierAuth plugin - CakePHP插件，用于分层，基于角色的简单授权.
    • Muffin/OAuth2 plugin - 使用OAuth2身份验证 league/oauth2-client.
    • Muffin/Tokenize plugin - 事件驱动的行为，可轻松生成一次性安全令牌.
    • MultiTenant plugin - 轻松构建支持SaaS的Web应用程序.
    • TinyAuth plugin - 认证和基于角色的（单/多）授权作为非常轻量级的方法.
    • Tools:Passwordable - 包含 Passwordable behavior 用于密码散列的DRY方法.
    • TwoFactorAuth plugin - 允许使用Google身份验证器或类似应用进行双因素身份验证，以生成一次性代码. 基于 RobThree/TwoFactorAuth 图书馆.
    • UserPermissions plugin - 允许用户组或单个用户查看特定页面.

    Caching¶

    用于缓存数据的插件.

    • Cache plugin - 用于将视图（HTML，CSV，JSON，XML，...）缓存为静态缓存文件.

    Code Analysis¶

    用于分析，解析和操作代码库的插件.

    • IdeHelper plugin - 通过向现有代码添加注释，帮助更好地支持IDE，类似于烘焙对新代码的注释.
    • TestHelper plugin - 提供测试增强功能和TDD支持作为浏览器后端.

    Debugging¶

    用于调试的插件.

    • Airbrake plugin 一个插件，可以将Airbrake与CakePHP无缝集成，以实现错误和异常.
    • DebugKit plugin - 事实上的调试标准.
    • ErrorEmail plugin - 将异常/错误信息通过电子邮件发送给您的开发团队的插件.
    • Execution order - 一个演示应用程序，用于显示文件，方法和回调的执行顺序.
    • Psa/FixtureCheck plugin - 一个插件，可帮助检测实时DB和夹具中的不匹配，以使基于夹具的测试更可靠，部署更安全.
    • Sentry plugin 一个插件，可以将Sentry与CakePHP无缝集成，以实现错误和异常.
    • Setup plugin - 包含调试和维护工具的轻量级安装插件.
    • Whoops plugin - 适合酷孩子的PHP错误和例外 filp/whoops.

    Dependency Injection¶

    实现依赖注入设计模式的插件.

    • PimpleDi plugin 允许基于Pimple库的依赖注入.
    • PipingBag plugin - 依赖注入容器插件，它添加了在使用对象实例及其依赖项之前配置它们的能力，并将它们存储到容器类中以便于访问.

    E-commerce¶

    用于支付款项和建立在线电子商务商店的插件和应用程序.

    • PaypalWPP plugin - 用于与Paypal Web Payments Pro进行通信的包，用于交易和有关您帐户的信息.

    Email¶

    用于发送和解析电子邮件的插件.

    • Elastic Email plugin - 用于通过Elastic Email API发送电子邮件的电子邮件传输插件.
    • EmailQueue plugin - 具有预览和发件人shell的电子邮件队列插件.
    • Gourmet/Email plugin - 电子邮件帮助，布局等.
    • Mailgun plugin - 用于通过Mailgun发送电子邮件的电子邮件传输插件.
    • SendGrid plugin - 用于通过SendGrid API发送电子邮件的电子邮件传输插件.
    • SparkPost plugin - 用于通过SparkPost API发送电子邮件的电子邮件传输插件.

    Environment¶

    环境插件.

    • Environments plugin - Plugin to handle environments.
    • Gourmet/Aroma plugin - 基于数据库的配置.
    • Settings plugin - 通过数据库管理设置的插件.
    • Setup plugin - 处理非常基本环境的插件.

    Files¶

    用于文件操作的插件.

    • FileStorage plugin - 抽象文件存储和上传插件.
    • FlyPie plugin - 使用Flysystem进行抽象文件系统访问.
    • FriendsOfCake/Upload plugin - 使用的可自定义插件 Flysystem 写入多个后端（Dropbox，FTP，S3，Local等）.
    • Image plugin - 图像行为与Cake内置的TranslateBehavior非常相似.
    • Josbeir/Filesystem plugin - 摘要 Flysystem +基于文件实体的上传插件.
    • Proffer plugin - 具有缩略图生成功能的可自定义上传插件.
    • Upload plugin - 上传文件的小插件.

    Filtering and Validation¶

    用于过滤和验证数据的插件.

    • Gourmet/Filters plugin - 额外的调度过滤器（维护，机器人，IP等）.
    • Gourmet/Validation plugin - Extra validation providers (Respect, IsoCodes, etc.) and rules.
    • HtmlPurifier plugin - 具有特征，行为和帮助的Purifier插件，允许您在需要的地方进行消毒和过滤. 您也可以配置多组过滤规则.
    • HtmlPurifier plugin - Purifier Plugin行为，用于在数据被封送到实体之前和/或保存之前清理数据.

    Geolocation¶

    地址编码地址和处理纬度和经度的插件.

    • Geo plugin - 包含 Geocoder behavior 和 GoogleMaps helper.

    I18n¶

    用于I18n（国际化）和L10n（本地化）的插件.

    • ADmad/I18n plugin - 一个带有I18n相关工具的插件.
    • Localized plugin - 本地化验证和即用型翻译PO文件.
    • ShadowTranslate plugin - 基于影子表的插件替换核心的翻译行为.
    • Transifex plugin - 通过Transifex API管理i18n PO文件和翻译.
    • Translate plugin - 通过网络后端轻松管理静态内容的翻译，包括. 从POT文件导入，自动建议和通过API自动翻译.
    • Translation plugin - 提取pot文件，翻译字符串（手动，谷歌，社区），将翻译导出到pot文件.

    Imagery¶

    用于处理图像的插件.

    • ADmad/Glide plugin - 使用的插件 Glide 图像处理库.
    • HtmlToImageView plugin - 使用将HTML视图渲染为图像（jpg或png） wkhtmltoimage.
    • Imagine plugin - 图像处理插件和包装 Imagine.
    • Thumber plugin - 使用创建缩略图的插件 intervention/image.

    Libs¶

    有用的库或工具不适合任何其他类别.

    • Aura.Intl - 功能强大的I18n库 - 用于CakePHP 3.x核心.
    • Capcake - 使用Capistrano部署CakePHP应用程序.
    • Chronos - 一个简单的独立DateTime API扩展（Carbon的后继者）.
    • Composer Installers - 多框架Composer库安装程序.
    • Composer/Packagist - 包和依赖管理器.
    • Graphviz - Graphviz库.
    • Jenkins - 私人（GitHub）回购的免费替代品.
    • Rocketeer - PHP任务运行器和部署包.
    • Travis CI - 持续集成平台 - 事实上的测试标准（GitHub）回购.
    • YamlRoute - 使用简单的YAML文件配置路由.

    Markup¶

    用于处理标记的插件.

    • Gourmet/CommonMark plugin - 添加 CommonMark Markdown解析.
    • Markup plugin - 允许使用基于PHP或JS的语法突出显示.

    Migration¶

    有关迁移和升级的插件和资源.

    • Migrations plugin - （DB）迁移插件.
    • Upgrade app - 2.x => 3.x的官方升级应用程序.
    • Upgrade app (extended) - 2.x => 3.x和3.x之间的扩展升级应用程序.
    • Upgrade/Migration Guide - 官方移民指南.

    Miscellaneous¶

    其他插件和库.

    • ActionsClass plugin - 使您能够将Controller操作作为单个类进行管理.
    • Ajax plugin - 一个易于处理AJAX请求的插件.
    • CakeAdmin plugin - 具有内置管理区域的非稳定用户管理插件.
    • CakeDC/Enum - 一个为您的应用添加枚举列表支持的插件.
    • CakeDto plugin - 为您的应用程序快速生成有用的数据传输对象（可变/不可变），替换凌乱的数组并通过类型化和自动完成来利用您的IDE.
    • CakeImpersonate plugin - 存储当前身份验证会话并为模拟用户创建新会话的组件. 用户无需重新登录即可恢复原始身份验证会话.
    • CakeMiddlewares - Cakephp中间件的集合.
    • Calendar plugin - 用于生成基本日历. 包括用于ICS日历文件生成的IcalView.
    • Comments plugin - 完全可自定义的评论插件.
    • CurrencyConverter plugin - 将货币转换为另一种货币的插件.
    • Dashboard plugin - 为您的蛋糕打造漂亮的仪表板.
    • DatabaseBackup plugin - 用于导出，导入和管理数据库备份的插件.

    • Heartbeat plugin - 监控应用程序的听力（例如，数据库是否可用且是最新的）.

    • Robotusers/Tactician plugin - Tactician命令总线集成工具.
